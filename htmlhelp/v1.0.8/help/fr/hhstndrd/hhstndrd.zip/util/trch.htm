Commande : trch

  TRCH convertit toutes les instances d'un caractäre par un autre.
  C'est vraiment utile si vous devez convertir un fichier texte entre
  UNIX (la fin de fichier est "^J") et DOS (la fin de fichier est "^M^J"
  ou "\n").

Syntaxe :

  trch c1 c2 < fichier > fichier
       < fichier = fichier d'entrÇe
       > fichier = fichier de sortie

Options :

  c1  caractäre1, par exemple 'A'
  c2  caractäre2, par exemple 'B'

Commentaires :

   TRCH prend en charge NLS (national language support, prise en charge
   des langues nationales).

Exemples :

  trch ^I * < test.txt >test2.txt    Remplace les tabulations par '*'
  trch e X < test.txt > test3.txt    Remplace 'e' par 'X'
  trch \n $ < test.txt > test4.txt   Remplace la fin de ligne ('\n') '$*'
  trch ^J ^M^J < unix.txt > dos.txt  Replaces le format UNIX par
                                     le format DOS
  trch \n ^J < dos.txt > unix.txt    Remplace le format DOS par
                                     le format UNIX

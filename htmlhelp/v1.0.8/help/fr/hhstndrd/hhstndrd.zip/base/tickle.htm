Commande : tickle

  TICKLE est un read-ahead (lecture anticipÇe) pour LBACACHE. Cela
  augmente la vitesse des lecteurs de disquettes.

Syntaxe :

  tickle [/CHS] [/LBA] [/?]

Options :

  /CHS  Lecture anticipÇe pour anciens disques durs activÇe
        (mode CHS - Cylinder/Head/Sector).
  /LBA  Lecture anticipÇe pour nouveaux disques durs activÇe
        (mode LBA - Logical Block Adressing).
  /?    Affiche l'aide.

Commentaires :

  TICKLE lit toute la piste de la disquette en une fois lorsque
  vous lisez quelque chose sur cette piste - c'est plus rapide car
  la disquette doit tourner moins souvent.
  La lecture anticipÇe pour les disquettes est toujours activÇe.
  Chargez d'abord LBACACHE avec l'option FLOP activÇe !
  N'utilisez pas LOADHIGH avec TICKLE, risque de perte de
  donnÇes / problämes DMA !

Exemples :

  lbacache FLOP
  tickle /lba

Otras maneras de obtener ayuda

  1. La mayor¡a de los comandos se describen sus funciones y opciones,
     mediante el uso de /? sobre ellos en la l¡nea de comandos.
     Por ejemplo:
     dir /?
  2. El comando FASTHELP (FDHELP o FSUITE04). Este es un clon de MS-DOS
     fasthelp. Si usted no puede encontrar lo que busca en la ayuda,
     puede ser en fasthelp, o viceversa.
  3. Los comandos WHATIS and APROPOS.
  4. El comando HELP (HTMLHELP).
  5. El subdirectorio doc en el directorio de DOS contiene muchos de los
     manuales detallados para diversos programas de FreeDOS.
  6. Visite el sitio web de FreeDOS, http://www.freedos.org.

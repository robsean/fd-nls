Comando: time

  TIME muestra o establece la hora del sistema.

Sintaxis:

  TIME [/T] 
  TIME [nuevo_tiempo]

Opciones:

  nuevo_tiempo  hh:mm
                hh:mm:ss
                hh:mm:ss.ss
                La hora para configurar su sistema.
                'hh' es la hora en un reloj de 12 o 24 horas.
                'mm' son los minutos.
                'ss.ss' son los segundos y cent‚simas de segundos.
  /T            muestra la hora sin solicitar una nueva configuraci¢n.
  /?            muestra la ayuda.

Comentarios:

  Escriba TIME sin par metros para mostrar el ajuste de hora actual y 
  un aviso para uno nuevo. Presione ENTER para mantener el mismo tiempo.
  TIME es un comando interno de command.com y no necesita otro archivo
  para poder trabajar.

Ejemplos:

  time /T
  time 15:10
  time 15:10:20
  time 15:10:20.75

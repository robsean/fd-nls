Comando: vol

  VOL muestra la etiqueta del volumen de disco y el n£mero de serie, si
  existen.

Sintaxis:

  VOL [unidad]
  VOL [/?]
      unidad  Especifica la letra de unidad para mostrar la etiqueta del
              volumen, por ejemplo, C:

Opciones:

  /?  Muestra la ayuda.

Comentarios:

  Cada letra de unidad puede tener una etiqueta de volumen propia. Puede
  crearse con "LABEL", el nombre no puede tener m s de 11 letras.
  VOL es un comando interno a command.com y no necesita otro archivo para
  poder ejecutarse.

Ejemplos:

  vol c:
  vol d:

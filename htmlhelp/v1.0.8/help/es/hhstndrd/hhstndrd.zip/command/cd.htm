Comando: cd

  CD muestra el nombre o cambia de directorio actual.

Sintaxis:

  CD [unidad] [ruta]
  CD [..] [-] [\] [/?]
     unidad  La letra de la unidad, p.ej. C:
     ruta    Ruta del dirctorio, p.ej. \ejemplo\

Opciones:

  ..      Especifica que desea cambiar al directorio principal.
  -       Si la caracter¡stica "£ltimo directorio" est  habilitada,
          cambia al £ltimo directorio.
  \       Cambia al directorio raiz.
  /?      Muestra la ayuda.

Comentarios:

  Escriba CD unidad: para mostrar el directorio actual en la unidad 
  especificada.
  Escriba CD sin par metros para visualizar la unidad y el directorio 
  actuales.
  Si no se especifica ninguna unidad, entonces se asume la unidad actual.
  Si no se especifica ninguna ruta, se muestra el directorio actual.
  Tambi‚n puede usar CHDIR en lugar de CD.
  CD es un comando interno de command.com y no necesita otro archivo para
  que funcione.

Ejemplos:

    cd c:\fdos     Cambia al directorio c:\fdos
    cd ..          Cambia de c:\fdos a c:\
    cd \           Cambia de c:\anydirectory\anysubdirectory a c:\
  si est s en c:\ :
    cd fdos        Cambia al directorio c:\fdos
    cd bin         Cambia al directorio c:\fdos\bin
    cd d:\example  Cambia al directorio en d: a d:\example, pero no cambia
                   a la unidad d:

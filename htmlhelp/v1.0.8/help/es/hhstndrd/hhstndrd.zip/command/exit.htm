Comando: exit

  EXIT sale del shell de comandos de FreeDOS actual.

Sintaxis:

  EXIT [/?]

Opciones:

  /?  Muestra la ayuda.

Comentarios:

  EXIT es un comando interno del command.com y no necesita otro archivo
  para que funcione.

Ejemplos:

  EXIT
  EXIT /?

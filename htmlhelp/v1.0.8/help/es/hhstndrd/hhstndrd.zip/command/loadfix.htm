Comando: loadfix

  LOADFIX carga un programa por encima de los primeros 64K de memoria y
  ejecuta el programa. Tambi‚n carga un programa exepacked.

Sintaxis:

  LOADFIX [unidad][ruta]nom_archivo [opciones]
          unidad       La letra de la unidad, por ejemplo C:
          ruta         El directorio, por ejemplo \ejemplo\
          nom_archivo  El archivo que desea cargar, p.ej. edit.exe

Opciones:

  opciones  Estas son las opciones para el programa exepacked que est s
            cargando (= nom_archivo). Escribe "nom_archivo /?" primero
            para encontrar estas opciones.

Comentarios:

  Use LOADFIX para cargar un programa si ha recibido el mensaje "archivo
  empaquetado da¤ado" al intentar cargarlo en memoria baja. LOADFIX es un
  comando interno de command.com y no necesita otro archivo para que 
  funcione.

Ejemplos:

  EDIT /?                              muestra las opciones para EDIT
  LOADFIX C:\example\edit.exe options

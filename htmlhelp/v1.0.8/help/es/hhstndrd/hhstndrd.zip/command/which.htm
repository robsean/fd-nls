Comando: which

  WHICH busca y muestra el archivo ejecutable para cada comando 
  especificado.

Sintaxis:

  which comando [/?]

Opciones:

  comando  El comando que se va a encontrar.
  /?       Muestra la ayuda.

Comentarios:

  WHICH es una utilidad nacida en el mundo UNIX que permite a los usuarios
  encontrar archivos ejecutables en los directorios enumerados en la 
  variable de entorno PATH.
  WHICH es un comando interno a command.com y no necesita otro archivo
  para poder ejecutarse.

Ejemplos:

  which xcopy  Busca en el ejecutable el comando xxopy.
               La salida es: "xcopy C:\FDOS\BIN\xcopy.exe"
  which copy   La salida es: "copy" (copy is a command.com command)

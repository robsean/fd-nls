Comando: ctty

  CTTY cambia el terminal del dispositivo utilizado para controlar su
  sistema.

Sintaxis:

  CTTY dispositivo
  CTTY [/?]

Opciones:

  dispositivo  El terminal del dispositivo que desea usar.
               Los dispositivos v lidos son prn, lpt1, lpt2, lpt3, con,
               aux, com1, com2, com3, com4, NUL.
  /?           Muestra la ayuda.

Comentarios:

  CTTY es un comando interno de command.com y no necesita ning£n otro 
  archivo para que funcione.

Ejemplos:

  CTTY lpt1
  CTTY /?

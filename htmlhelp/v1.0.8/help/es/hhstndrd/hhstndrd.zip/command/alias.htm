Comando: alias

  ALIAS Crea, muestra o borra un alias para usar en la linea de comandos.

Sintaxis:

  ALIAS [variable[=][cadena]]
  ALIAS [/?]

Opciones:

  variable  Especifica el nombre del alias.
  cadena    Especifica una series de caracteres para asignar a un alias.
  /?        Muestra la ayuda.

Comentarios:

  Tipo ALIAS sin parametros muestra los alias actuales.
  Tipo "ALIAS variable=" (Sin cadena) para borrar el alias de nuevo.
  ALIAS es un comando interno de command.com y no necesita ning£n otro
  archivo para trabajar.

Ejemplos:

  "Alias"            Muestra una lista de alias actuales, si existen
  "Alias text=edit"  Si tu tecleas "text" + INTRO se abre el programa edit
  "Alias text="      La variable text es borrada. Si tu tecleas "text", el
                     editor no se abrir , se mostrar  un mensaje de error:
                     message: command "text" not found!

Comando: install

  INSTALL ejecuta un programa TSR (un programa que permanece en la
  memoria), sin un entorno, en lugar de usar algo como lh o loadhigh.
  INSTALL carga un programa TSR en memoria de bajo nivel o convencional,
  mientras que INSTALLHIGH carga el programa TSR en memoria de alto nivel.
  INSTALL es un comando de CONFIG.SYS / FDCONFIG.SYS.

Sintaxis:

  install=[unidad][ruta]nombre_archivo [opciones]
          unidad          La letra de la unidad, por ejemplo C:
          ruta            El directorio, por ejemplo \ejemplo\
          nombre_archivo  El nombre del archivo del TSR a ejecutar,
                          por ejemplo test.sys

Opciones:

  opciones  Estas son las opciones que se parar n al TSR. No son opciones
            para el comando 'INSTALL'.

Comentarios:

  INSTALL carga el programa especificado por unidad:\ruta\nombre_archivo.
  Generalmente se una para cargar programas TSR (terminar y permanecer
  residente) con un bloqueo de entrono m¡nimo. Las opciones son para el
  programa en s¡, consulte la documentaci¢n que vino con su software
  particular para conocer las opciones y el uso admitido. INSTALL es un

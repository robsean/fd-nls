Comando: lastdrive

  LASTDRIVE establece la £ltima letra a la que se puede asignar una unidad.
  LASTDRIVE es un comando de CONFIG.SYS / FDCONFIG.SYS.

Sintaxis:

  lastdrive=x
    donte x es la £ltima letra que estar  disponible.

Opciones:

  - ninguna -

Comentarios:

  De forma predeterminada, FreeDOS establecer  x lo suficientemente alto
  para todas las unidades de disquete y particiones de disco duro
  detectables, adem s de una para un CD y otra unidad. Es posible que desee
  establecer x m s alto si desea utilizar m s de una unidad que FreeDOS no
  puede detectar autom ticamente (unidades de CD, unidades zip, etc.).
  Establecer x m s alto de lo necesario desperdicia una peque¤a cantidad de
  memoria.
  LASTDRIVE es un comando interno de kernel.sys y no necesita otro archivo
  para poder trabajar.

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    lastdrive=z  habilita las letras de unidad de la A a la Z.

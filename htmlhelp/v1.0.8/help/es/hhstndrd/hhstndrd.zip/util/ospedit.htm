Comando: ospedit

  OSPEDIT es un editor de texto gratuito.

Sintaxis:

  ospedit [unidad][ruta][archivo]
          unidad   La letra de la unidad, ejemplo C:
          ruta     El directorio, ejemplo \carpeta\
          archivo  El archivo, ejemplo testo.txt.
                   Tambi‚n puede abrir archivos dentro del programa.

Opciones:

  - ninguna -

Comentarios:

  Un editor como OSPEDIT puede ser usado para crear archivos de texto 
  simple. se puede tambi‚n utilizar para modificar los archivos como
  CONFIG.SYS / FDCONFIG.SYS y AUTOEXEC.BAT.

Ejemplos:

  ospedit c:\example.txt   abre c:\example.txt para escribir en el
                           (archivo example.txt ya debe existir)
  ospedit c:\autoexec.bat
  ospedit c:\fdconfig.sys

Comando: pause

  PAUSE suspende el procesamiento de un programa por lotes y muestra
  el mensaje:
    "Pulse cualquier tecla para continuar...." 
  o un mensaje opcional especificado.
  PAUSE es un comando de ARCHIVO POR LOTES / AUTOEXEC.BAT.

Sintaxis:

  PAUSE [mensaje]

Opciones:

  mensaje  El mensaje que desea enviar.

Comentarios:

  PAUSE es un comando interno de command.com y no necesita ning£n otro
  archivo para funcionar.

Ejemplos:

  EN UN ARCHIVO .BAT / AUTOEXEC.BAT:
    pause
    pause Presione cualquier tecla

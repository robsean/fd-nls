Comando: mscdex

  MSCDEX es el controlador de CD-ROM de MS.
  FreeDOS usa el comando SHSUCDX en lugar de MSCDEX.

Sintaxis:

  MSCDEX [/E/K/S/V] [/D:dispositivo] [/L:unidad] [/M:bufferes]

Opciones:

  /E               En caso de que el sistema use memoria de expansi¢n,
                   debe usarse.
  /K               Se utilizan estructuras de archivos KANJI en lugar de
                   las estructuras de archivos normales.
  /S               La unidad de CD-ROM puede ser utilizada por sistemas
                   de red de MS.
  /V               Muestra informaci¢n sobre el uso de RAM.
  /D:dispositivo   El nombre del controlador de dispositivo que est 
                   instalado para controlar la unidad de CD-ROM.
  /L:unidad        La letra de unidad que utilizar  la unidad de CD-ROM.
  /M:bufferes      El n£mero de memorias que se utilizan para almacenar
                   la tabla de ruta de una unidad de CD-ROM.

  FreeDOS usa el comando SHSUCDX en lugar de MSCDEX.

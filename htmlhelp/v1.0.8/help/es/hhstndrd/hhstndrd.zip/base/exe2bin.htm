Comando: exe2bin

  EXE2BIN convierte programas EXE a formato binario. Esta es una
  herramienta para desarrolladores de software.

Sintaxis:

  EXE2BIN [opciones] archivo_exe[.exe] [archivo_bin]
          archivo_exe  El archivo a convertir
          archivo_bin  El nuevo archivo binario

Opciones:

  /Q        Suprimir mensajes informativos.
  /H        Mostrar el encabezado del exe.
  /R        Mostrar reubicaciones.
  /L=(seg)  Reubicar el ejecutable al segmento (seg).
  /X        Comportamiento extendido, para archivos > 64KB.
  /?        Muestrar la ayuda.

Comentarios:

  - ninguno -

Ejemplos:

  exe2bin c:\fdos\bin\ctmouse.exe c:\ctmouse

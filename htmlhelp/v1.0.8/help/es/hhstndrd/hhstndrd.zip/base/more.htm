Comando: more

  MORE muestra un archivo de texto o la salida de un comando una patalla
  a la vez.

Sintaxis:

  more < [unidad][ruta]archivo
  comando | MORE [/Tn] 
  MORE [/Tn] archivo
  MORE [/Tn] < archivo
  MORE [/?]
       unidad    Especifica la letra de la unidad donde est  el archivo,
                 por ejemplo C:
       ruta      Especifica la ruta a donde est  el archivo, por 
                 ejemplo \ejemplo\
       archivo   Especifica el archivo que desea mostrar, p. ej. text.txt
       comando   Un comando cuya salida canalizar  al porgrama more.

Opciones:

  Tn  Establece el tama¤o del tabulador en n, n tiene que estar en el
      rango entre 1 y 9.
  /?  Muestra la ayuda.
  Llaves:
    Espacio  P gina siguiente (solo mientras se ve un archivo)
    N n      Archivo siguiente (solo mientras se ve un archivo)
    Q q      Salga del programa (solo mientras ve un archivo)

Comentarios:

  MORE de FreeDOS admite comodines (* o ?) en los nombres de archivo.
  MORE admite NLS (soporte de idioma nacional).

Ejemplos:

  more < c:\config.sys
  fdisk /dump | more
  more /T4 *.sys
  more fdconfig.sys autoexec.bat

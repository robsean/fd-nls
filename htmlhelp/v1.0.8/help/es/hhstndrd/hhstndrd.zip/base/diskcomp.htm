Comando: diskcomp

  DISKCOMP compara el contenido de dos disquetes, pista por pista.

Sintaxis:

  diskcomp [unidad [unidad]] [/1] [/8] [/4[0]] [/b] [/?]
           unidad  La letra de unidad para un disquete. Si solo se
                   proporciona una letra de unidad, se asume la unidad actual
                   (si es una unidad de disquete). Si no se
                   proporcionan letras de unidad, se asume que ambas son
                   la unidad actual (si ambas son un disquete). Si ambas
                   unidades son iguales diskcomp leer  el disquete en la unidad,
                   y despu‚s deber  cambiar el disquete.

Opciones:

  /1   Comparar solo los primeros lados de los discos, incluso si son 
       discos de doble cara.
  /8   Comparar solo los primeros 8 sectores por pista.
  /40  Comparar solo las primeras 40 pistas (atajo: /4).
  /b   Obligarle a diskcomp a usar el BIOS para acceder a la unidad de 
       disquete. (El BIOS es m s lento, pero puede ser necesario para
       la compatibilidad en algunas m quinas).
  /?   Mostrar la ayuda.

Comentarios:

  Este programa compara dos discos (m ximo 32 MB cada uno) y muestra el
  resumen md5sum de los contenidos, e indica si el contenido del disco
  difiere (muestra un mensaje y establece el EXITCODE (ERRORLEVEL)
  adecuadamente).
  Si solo se le indica una unidad deber  cambiar los discos despu‚s de leer el
  primer disco. Si no ofrece ninguna opci¢n de unidad, se comparar n dos
  discos en la unidad actual.

Ejemplos:

  diskcomp A: A:  compara dos discos

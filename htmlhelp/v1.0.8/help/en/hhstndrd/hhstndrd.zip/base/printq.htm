Command: printq

  PRINTQ is a small program that utilizes the prnspool module,
  which is an interface to the DOS program PRINT.COM

Syntax:

  - none -

Options:

  - none -

Comments:

  PRINT.COM has to be installed before.

Examples:

  - none -

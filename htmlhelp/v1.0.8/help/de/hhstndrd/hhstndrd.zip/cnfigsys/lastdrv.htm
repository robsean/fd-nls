Befehl: lastdrive

  Lastdrive ist ein CONFIG.SYS / FDCONFIG.SYS-Befehl.
  Lastdrive vergibt den letzten Laufwerksbuchstaben, der einem Laufwerk
  zugeordnet werden kann.

Syntax:

  lastdrive=x
    wobei x der letzte verfuegbare Buchstabe ist.

Optionen:

  - keine -

Kommentar:

  Standardmaessig setzt FreeDOS x hoch genug, dass fuer alle Disketten-
  laufwerke und alle angezeigten Festplattenpartitionen sowie fuer ein
  CD-ROM-Laufwerk oder ein anderes Laufwerk Buchstaben vorhanden sind.
  Sie koennen x hoeher einstellen, wenn Sie mehr als ein Laufwerk
  verwenden wollen, das FreeDOS nicht automatisch erkennt (CD Laufwerke,
  ZIP-Laufwerke usw.). Wenn Sie x hoeher als notwendig stellen (z.B.
  lastdrive=Z) verschwendet dies ein wenig Speicher.
  LASTDRIVE ist ein interner Befehl, d.h. der Befehl ist in der kernel.sys
  enthalten. Es wird zur Ausfuehrung des Befehls kein zusaetzlicher
  Befehl (Datei) benoetigt.

Beispiel:

  IN DER CONFIG.SYS / FDCONFIG.SYS:
    lastdrive=z  ermoeglicht Laufwerksbuchstaben von A - Z.

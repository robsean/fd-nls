Befehl: switches

  SWITCHES richtet das Verhalten der Bootzeitprozedur ein.
  SWITCHES ist ein CONFIG.SYS / FDCONFIG.SYS-Befehl.

Syntax:

  switches=/E[:xxx] /F /K /N /W

Optionen:

  /F   uebergeht die Verzoegerungszeit zum Ueberpruefen ob F5/F8 gedrueckt
       wurde bevor die config.sys ausgefuehrt wird [analog zu 
       SYS CONFIG skipconfigseconds=0]. F5 und F8 werden nur dann 
       ausgefuehrt, wenn Sie gedrueckt waren, bevor DOS bootet, aber 
       nachdem die Tastatur gestartet ist - man kann den richtigen Moment
       sehr leicht verpassen.
  /N   Schaltet die F5/F8 Unterstuetzung aus [vergleichbar mit kernel
       config (SYS CONFIG, run SYS CONFIG /? fuer Erlaeuterungen)
       skipconfigseconds=-1]. F5 (uebergehe config) und F8 (durchlaufe
       die config.sys in einzelnen Schritten) werden ignoriert.
  /K   Erzwingt die Behandlung der Tastatur als 86-Tasten Tastatur,
       nicht 102/105 Tasten. Das kann bei BIOSen oder Treibern hilfreich
       sein, die 102/105 Tasten Tastaturen nicht oder nur bedingt
       unterstuetzen koennen.
  /E   ermoeglicht das Verschieben von EBDA (Extended BIOS Data Area),
       optional kann eine Groesse in Kilobyte angegeben werden [xxx, im
       Bereich von 48-1024].
  /W   wird in FreeDOS NICHT unterstuetzt. Diese Option wuerde bei MS DOS
       fuer Windows 3.0 einen Merker setzen, um das Laden der wina20.386
       aus dem Stammverzeichnis zu uebergehen.

Kommentar:

  SWITCHES ist ein interner Befehl, d.h. der Befehl ist in der kernel.sys
  enthalten. Es wird zur Ausfuehrung des Befehls kein zusaetzlicher
  Befehl (Datei) benoetigt.

Beispiel:

  IN DER CONFIG.SYS / FDCONFIG.SYS:
    switches=/E[:48] /F /K /N /W

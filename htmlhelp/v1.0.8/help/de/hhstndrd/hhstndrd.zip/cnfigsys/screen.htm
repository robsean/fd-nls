Befehl: screen

  SCREEN setzt die Anzahl von Textzeilen auf dem Bildschirm.
  SCREEN ist ein CONFIG.SYS / FDCONFIG.SYS-Befehl.

Syntax:

  screen=xx

Optionen:

  xx (Codes):
    0x11  8x14 Font (28 Zeilen VGA, 25 Zeilen EGA)
    0x12  8x8  Font (50 Zeilen VGA, 43 Zeilen EGA)
    0x14  Standard VGA Font (25 Zeilen VGA)

Kommentar:

  SCREEN schaltet in den Videomode xx (INT10/11xx/000). Einige neuere
  Grafikkarten koennen keine 8x14 Fonts im BIOS haben. In diesem Fall 
  kann ein Treiber geladen werden um ein passendes Font in den RAM zu 
  laden, aber SCREEN=0x11 sollte nicht verwendet werden. Wenn xx weniger
  als 15 ist (0x0f), wird es als SCREEN Modusnummer behandelt, z.B. 1 fuer
  CGA 40x25 Farbtext oder 7 fuer Monochromtext.
  SCREEN ist ein interner Befehl, d.h. der Befehl ist in der 
  kernel.sys enthalten. Es wird zur Ausfuehrung des Befehls kein 
  zusaetzlicher Befehl (Datei) benoetigt.

Beispiel:

  IN DER CONFIG.SYS / FDCONFIG.SYS:
    screen=0x12

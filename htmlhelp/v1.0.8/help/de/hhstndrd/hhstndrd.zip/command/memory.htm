Befehl: memory

  MEMORY zeigt den Speicher an, der von FreeCOM
  (command.com) intern verwendet wird.

Syntax:

  MEMORY [/?]

Optionen:

  /?  Zeigt Informationen ueber den Befehl an

Kommentar:

  MEMORY zeigt an, wieviel Speicher von Alias, der History,
  dem Verzeichnisstack, dem "Letztes Verzeichnis"-Cache, der
  Swapinfo usw. gebraucht wird.
  MEMORY ist ein interner Befehl, d.h. der Befehl ist in der command.com
  enthalten. Es wird zur Ausfuehrung des Befehls kein zusaetzlicher 
  Befehl (Datei) benoetigt.

Beispiel:

  memory /?
  memory

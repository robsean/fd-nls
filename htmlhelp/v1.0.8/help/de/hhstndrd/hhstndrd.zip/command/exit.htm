Befehl: exit

  EXIT beendet die aktuelle FreeDOS Command Shell.

Syntax:

  EXIT [/?]

Optionen:

  /?  Zeigt die Hilfe

Kommentar:

  EXIT ist ein interner Befehl, d.h. der Befehl ist in der command.com
  enthalten. Es wird zur Ausfuehrung des Befehls kein zusaetzlicher 
  Befehl (Datei) benoetigt.

Beispiel:

  EXIT
  EXIT /?

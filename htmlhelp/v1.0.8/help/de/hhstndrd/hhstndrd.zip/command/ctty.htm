Befehl: ctty

  CTTY wechselt das Terminalgeraet, das zur Steuerung Ihres 
  Systems verwendet wird.

Syntax:

 CTTY Geraet
 CTTY [/?]

Optionen:

  Geraet  Das Terminalgeraet das Sie verwenden moechten.
          Gueltige Geraete sind: prn, lpt1, lpt2, lpt3, con,
          aux, com1, com2, com3, com4, NUL.
  /?      Zeigt die Hilfe

Kommentar:

  CTTY ist ein interner Befehl, d.h. der Befehl ist in der command.com
  enthalten. Es wird zur Ausfuehrung des Befehls kein zusaetzlicher 
  Befehl (Datei) benoetigt.

Beispiel:

  CTTY lpt1
  CTTY /?

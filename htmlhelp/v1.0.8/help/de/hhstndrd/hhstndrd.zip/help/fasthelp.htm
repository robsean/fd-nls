Befehl: fasthelp (fdhelp)

  FASTHELP ist ein fasthelp-Clone fuer FreeDOS. Zeigt Hilfe
  zu einem Thema an, im Stil der Linux man-pages.

Syntax:

  fasthelp [Thema]

Optionen:

  Thema  Das Thema, ueber das Sie Informationen wollen.
  Umgebung:
    PAGER:    Alternativprogramm, um eine Textdatei anzuzeigen.
    HELPPATH: Verzeichnis, das die Hilfe-Dateien enthaelt.
    LANG:     Bevorzugte Sprache fuer die Hilfe-Dateien.

Kommentar:

  Verwenden Sie die HELPPATH und LANG Umgebungsvariablen um die Lage und
  die Sprache der Hilfedateien richtig zuzuordnen. Der Standardpfad ist 
  \help und die Standardsprache ist EN (englisch).
  Es gibt auch die PAGER Umgebungsvariable, die dazu benutzt werden kann,
  ein alternatives Programm fuer die Anzeige der Hilfedateien zu waehlen.
  Aufrufe mit FASTHELP werden die Textdateien an das PAGER Programm 
  (Alternativprogramm) umleiten.
  Wenn Sie mehr ueber Umgebungsvariablen wissen wollen, sehen Sie bitte
  unter SET nach.
  FASTHELP wird auch "FSUITE" oder "FSUITE04" oder "FS04de" (deutsch)
  oder "FSUITE04de" (deutsch) genannt.

Beispiel:

  fasthelp       Zeigt Ihnen eine Liste der Themen
  fasthelp edit  Zeigt Ihnen Informationen ueber den Befehl edit

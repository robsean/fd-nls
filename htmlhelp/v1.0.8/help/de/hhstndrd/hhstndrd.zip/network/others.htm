Netzwerken unter FreeDOS - Andere

  Obwohl die erwaehnten Bereiche ueber Netzwerken unter DOS-Systemen die
  meisten Programme abdecken sollten, gibt es doch ein paar Ausnahmen.
  Eine davon ist "Invisible LAN (unsichtbares LAN", (vgl.: http://
  www.invisiblesoft.com/invlan/index.shtml), eine NetWare-aehnliche DOS-
  Anwendung, die mit einem eigenen Protokoll namens "TransBIOS" daherkommt.
  Eine andere Spezialloesung um schnell ein Netzwerk hoch- bzw. zum
  Laufen zu bekommen sind Netzwerkbootdisketten. Sie sind in der Lage, mit
  einem weiten Spektrum an Hardware zu arbeiten und verwenden verschiedene
  Protokolle. Bootdisketten werden hauptsaechlich fuer Backup (Sicherung),
  Restore (Wiederherstellung) und Reparaturen verwendet. Sie koennen
  aeusserst hilfreich sein.
  Beispiele sind:
    * Bart's Netzwerk Boot Disk (vgl.: http://www.nu2.nu/bootdisk/
      network/ (*01))
    * Drivesnapshot (vgl.: http://drivesnapshot.de/en/index.htm (*02))
    * Netbootdisk (vgl.: http://www.netbootdisk.com/ (*03))
  Die gesetzliche Situation ist ungluecklich: Einige Pakete scheinen
  Dateien von MS Client zu enthalten, waehrend Microsofts Lizenzbe-
  dingungen eine solche Verteilung nicht zulassen.

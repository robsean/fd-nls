Netzwerken unter FreeDOS - NDIS

  Die NDIS (Network Driver Interface Specification = Netzwerktreiberverbin-
  dungsspezifikation) ist von Microsoft und 3Com und liefert Die Faehigkeit
  der Netzwerkverbindung.
  Wie ODI uebergibt NDIS eine einheitliche Schnittstelle an die Anwendung,
  unabhaengig von der Netzwerkmethode, aber eine andere als die der ODI-
  oder der Pakettreiber.
  Die  NDIS Treiber verbrauchen von allen verwendbaren Treibern den
  meisten Arbeitsspeicher; sie sollten deshalb nur dann verwendet werden,
  wenn kein anderer Treiber verfuegbar ist ODER die Software, die Sie
  verwenden, NUR mit NDIS funktioniert.

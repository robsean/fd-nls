Externe Befehle:

  Diese Dateien sind Teil der Standard FreeDOS Distribution.
  Damit sie funktionieren, benoetigt man eine existente ausfuehrbare
  Datei (*.com oder *.exe-Datei) und FreeDOS muss bereits gestartet
  sein. Man kann sie ausfuehren, indem man den Namen eintippt, z.B.
  "edit" oder "format" und - wenn notwendig - zusaetzlich die Option/
  den Schalter usw. vgl. SYNTAX. Das Programm sollte sich im aktuellen
  Verzeichnis befinden, andernfalls sollten Sie vorher den CD oder
  den PATH Befehl benutzen.

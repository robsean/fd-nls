Befehl: printq

  PRINTQ ist eine kleines Programm, das das prnspool Modul verwendet,
  welches eine Schnittstelle zum DOS Programm PRINT.COM ist.

Syntax:

  - keine -

Optionen:

  - keine -

Kommentar:

  PRINT.COM muss zuvor installiert sein.

Beispiel:

  - keines -

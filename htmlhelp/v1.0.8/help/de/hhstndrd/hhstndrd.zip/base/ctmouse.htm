Befehl: ctmouse

  CuteMouse (CTMOUSE) ist der FreeDOS Maustreiber.

Syntax:

  ctmouse [Optionen]
  LH ctmouse [Optionen]
  LOADHIGH ctmouse [Optionen]

Optionen:

  /3        3-Tasten-Modus erzwingen (nur fuer Microsoft- und
            PS/2-Maeuse).
  /B        Programm abbrechen, wenn Mausfunktionen bereits aktiv sind.
  /L        Linkshaendermodus (Vorgabe ist Rechtshaendermodus).
  /M        Probiere *alten* Mouse Systems / Genius Modus falls keine PnP
            Maus.
  /N        CuteMouse als neues residentes Programm (TSR) laden, selbst
            dann, wenn CTMOUSE bereits resident ist (sinnvoll fuer
            Batchdateien, die Cutemouse am Ende wieder entladen).
  /O        Verwende PS/2 / USB BIOS Mausrad Erkennung (experimentell).
  /P        PS/2-Mausmodus erzwingen.
  /R[h[v]]  Horizontale/vertikale Aufloesung (h,v=0-9; Wenn keine Option
            angegeben wird, das Argument hinter /R 0 ist oder hinter /R
            keine Werte angegeben werden, wird wird die Standardaufloesung
            verwendet. Wenn das zweite Argument fehlt, wird das erste
            Argument auch als zweites Argument verwendet.
  /S[c[i]]  Seriellen Mausmodus erzwingen an COM c (c=1-4, Standard ist
            COM1) mit IRQ i (i=1-7, Standard ist IRQ 4 fuer COM 1/3 und
            IRQ3 fuer COM 2/4).
  /U        CuteMouse Treiber deinstallieren.
  /V        Suche umkehren: PS/2 nach serieller Maus finden.
  /Y        Erzwingt die Suche nach einer seriellen Maus, schaltet aber
            anders als /S die PS/2  Unterstuetzung nicht aus.
  /W        TSR nicht automatisch in den UMB laden.
  /?        Zeigt die Hilfe.

Kommentar:

  CTMOUSE unterstuetzt keine USB-Maeuse.
  CTMOUSE Kann sowohl in der CONFIG.SYS / FDCONFIG.SYS als auch
  in der AUTOEXEC.BAT oder in der Kommandozeile geladen werden.
  CTMOUSE2 ist eine neue Version von CuteMouse die das Scrollrad
  unterstuetzt. Es hat eine Scrollrad API (Anwendungsprogrammierer-
  schnittstelle) so dass die FreeDOS Programme das Scrollrad verwenden
  koennen. Besuchen Sie die CuteMouse Webseite unter: 
  http://cutemouse.sourceforge.net
  CTMOUSE hat folgende Beendigungswerte (EXITCODES/ERRORLEVEL):
    0  PS/2, Microsoft oder Logitech Maus gefunden und CTMOUSE
       installiert; Entladen erfolgreich; /? Option verwendet.
    1  CTMOUSE im Mouse Systems Modus installiert; Entladen
       funktionierte nicht, weil CTMOUSE nicht geladen war.
    2  Der residente Teil schaltete in den PS/2, Microsoft
       oder Logitech Modus; das Entladen funktionierte nicht,
       weil die Treiberinterrupts abgefangen wurden.
    3  Der residente Teile schaltete in den Mouse Systems Modus.
    4  Maustreiber bereits geladen (wird nur bei Option /B ausgegeben).
    5  Maus nicht gefunden; ungueltige Option verwendet.

Beispiel:

  IN DER CONFIG.SYS / FDCONFIG.SYS:
    devicehigh=C:\fdos\bin\ctmouse  laedt ctmouse in den hohen Speicher
  IN DER AUTOEXEC.BAT ODER IN DER KOMMANDOZEILE
    lh ctmouse  laedt ctmouse in den hohen Speicher
    ctmouse /u  entlaedt den Maustreiber
    ctmouse /?  Zeigt die Hilfe (Optionen)

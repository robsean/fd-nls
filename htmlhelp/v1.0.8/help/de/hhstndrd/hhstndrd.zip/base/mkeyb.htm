Befehl: mkeyb

  MKEYB ist ein kompakter Tastaturtreiber.

Syntax:

  mkeyb [Optionen] Code
  keyb [Optionen] Code

  Laedt die Tastaturen der verschiedenen Laender.
  Benoetigt etwa 500 Bytes (abhaengig vom Land).
  Lage:  

  Code    Tastatur                Erfordert Code Page: 
  ====================================================
   BE     Belgien
   BR     Brasilien
   BX     Brasilien (erweitert)
   DK     Daenemark
   FR     Frankreich
   GR     Deutschland
   GRC    Deutschland (mit COMBIs)
   HE     Israel
   IT     Italien
   LA     Latein Amerika
   NL     Niederlande
   NO     Norwegen
   PL     Polen
   PO     Portugal
   RU     Russland                       866
   SF     Schweiz (Franzoesisch)
   SG     Schweiz (Deutsch)
   SP     Spanien                        850
   SU     Finnland
   SV     Schweden
   UK     United Kingdom (Grossbritannien)

Optionen:

  /l  Listet alle verfuegbaren Tastaturen
  /u  Entlaedt den Tastaturtreiber
  /t  Testet den angegebenen Tastaturtreiber, wird nicht resident
  /?  Zeigt die Hilfe

Kommentar:

  MKEYB (KEYB) laedt sich automatisch so hoch wie moeglich.
  Es ist deshalb nicht notwendig, LH / LOADHIGH zu verwenden.
  MKEYB (KEYB) wird mit dem Paket MKEYB040 ausgeliefert.

Beispiel:

  mkeyb GR     laedt den deutschen Tastaturtreiber
  mkeyb /u     entlaedt den Tastaturtreiber
  mkeyb GR /t  testet den deutschen Tastaturtreiber, wird nicht resident

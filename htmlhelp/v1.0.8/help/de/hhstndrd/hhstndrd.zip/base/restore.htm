Befehl: restore

  RESTORE stellt aus den Sicherungsdateien die Dateien wieder her.
  BACKUP sichert Dateien indem es Sicherungsdateien erstellt.
  Die Befehle BACKUP / RESTORE gibt es derzeit unter FreeDOS nicht.

Syntax:

  - keine -

Optionen:

  - keine -

Kommentar:

  - none -

Beispiel:

  - keines -

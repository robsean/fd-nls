Befehl: find

  FIND zeigt Zeilen einer oder mehrerer Dateien an, die eine bestimmte
  Zeichenfolge enthalten.

Syntax:

  FIND [ /C ] [ /I ] [ /N ] [ /V ] "Zeichenfolge" 
       [[Laufwerk][Pfad][Datei1]]
  FIND [/?]
       Laufwerk  Der Laufwerksbuchstabe, z.B: C:
       Pfad      Der zu waehlende Pfad, z.B.: \Beispiel\
       Datei1    Die zu durchsuchende Datei, z.B. test.exe
                 Anmerkung: Wenn Sie keine Datei angeben, erwartet
                 find eine Eingabe in der Konsole, z.B. find "auer" -
                 Sie geben "Die Mauer ist hoch" ein und find bestaetigt,
                 dass "auer" in diesem Satz enthalten ist. Tippen Sie
                 Strg-Z um den Befehl zu beenden.

Optionen:

  /C             Zeigt die Anzahl der Zeilen an, die eine bestimmte
                 Zeichenfolge enthalten.
  /I             Ignoriert Gross- und Kleinschreibung beim Vergleichen.
  /N             Nummeriert die angezeigten Zeilen, beginnend mit 1.
  /V             Dreht die Suche um: Zeigt alle Zeilen an, die eine
                 bestimmte Zeichenfolge NICHT enthalten.
  "Zeichenfolge" Die Zeichenfolge, die Sie finden wollen, z.B. "Wasser"
                 oder "0123".
                 Die Zeichenfolge muss in Anfuehrungszeichen stehen.
  /?             Zeigt die Hilfe.

Kommentar:

  FIND unterstuetzt NLS (national language support - nationale Sprach-
  unterstuetzung).
  FIND hat folgende Beendigungswerte (EXITCODES/ERRORLEVEL):
    0  gefunden
    1  keines gefunden
    2  Suche nicht vervollstaendigt

Beispiel:

  find /c loadhigh c:\autoexec.bat
  find /c devicehigh c:\config.sys

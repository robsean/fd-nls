Befehl: scandisk

  SCANDISK ist ein MS-Programm zum Testen von Medien 
  (Disketten / Festplatten). 
  SCANDISK wird unter FreeDOS nicht verwendet. Bitte benutzen Sie 
  stattdessen CHKDSK oder DOSFSCK.

Syntax:

  - keine -

Optionen:

  - keine -

Kommentar:

  - keiner -

Beispiel:

  - keines -

Siehe auch:

  chkdsk
  dosfsck

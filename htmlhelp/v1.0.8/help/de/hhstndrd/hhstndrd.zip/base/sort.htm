Befehl: sort

  SORT sortiert eine Datei oder Eingabe fuer die Ausgabe.
  Die Eingangsdatei wird nicht veraendert.

Syntax:

  sort [/R] [/N] [/+num] [/A] [/?] [Laufwerk][Pfad]Dateiname
  command | sort [/R] [/N] [/A] [/+num] [/?]
       Laufwerk   Der Laufwerksbuchstabe, z.B: C:
       Pfad       Der zu waehlende Pfad, z.B.: \Beispiel\
       Dateiname  Der zu ladende Treiber (Dateiname), z.B. geordnet.txt

Optionen:

  /R     Sortiert in umgekehrter Reihenfolge (Z - A).
  /A     Sortiert nach ASCII, ignoriert COUNTRY.
  /N     Ermoeglicht NLS / CODEPAGE / COUNTRY basierte Sortier
         reihenfolge.
  /+num  Fange mit dem Sortieren bei Spalte num jeder Textzeile an.
         Standard=1.
  /?     Zeigt die Hilfe.

Kommentar:

  Die zu sortierende Datei wird durch den Befehl nicht veraendert, nur 
  ggfs. die Ausgabedatei.
  SORT unterstuetzt NLS (national language support - nationale Sprach-
  unterstuetzung).

Beispiel:

  Zeigt das aktuelle Verzeichnis in alphabetischer Reihenfolge:
    dir |sort  
  Sortiert eine Datei und schreibt sie in eine andere Datei:
    sort unordered.txt > ordered.txt  
    sort           -ENTER-   (alternativ: sort /+2)
    this           -ENTER-
    is             -ENTER-
    a              -ENTER-
    short          -ENTER-
    sentence       -ENTER-
  (CTRL-Z)  (beendet den Befehl und fuehrt ihn aus)
    gibt aus:                 sortiert den zweiten Buchstaben
                              pro Zeile:            
    a                         a
    is                        sentence 
    sentence                  this
    short                     short
    this                      is

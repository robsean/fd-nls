Befehl: diskcomp

  DISKCOMP vergleicht den Inhalt von zwei Disketten, Spur fuer Spur.

Syntax:

  diskcomp [Laufwerk [Laufwerk]] [/1] [/8] [/4[0]] [/b] [/?]
           Laufwerk  Der Laufwerksbuchstabe der Diskette. Wenn nur ein 
                     Laufwerksbuchstabe angegeben wird, wird davon 
                     ausgegangen, dass dies auch das Vergleichslaufwerk
                     sein soll (bei Disketten). Wenn keine Laufwerksbuch-
                     staben angegeben werden, wird davon ausgegangen, dass
                     das Arbeitslaufwerk auch das Vergleichslaufwerk ist
                     (bei Disketten). Wenn beide Laufwerke (Quelle und 
                     Vergleichsziel) identisch sind, muessen die Disketten
                     zum Vergleichen mehrfach gewechselt werden.

Optionen:

  /1   Vergleicht nur die erste Seite der Disketten, selbst wenn es
       doppelseitige Disketten sind. 
  /8   Vergleicht nur die ersten 8 Sektoren pro Spur.
  /40  Vergleicht nur die ersten 40 Spuren (Abkuerzung: /4)
  /b   Zwingt diskcomp, das BIOS zu verwenden, um auf das Floppylaufwerk
       zuzugreifen. (Das BIOS ist langsamer, dies kann aber aus 
       Kompatibilitaetsgruenden auf manchen Maschinen erforderlich sein.)
  /?   Zeigt die Hilfe

Kommentar:

  Das Programm vergleicht zwei Disks (maximal jeweils 32 MB) und zeigt
  die md5-Summen-Uebersicht der Inhalte. Es meldet, ob die Diskinhalte
  unterschiedlich sind (zeigt eine Meldung und setzt den entsprechenden
  ERRORLEVEL).
  Wenn Sie nur einen Laufwerksbuchstaben angeben, muessen Sie die Disks
  nach dem Einlesen der ersten Disk wechseln. Wenn Sie keine Laufwerks-
  optionen angeben, werden im aktuellen Laufwerk zwei Disks miteinander
  verglichen.

Beispiel:

  diskcomp A: A:  vergleicht zwei Disketten

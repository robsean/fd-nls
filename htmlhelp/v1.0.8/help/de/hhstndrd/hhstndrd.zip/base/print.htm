Befehl: print

  PRINT druckt im Hintergrund eine Datei waehrend Sie andere
  DOS-Befehle ausfuehren.

Syntax:

   print [Optionen] [[Laufwerk:][Pfad]Datei] [[drive:][path]file] [...]
           Laufwerk:Pfad\Datei Die Datei die Sie drucken wollen.

Optionen:

  /D:device  Spezifiziert den Namen des Geraetes auf dem gedruckt wird.
             Wenn nichts angegeben, wird Print standardmaessig das Geraet
             LPT (Drucker) verwenden.
  /B:size    Setzt die Groesse der Puffer in Bytes im Bereich von
             512-16.384 Bytes. Standard ist 512.
  /U:ticks   Die Verzoegerungszeit im Bereich von 1-255, die Print wartet,
             bis der Drucker fuer den Druckauftrag bereit ist.
  /M:ticks   Die maximale Verzoegerungszeit, um ein einzelnes Zeichen an
             den Drucker zu senden, im Bereich von 1-255.
  /S:ticks   Die Verzoegerungszeit im Bereich 1-255 fuer den Hintergrund-
             Spoolerprozess. Standard ist 2. Grosse Zahlen beschleunigen
             das Drucken, bremsen aber andere DOS Programme.
  /Q:nn      Die Anzahl der Dateien, die in der Druckerwarteschleife sein
             duerfen, im Bereich 1-32. Standard ist 10.
  /T         Entfernt alle Dateien in der Druckerwarteschleife. Beendet
             den aktuell zu druckenden Auftrag.
  /C         Beendet Auftraege aus der Druckerwarteschlange. Wenn dies
             zusammen mit einer Liste von Dateinamen verwendet wird,
             werden nur diese Dateien aus der Schlange entfernt. Wenn die
             Option fuer sich allein verwendet wird, werden alle Druckauf-
             traege (bis auf den aktuellen Auftrag) beendet. 
  /P         Fuegt der Druckerwarteschlange eine Datei hinzu. Wenn diese
             Option einer Dateiliste vorangestellt wird, werden nur diese
             Dateien gedruckt. Sie koennen /P und /C in der Kommandozeile
             zusammen verwenden.

Kommentar:

  Wenn PRINT keine Optionen mitgegeben werden, wird unterstellt, dass
  /P gewaehlt ist. Wenn keine Optionen und keine Dateien angegeben werden,
  zeigt PRINT den Inhalt der Druckerwarteschlange an.
  Dieser PRINT Befehl akzeptiert nur /1 /2 /3 als "D:lpt1" usw.
  gleichwertige Option oder einen Dateinamen um eine Datei zu drucken.
  Bitte verwenden Sie das freie PRINTQ-Tool um "eine Datei zu drucken",
  "print /t" um die Warteschlange inclusive aktuellen Auftrag zu loeschen,
  "print /c" um die Warteschlange zu loeschen und "print /p Datei" um
  eine Datei zur Warteschlange hinzuzufuegen.

Beispiel:

  print c:\foo.txt c:\test.txt

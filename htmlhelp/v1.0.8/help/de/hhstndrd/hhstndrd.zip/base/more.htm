Befehl: more

  MORE zeigt den Inhalt einer Textdatei oder die Ausgabe eines Befehls
  seitenweise an.

Syntax:

  more < [Laufwerk][Pfad]Datei
  Befehl | MORE [/Tn] 
  MORE [/Tn] Datei
  MORE [/Tn] < Datei
  MORE [/?]
       Laufwerk  Spezifiziert den Laufwerksbuchstaben auf dem sich die 
                 Datei befindet, z.B. C:
       Pfad      Spezifiziert den Pfad zu der Datei,z.B. \Beispiel\
       Datei     Spezifiziert die Datei die Sie anzeigen lassen wollen,
                 z.B. test.txt
       Befehl    Der Name des Befehls, dessen Ausgabe Sie an den 
                 more-Befehl weiterleiten wollen

Optionen:

  Tn  Setzt Tabulatorgroesse auf n, wobei n zwischen 1 und 9 sein muss
  /?  Zeigt die Hilfe
  Tasten:
    Leertaste  Naechste Seite (nur waehrend des Betrachtens)
    N n        Naechste Datei (nur waehrend des Betrachtens)
    Q q        Programm beenden (nur waehrend des Betrachtens)

Kommentar:

  FreeDOS MORE unterstuetzt Wildcards (* und ?) in Dateinamen.
  MORE unterstuetzt NLS (national language support - nationale Sprach-
  unterstuetzung).

Beispiel:

  more < c:\config.sys
  fdisk /dump | more
  more /T4 *.sys
  more fdconfig.sys autoexec.bat

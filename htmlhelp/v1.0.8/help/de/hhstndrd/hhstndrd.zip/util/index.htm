Utilities (Zubehoer):

  Diese UTILITIES sind nicht Teil der Standard FreeDOS Distribution, aber
  sie funktionieren selbstverstaendlich unter FreeDOS. Um sie auszufuehren,
  benoetigen Sie eine ausfuehrbare Datei (*.com oder *.exe-Datei) und 
  FreeDOS muss bereits hochgefahren sein.
  Sie koennen das Programm ausfuehren, indem Sie den Namen des Programmes
  eintippen, z.B. "tee" oder "foxcalc". Das Programm sollte sich im
  aktuellen Verzeichnis befinden, ausser Sie haben zuvor mit PATH
  einen Pfad zu dem Programm gelegt oder sind mit CD dorthin gewechselt.
  In diesem Bereich finden Sie nur eine kleine Liste von Tools, die oft
  zusammen mit FreeDOS verwendet werden. Wenn Sie ein spezielles Tool
  brauchen, suchen Sie bitte im Internet danach.

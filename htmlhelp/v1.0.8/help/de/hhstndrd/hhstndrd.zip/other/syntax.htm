Hilfe zum Thema "Syntax":

  Bei den meisten Hilfethemen werden Sie etwas aehnliches wie das hier
  vorfinden:

    command required [optional] [/switch] [/A|/B]

  Hier eine Erklaerung der unterschiedlichen Teile dieser Zeile:

    command
  Das ist der Name des Befehls.
   
    required
  Alles was dem Befehlsnamen folgt und nicht in eckigen Klammern ist, 
  [und] ist ein erforderlicher Parameter.
  Der Befehl wird einen Fehler ausgeben, wenn er diese Information 
  nicht erhaelt.
   
    [optional]
  Optionen werden in eckigen Klammern geschrieben [und]. Diese 
  veraendern die Art und Weise wie der Befehl laeuft. Die eckigen Klammern 
  muessen nicht eingetippt werden, wenn der Befehl benutzt wird.

    [/switch]
  Optionen, die mit einem / beginnen werden Schalter (switch) genannt. 
  Das / Symbol muss normalerweise eingetippt werden, wenn der Schalter 
  benuetzt wird.

    [/A|/B]
  Wenn ein | Symbol angezeigt wird, bedeutet dies, dass es zwei 
  Moeglichkeiten gibt.
  Entweder ist /A oder /B erlaubt, aber nicht beide.
  Das | muss nicht eingetippt werden, wenn der Befehl benutzt wird.

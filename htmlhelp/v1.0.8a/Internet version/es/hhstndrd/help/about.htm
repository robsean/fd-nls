Acerca de FreeDOS:

  FreeDOS es un sistema operativo libre compatible con DOS para sistemas
  compatibles con IBM-PC, compuesto por muchos programas diferentes y
  separados que actúan como "paquetes" para el proyecto global de FreeDOS.
  FreeDOS tiene como objetivo ser completo, libre, 100% compatible con el
  sistema operativo MS-DOS (en su mayoría logrado excepto conpatibilidad
  con Windows, el modo estandar de Windows trabaja con FreeDOS, pero en el
  modo 386 / WfW 3.11 no lo es). Una breve lista de cosas posibles con
  FreeDOS hoy:
     * Facilidad de arranque múltiple con Win95-2003 y NT/XP/ME.
     * Sistema de archivos FAT32 y soporte de disco grande (LBA).
     * Soporte LFN (con varias herramientas, y FREECOM (COMMAND.COM)).
     * LBACACHE - caché de disco (discos duros en modos CHS y LBA, 
       disquetes).
     * Administradores de memoria: (HIMEM), (EMM386), UMBPCI, HIMEMX, 
       JEMM386, JEMMEX.
     * SHSUCDX (reemplazo de MSCDEX) y controlador de CD-ROM (XCDROM.SYS)
       / UIDE.SYS.
     * CUTEMOUSE - controlador de ratón con soporte de rueda de 
       desplazamiento.
     * FDAPM - APM información/control/suspender/apagado, acelerador ACPI,
       equipo de ahorro de energía.
     * (XDMA.SYS) - (UDMA.SYS) - UIDE.SYS controlador para DOS: hasta 4 
       discos duros. UIDE.SYS también soporta SATA.
     * MPXPLAY.SYS - reproductor de medios entre ellos MP3, OGG, WMV...
       con AC97 integrados y controladores SB16.
     * 7zip, info-zip, zip y unzip... los archivadores modernos están
       disponibles para DOS.
     * EDIT / SETEDIT - editores de texto de múltiples ventanas.
     * HTMLHELP - visor de ayuda, puede leer la ayuda directamente desde
       un archivo zip.
     * PG - visor de texto de gran alcance (similar a LIST de V.D.Buerg)
     * Muchos programas de modo de texto portan desde Linux gracias a 
       DJGPP.
     * GRAPHICS - Impresión en grises en impresoras ESC/P, HP PCL y 
       PostScript.
     * Soporte NLS: ayuda ("/?") a los comandos se puede mostrar en 
       diferentes idiomas (no en todos los comandos).
     * etc.
  FreeDOS no tiene interfaz gráfica de usuario (GUI). Si desea apagar el
  ordenador, simplemente cierre el programa con el que está trabajando y
  pulse el botón "apagar".

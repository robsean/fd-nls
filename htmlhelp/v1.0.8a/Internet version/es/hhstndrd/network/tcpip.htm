Redes de FreeDOS - TCP/IP

  El Protocolo de Control de Transmisión (TCP) y el Protocolo de Internet
  (IP) es ahora el protocolo más utilizado para conectarse a una red.
  TCP/IP fue ideado como un estándar abierto mediante el sistema de RFC,
  que se utiliza para conectarse a Internet. De los principales
  protocolos en uso, TCP/IP es el único que fue originalmente diseñado
  para ser utilizado a través de redes de área amplia y no sólo en
  las LAN. Ha habido un número de implementaciones de la pila de TCP/IP
  para DOS (ya que no fue estandarizado por el sistema operativo). El
  más exitoso, ya que todavía está en uso, es WatTCP (Waterloo
  TCP/IP Kernel) disponible de Erick Engelke. Es de uso gratuito, pero
  el manual del programador debe ser comprado. Una versión de 32 bits
  de WatTCP también est disponible.
  La pila KA9Q TCP/IP se utiliza en otras aplicaciones, pero se encuentra
  en la mayor parte de las diversas versiones del programa NOS NET.EXE,
  que incluye aplicaciones (ftp, smtp, etc.) como parte del programa.
  La mayoría de las pilas TCP/IP restantes no tenían ningún apoyo de
  terceros, por lo que cuando los programadores originales dejaron de
  desarrollar, todo el desarrollo de aplicaciones se detuvo también.
  Estos incluyen el MIT/CMU pcip, Trompeta TCP/IP, TinyTCP y pilas
  FreePascal TCP/IP. En menor medida se incluyen también las pilas
  de las demás soluciones comerciales Microsoft y Novell TCP/IP.

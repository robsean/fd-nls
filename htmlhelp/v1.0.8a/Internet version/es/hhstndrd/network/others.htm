Redes de FreeDOS - Otros

  Mientras que las categorías mencionadas acerca de la creación de redes
  sistemas DOS debe cubrir la mayoría de los programas, hay algunas
  excepciones. Uno de ellos es "LAN Invisible", (ver:
  http://www.invisiblesoft.com/invlan/index.shtml) una aplicación de DOS
  NetWare como que incluso viene con un protocolo propio llamado
  "TransBIOS".
  Otra solución especial para conseguir un rápido acceso a la red y estar
  corriendo discos de arranque de red. Estos son capaces de trabajar con
  una amplia gama de hardware y utilizar varios protocolos. Discos de
  arranque se utilizan sobre todo para la copia de seguridad, restaurar
  y reparar. Pueden ser de gran ayuda.
  Ejemplos son:
    * Bart's Network Boot Disk (see: http://www.nu2.nu/bootdisk/
      network/ (*01))
    * Drivesnapshot (see: http://drivesnapshot.de/en/index.htm (*02))
    * Netbootdisk (see: http://www.netbootdisk.com/ (*03))
  La situación legal es lamentable: Algunos paquetes parecen incluir
  archivos de cliente MS, mientras que la licencia de Microsoft no
  permite dicha distribución.

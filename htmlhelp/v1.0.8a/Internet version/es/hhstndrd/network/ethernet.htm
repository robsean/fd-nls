Redes de FreeDOS - Ethernet

  Ethernet es una familia de tecnologías de red de ordenador basados en
  tramas para redes de área local (LAN). El nombre viene del concepto
  físico del éter. Tarjetas de interfaz de red (NIC), como 3Com y otros,
  se pueden encontrar para variar las diferentes redes. El más común es
  Ethernet (en diversos gustos) Token Ring y ARCNET. NIC son más comúnmente
  disponible para PCI, pero las antiguas tarjetas ISA todavía pueden estar
  disponibles. La mayoría de los PC recientes disponen de interfaces de
  red integradas en la placa base. Laptops normalmente también soportan
  redes inalámbricas o WiFi. La conexión Ethernet es el método más fácil
  de utilizar ADSL a través de un módem o router. Aunque puede ser posible
  utilizar un controlador USB sería más difícil.
  Si se decide Ethernet, necesitará un controlador compatible, como
  Crynwr, ODI o NDIS. El protocolo más común es TCP/IP, pero muchas redes
  también apoyan IPX/SPX. Para compartir datos con otros sistemas, tendrá
  que NFS o CIFS.

Redes de FreeDOS - Instalación

  La mayoría de aplicaciones TCP/IP para DOS utiliza las interfaz de
  controladores de paquetes. Así que para ser capaz de ejecutarlos,
  nosotros necesitamos instalar los controladores apropiados primero.
  Para instalar un controlador de paquetes genérico es la forma más
  sencilla. Pero esto no es posible algunas veces, como las tarjetas de
  red modernasa menudo solo vienen con controladores ODI y NDIS. De todos
  modos: controlodor ODI o NDIS puede también ser usado como controlador
  de paquetes a través de convertidores especiales ("shims"). Esto es
  sólo un poco más complicado.
  Los siguientes capítulos describen donde obtener contoladores y
  convertidores y como configurarlos:

    * [Instalación de controladores ODI]

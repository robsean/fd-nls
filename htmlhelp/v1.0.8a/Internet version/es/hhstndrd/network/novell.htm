  se puede descargar desde el sitio de Novell y utilizar de forma
  gratuita:

  Novell, former market leader for local networks, provided "LAN Work
  Place for DOS" which also included a TCP/IP protocol stack and
  numerous programs. Some software for DOS can still be downloaded
  from Novell's site and used for free:
    * El NetWare ODI Client 16 Driver para MS-DOS (1996)
      (http://www.novell.com/coolsolutions/tools/13555.html (*01))
      y el NetWare Client 32-Driver (1998)
      (http://www.novell.com/coolsolutions/tools/downloads/dw271e.zip
      (*02)) permiten a los usuarios de DOS conectarse a un servidor
      de Novell NetWare.
    * El nucleo TCP/IP de Novell "TCP16.EXE" (1996)
      (http://bbs.actapricot.org/files/area37/tcp16.exe (*03)) todavía
      se puede utilizar para ejecutar aplicaciones TCP/IP directamente
      sobre ODI (véase el capítulo Otros). En los sitios
      anteriores de Novell incluye una advertencia de que este software
      no es ni mantiene tampoco apoyado por más tiempo.
  Datos útiles sobre clientes DOS de Novell todavía se puede encontrar
  en el grupo de noticias "novell.support.os.client.dos-win3x"
  (http://groups.google.it/group/novell.support.os.client.dos-win3x
  (*04)) que estuvo activa hasta 2004, más reciente información y
  enlaces útiles se pueden encontrar en "Soluciones de DOS"
  (http://www.dossolutions.pwp.blueyonder.co.uk/network.htm#nware
  (*05)).

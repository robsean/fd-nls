Comando: fcbs

  FCBS establece el número de bloques de control de archivos.
  FCBS es un comando de CONFIG.SYS / FDCONFIG.SYS.

Sintaxis:

  fcbs=nnn

Opciones:

  nnn    El número de bloques de control de archivos, en el rango 1-255.
         El valor predeterminado es 4.

Comentarios:

  FCBS establece el número de bloques de control de archivos para reservar
  espacio.
  Como los bloques de control de archivos han sido reemplazados por
  identificadores de archivo (ver ARCHIVOS) y FreeDOS simula dinámicamente
  FCBS a partir de los datos del identificador según sea necesario, FreeDOS
  simplemente ignora nnn.
  FCBS es un comando interno de kernel.sys y no necesita ningún otro 
  archivo para funcionar.

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    fcbs=4

Comando: stacks

  STACKS agrega pilas para manejar las interrupciones de hardware.
  STACKS es un comando de CONFIG.SYS / FDCONFIG.SYS.

Sintaxis:

  stacks=nn,mmm

Opciones:

  nn   El número de pilas en el rango de 8-64 y 0.
  mmm  El tamaño de cada pila, en bytes, en el rango 32-512.

Comentarios:

  Este comando no es necesario en instalaciones estándar. STACKS cambiar
  el número y el tamaño de las pilas de interrupción de hardware 
  disponibles. nn es el número de pilas diferentes y mmm es el tamaño en
  bytes de cada una. En algunos casos, puede usar "stacks=0,0" para usar
  solo pilas estándar en lugar de permitir que DOS asigne pilas
  adicionales para interrupciones de hardware. 
  STACKS es un comando interno de kernel.sys y no necesita ningún otro
  archivo para funcionar.

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    stacks=16,256

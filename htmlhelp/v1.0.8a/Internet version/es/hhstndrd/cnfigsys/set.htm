Comando: set (comando config.sys)

  SET establece la variable de entorno en el valor proporcionado.
  SET es un comando de CONFIG.SYS / FDCONFIG.SYS.

Sintaxis:

  SET [variable[=[cadena]]]
      variable  Especifica el nombre de la variable de entorno.
      cadena    Especifica una serie de caracteres para asignar la 
                variable.

Opciones:

  - ninguna -

Comentarios:

  SET establece la variable de entorno en el valor proporcionado.
  SET es un comando interno de kernel.sys y no necesita ningún otro
  archivo para funcionar. También existe un comando SET en COMMAND.COM.

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    set home=C:\FDOS\bin

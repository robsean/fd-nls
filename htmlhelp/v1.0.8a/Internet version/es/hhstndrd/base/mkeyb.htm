Comando: mkeyb

  MKEYB es un controlador de teclado compacto.

Sintaxis:

  mkeyb [opciones] código
  keyb [opciones] código

  Vuelve a asignar las claves según sea necesario en ciertos países.
  Utiliza aproximadamente 500 bytes (según el país).
  Localizaciones

  Código  Ubicación               Página de Códigos Requisito
  ===========================================================
   BE     Bélgica
   BR     Brasil
   BX     Brasil (extendido)
   DK     Dinamarca
   FR     Francia
   GR     Alemania
   GRC    Alemania (con COMBIs)
   HE     Hebreo
   IT     Italia
   LA     América Latina
   NL     Países bajos
   NO     Noruega
   PL     Polonia
   PO     Portugal
   RU     Rusia                          866
   SF     Suizo (Francés)
   SG     Suizo (Alemán)
   SP     España                          850
   SU     Finlandia
   SV     Suecia
   UK     Reino Unido

Opciones:

  /l  Lista todos los declados disponibles.
  /u  Descarga el controlador del teclado.
  /t  Prueba un controlador de teclado específico, no seas residente.
  /?  Muestra la ayuda.

Comentarios:

  MKEYB (KEYB) se carga automáticamente lo más alto posible
  - por lo tanto, no es necesario utilizar LH / LOADHIGH.
  MKEYB (KEYB) viene con el paquete MKEYB040.

Ejemplos:

  mkeyb GR     carga el controlador de teclado alemán.
  mkeyb /u     descarga el controlador del teclado.
  mkeyb GR /t  prueba el controlador de teclado GERman, no seas 
               residente

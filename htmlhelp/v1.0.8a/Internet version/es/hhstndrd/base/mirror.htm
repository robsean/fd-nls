Comando: mirror

  MIRROR registra información sobre el disco, como el sector de
  arranque, la tabla de asignación de archivos y el directorio raíz para
  una posible recuperación de datos.

Sintaxis:

  MIRROR [unidad]
  MIRROR [/PARTN]
  MIRROR [/?]

Opciones:

  unidad  Guarda información UNFORMAT en una letra de unidad, p. ej. X:
  /PARTN  Guarda las tablas de partición a:\partnsav.fil para su 
          recuperación futura a través de UNFORMAT / PARTN.
  /?      Muestra la ayuda.

Comentarios:

  MIRROR almacena una copia del sector de arranque, las tablas FAT y el
  directorio raíz en una "ubicación de almacenamiento" cerca del final
  del disco. Al usar el modificador /partn con el comando mirror, puede
  guardar una copia de seguridad de las tablas de partición en
  partnsav.fil en la unidad a:.
  Nota: La imagen espejo creada por este programa solo se puede recuperar
  utilizando el programa UNFORMAT para FreeDOS.

Ejemplos:

  mirror d:
  mirror /partn

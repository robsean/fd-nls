Comando: keyman

  KEYMAN es un grabador de macros simple que se instala como un programa
  residente en memoria que permite una (re)definición simple de claves.

Sintaxis:

 [loadhigh] keyman

Opciones:

  - ninguna -

Comentarios:

  KEYMAN solo se puede cargar si XKEYB ya reside en la memoria.
  Se activa mediante una tecla de acceso rápido (F11 por defecto).
  Después de la activación, KEYMAN abrirá una ventana en la pantalla.
  Ahora puede presionar una tecla a la que desea adignar un nuevo macro
  o presionar la tecla de acceso rápido nuevamente para redefinirla. Si
  presionó una tecla que ya tenía una macro asignada, la macro actual se
  imprimirá en la ventana. Esta macro se puede editar usando la tecla
  RETROCESO para borrar las teclas que se muestran (para incluir una
  pulsación de RETROCESO como parte de la macro, presione ALT y "8" en el
  teclado numérico). Después de presionar nuevamente la tecla de acceso
  rápido, se asignará la macro y la ventana desaparecerá.
  NOTA:
  Si la tecla que presionó NO tenía una macro asignada, esa tecla se
  configurará de manera predeterminada como la primera pulsación de tecla
  de la macro (es decir, si presionó CTRL-D, la macro consistirá
  inicialmente en la pulsación de tecla "CTRL-D"). Si la nueva macro
  consistía en más de un carácter, se creará automáticamente una XString
  y se asignará a la tecla (si se asignó suficiente espacio XString a
  través del modificador /Xnn cuando se instaló XKEYB). Si presionó la
  tecla de acceso rápido después de la activación de KEYMAN, KEYMAN le
  pedirá que presione la nueva tecla de acceso rápido. Después de
  presionar una tecla, la ventana desaparecerá y la tecla presionada se
  convertirá en la nueva tecla de acceso rápido. Si desea guardar una
  asignación de clave creada interactivamente, puede hacerlo utilizando
  el comando LISTXDEF > xyz.KEY KEYMAN viene con el paquete XKEYB.

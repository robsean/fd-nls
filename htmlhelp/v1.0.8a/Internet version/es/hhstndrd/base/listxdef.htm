Comando: listxdef

  LISTXDEF enumera todas las asignaciones de teclas XKeyb actuales a
  STDOUT.

Sintaxis:

  listxdef

Opciones:

  - ninguna -

Comentarios:

  LISTXDEF se usa para generar (en STDOUT) el archivo de definición que
  XKEYB usa actualmente (incluidas las teclas definidas con KEYMAN). Se
  usa más comúnmente para guardar un archivo de definición personalizado
  (es decir, uno creado al cargar uno de los archivos de definición
  estándar con XKEYB y luego asignar macros adicionales usando KEYMAN).
  Esto se hace usando el comando:
    KEYMAN > xyz.KEY
  Consulte el manual XKEYB.TXT para obtener más información.
  COMANDOS RELACIONADOS:
    XKEYB    Versión principal de XKEYB.
    XKEYBRES Versión "Lite" de XKEYB.
    KEYMAN   Grabador de macros XKEYB.
  LISTXDEF viene con el paquete XKEYB.

Ejemplos:

  listxdef > myfile.key guarda todas sus definiciones de macro.

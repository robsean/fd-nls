Comando: fasthelp (fdhelp)

  FASTHELP es un clon de ayuda rápida para FreeDOS. Muestra ayuda sobre
  el tema, estilo de páginas de manual de Linux.

Sintaxis:

  fasthelp [tema]

Opciones:

  tema          El tema sobre el que desea información.
    Medio ambiente:
      PAGER:    programa de buscapersonas para mostar un archivo de
                texto.
      HELPPATH: Directorio que contiene sus archivos de ayuda.
      LANG:     Idioma preferido para sus archivos de ayuda.

Comentarios:

  Utilice las variables de entorno HELPPATH y LANG para especificar
  la ubicación y el idioma de los archivos de ayuda, respectivamente.
  La ruta predeterminada es ../help y el idioma predeterminado es EN
  (inglés).
  También está la variable de entorno PAGER, que se puede usar para
  configurar un programa alternativo para mostrar archivos de ayuda de
  texto. Las llamadas a fasthelp desviarán los archivos de texto al
  programa PAGER. Consulte SET para obtener más información sobre cómo
  configurar las variables de entorno.
  Fasthelp también se llama "FSUITE" o "FSUITE04" o "FS04de" (alemán) o
  "FSUITE04de" (alemán).

Ejemplos:

  fasthelp       Muestra la lista de temas
  fasthelp edit  Muestra información sobre la edición

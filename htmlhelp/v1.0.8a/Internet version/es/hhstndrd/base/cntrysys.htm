Comando: country.sys

  COUNTRY.SYS es el archivo de ayuda de donde COUNTRY obtiene su 
  información. COUNTRY.SYS debe cargarse en CONFIG.SYS / FDCONFIG.SYS.
  Cuando FreeDOS ya se está ejecutando, puede cargar COUNTRY.SYS más
  tarde con DEVLOAD.

Sintaxis:

  - ninguna -

Opciones:

  - ninguna -

Comentarios:

  COUNTRY habilita / establece características internacionales de DOS
  seleccionando un código de país y (solo con el kernel inestable /
  desarrollador) opcionalmente también una página de códigos y un archivo
  de país. El kernel estable solo usa el valor nnn. Para soporte
  completo, use el kernel inestable / desarrollador. También necesita un
  archivo COUNTRY.SYS para este último, mientras que el primero ignora el
  argumento del nombre de archivo y solo establece los formatos de 
  fecha/hora/número, utlizando valores compilados en el núcleo para
  algunos países comunes. No establece el orden de clasificación ni las
  tablas de traducción en mayúsculas / minúsculas. NLSFUNC solo se puede
  usar con el kernel inestable. COUNTRY.SYS tiene información completa de
  NLS (COUNTRY, UCASE, LCASE, FUCASE, COLLATE, DBCS y YESNO) para los
  siguientes países:

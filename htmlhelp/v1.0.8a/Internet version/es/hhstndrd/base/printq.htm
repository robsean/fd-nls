Comando: printq

  PRINTQ es un pequeño programa que utiliza el módulo prnspool, que es
  una interfaz para le programa DOS PRINT.COM

Sintaxis:

  - ninguna -

Opciones:

  - ninguna -

Comentarios:

  PRINT.COM debe instalarse antes.

Ejemplos:

  - ninguno -

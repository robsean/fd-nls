Comando: kernel

  El KERNEL (KERNEL.SYS) es el corazón de FreeDOS.
  FreeDOS tiene un KERNEL estable y uno en DESARROLLO (inestable).

Sintaxis:

  - ninguna -

Opciones:

  - ninguna -

Comentarios:

  El KERNEL es necesario para cargar controladores a través de CONFIG.SYS/
  FDCONFIG.SYS y para hacer que COMMAND.COM u otros shells se ejecuten en
  FreeDOS. Mientras que KERNEL carga CONFIG.SYS / FDCONFIG.SYS, SHELL 
  (por ejemplo, COMMAND.COM) carga AUTOEXEC.BAT.
  Para instalar el KERNEL en un disquete/HD, debe ejecutar el comando SYS.
  SYS compia el KERNEL en un lugar especial en el medio de arranque; de lo
  contrario, no se encuentra al arrancar y el sistema no funcionará. Por
  esta razón, KERNEL.SYS SIEMPRE debe estar en el directorio raíz del
  medio de arranque.
  Para fines estándares, utilice el KERNEL estable.

Ejemplos:

  - ninguno -

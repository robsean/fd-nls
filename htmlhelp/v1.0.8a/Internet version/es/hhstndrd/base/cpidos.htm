CPIDOS - Paquete de archivos CPI (información de página de códigos)

v3.0 - páginas de códigos DOS

  Este paquete contiene archivos de información de página de códigos
  comprimidos UPX con páginas de códigos DOS, que son del tipo DISPLAY.
  La herramienta oficial de FreeDOS para prepararlos y seleccionarlos es
  MODE. Los diseños de teclado disponibles en KPDOS están destinados a
  usar los archivos CPI contenidos en este paquete. A continuación se
  muestra la lista de esos archivos y las páginas de códigos que
  contienen:

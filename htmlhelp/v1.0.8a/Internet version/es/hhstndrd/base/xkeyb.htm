Comando: xkeyb

  XKEYB es un controlador de teclado con cierto parecido con el 
  conocido KEYB.

Sintaxis:

  xkeyb archivo de configuración [/Xnn] [/U] [/Q]
    mientras que el archivo de configuración es el código del país
    ("GR.key" o "US.key")

Opciones:

  /Xnn  Este modificador se debe usar para establecer explícitamente la
        cantidad de espacio de almacenamiento que se reservará para 
        XStrings (donde nn es el tamaño de memoria indicado en bytes).
  /U    (Desinstalar), elimina la copia residente de XKeyb de la
        memoria (si está instalada).
  /Q    (Silencioso), suprime la salida del párrafo LIST en los archivos
        de configuración.

Comentarios:

  Al igual que con KEYB, XKEYB también es un controlador de teclado para
  computadoras MS-DOS, que se instala como un programa residente en
  memoria y se puede configurar para diferentes países utilizando
  parámetros. Sin embargo, XKEYB permite muchas más opciones de
  configuración que KEYB. Por lo tanto, puede configurar XKEYB no solo
  para diferentes países, sino también para sus propias preferiencias
  personales. Por un lado, XKEYB permite la asignación gratuita de todas
  las teclas en todos los niveles (normalmente con SHIFT, CTRL, ALT y
  CTRL-ALT), mientras que, por otro lado, ofrece la flexibilidad de
  definir cadenas de caracteres para teclas individuales en ciertos
  niveles. Además, para cada tecla se puede determinar por superado si
  está influenciado por BLOQ MAYUS, BLOQ NUM o SCROLL LOCK, y si las
  teclas SHIFT, CTRL y ALT determinan qué valores se emiten. El
  programador cuenta con un PI (interfaz de programa) para XKEYB. Permite
  el acceso a muchas de las funciones XKEYB.
  XKEYB viene con el paquete XKEYB.

Ejemplos:

  XKEYB GR /X480
  XKEYB /X480 GR.key

Comando: ctmouse (Cutemouse)

  CuteMouse (CTMOUSE) es el controlador de ratón de FreeDOS.

Sintaxis:

  ctmouse [opciones]
  LH ctmouse [opciones]
  LOADHIGH ctmouse [opciones]

Opciones:

  /3        Forzar el modo de 3 botones (solo para ratones Microsoft
            y PS/2).
  /B        Cancela la ejecución si los servicios del ratón ya están
            presentes.
  /L        Cambiar botón izquierdo y derecho.
  /M        Pruebe *antiguo* sistemas de ratón / Genius para ratones
            que no sean PnP.
  /N        Cargue CuteMouse como nuevo TSR, incluso si CuteMouse ya
            está cargado (útil para archivos por lotes que descargan
            CuteMouse al final).
  /O        Habilite la detección de rueda USB de PS2 y BIOS (podría
            bloquearse).
  /P        Fuerza el modo de ratón PS/2.
  /R[h[v]]  Resolución horizontal/vertical: h,v = 1-9, o 0 para auto.
            (sin valor = uso predeterminado: automático para h, o 
            "como h" para v).
  /S[c[i]]  Forzar el modo de ratón serie en el puerto COM c (c=1-4,
            el valor predeterminado es IRQ4 para COM 1/3 e IRQ3 para
            COM 2/4).
  /U        Desistalar controlador, eliminar TSR de la memoria.
  /V        Búsqueda inversa: encuentra PS/2 después del ratón serial.
  /Y        Fuerza la búsqueda en serie del ratón, pero, a diferencia
            de la opción /S no deshabilita la compatibilidad con PS/2.
  /W        No permita que CuteMouse se mueva a UMB.
  /?        Muestra la ayuda.

Comentarios:

  CTMOUSE no es compatible con ratones USB.
  CTMOUSE se puede cargar en CONFIG.SYS / FDCONFIG.SYS o en AUTOEXEC.BAT
  o en la línea de comando. CTMOUSE2 es una nueva ramificación de 
  CuteMouse que admite la rueda del ratón. Proporciona una rueda API
  (interfaz del programador de aplicaciones), para que los programas
  de FreeDOS puedan usar la rueda del ratón. Visite el sitio web de
  CuteMouse en http://cutemouse.sourceforge.net
  CTMOUSE tiene los siguientes EXITCODES (ERRORLEVEL):
    0  PS/2, ratón Microsoft o Logitech encontrados y CTMOUSE instalado;
       Descarga exitosa; /? opción utilizada.
    1  CTMOUSE instalado en modo ratón del sistema; La descarga falló
       porque CTMOUSE no se cargó.
    2  La parte residente cambió el modo PS/2, Microsoft o Logitech; La
       descarga falló porque las interrupciones del conductor fueron
       interceptadas.
    3  La parte residente cambió el modo ratón del sistema.
    4  Servicios de ratón ya presentes (devueltos solo para la opción /B)
    5  Ratón no encontrado; Opción inválida utilizada.

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    devicehigh=C:\fdos\bin\ctmouse  carga ctmouse en la memoria superior
  EN AUTOEXEC.BAT O EN LINEA DE COMANDOS:
    lh ctmouse  carga ctmouse en memoria superior
    ctmouse /u  descarga el controlador del ratón
    ctmouse /?  Muestra la ayuda (opciones)

Comando: format

  FORMAT formatea un disco duro o disquete. Esto prepara el medio para
  el uso con FreeDOS.

Sintaxis:

  FORMAT unidad [/V[:etiqueta]] [/Q] [/U] [/F:tamaño] [/S] [/D]
  FORMAT unidad [/V[:etiqueta]] [/Q] [/U] [/T:pistas /N:sectores][/S][/D]
  FORMAT unidad [/V[:etiqueta]] [/Q] [/U] [/4] [/S] [/D]
  FORMAT [/?]
         unidad  La letra de unidad, p. ej. C:

Opciones:

  /1            Formatea un disco de una cara (160k/180k).
  /4            Formatee un disquete de 160k/360k en una unidad de
                disquete de 1.2 MB.
  /8            Formatee un disco de 5-1/4 pulgadas con 8 sectores por
                pista (160k/320k, DOS 1.x).
  /A            Fuerza la alineación 4k para el formato FAT32.
  /B            Reserve espacio para hacer un disco de arranque (es
                ficticio y no se puede combinar con /s (sys).
  /D            Sea muy detallado y muestre resultados de depuración.
                Para informes de errores.
  /F:tamaño     Especifica el tamaño del disquete a formatear. Los
                tamaños normales son: 360, 720, 1200, 1440 o 2880 
                (unidad: kiloBytes). 160,180,320 también deberían
                funcionar. /F:0 muestra una lista de tamaños permitidos.
  /N:sectores   Especifica el número de sectores por pista en un disquete.
  /Q            Formatos rápidos del disco. Si no se combina con /U, se
                puede UNFORMAT y conserva las marcas de clúster 
                incorrectas (/Q /U no estan).
  /S            Llama a SYS para que el disco sea de arranque y para
                agregar archivos del sistema.
  /T:pistas     Especifica el número de pistas en un disquete.
  /U            Formatea incondicionalmente el disco. Formato de bajo
                nivel si disquete.
                NOTA: ¡Esta opción puede durar MUCHO MUCHO tiempo!
                Presione ESC y espere para salir /U.
  /V:etiqueta   Especifica una etiqueta de volumen para el disco,
                almacena la fecha y la hora del mismo (no es para discos
                de 160k/320k).
  /Y            Omitir solicitud de confirmación.
  /Z:ayuda_larga:
                Proporciona información detallada sobre el uso técnico.
  /Z:espejo     Guarda los "datos de MIRROR" UNFORMAT, sin formatear el
                disco.
  /Z:en_serio   Suprime la solicitud de confirmación cuando usa el
                formato con un disco duro.
  /Z:sin_formato
                Desformatea el disco, si los datos del MIRROR están
                actualizados.
  /?            Muestra la ayuda.

Comentarios:

  Advertencia: si bien el formato es compatible con FAT32, ejecutar bajo
  Win9x para crear FAT32 no funciona bien.
  Los modos para DISQUETE son: intenta utilizar el formato rápido y seguro.
  Use el formato de bajo nivel solo cuando sea necesario. El formato
  rápido y seguro guarda los datos mirror para UNFORMAT. Los modos para
  HARDDISK son: Intenta utilizar un formato rápido y seguro. Utilice el
  formato completo rápido solo restablece el sistema de archivos. Si desea
  forzar el formato de bajo nivel (disquete) o desea escanear toda la
  superficie del disco y borrar todo el contenido (disco duro), use /U.
  FORMAT /Q /U es un formato completo rápido (¡sin formato / escaneo /
  borrado de bajo nivel!)
  FORMAT /Q es un formato de guardado rápido (guarde los datos mirror si
  es posible) ¡Los datos mirror siempre sobrescribirán el final del área
  de datos! FORMAT selecciona automáticamente un modo (ver arriba) si no
  selecciona /Q ni /U.

  Los tipos de FAT admitidos son: FAT12, FAT16, FAT32, todos con mirror
  / unformat. Los tamaños de disquete admitidos son: 160k, 180k, 320k
  y 1200k para 5,25 pulgadas y 720k y 1440k (2880k nunca probados hasta
  ahora) para unidades de 3,5 pulgadas. Los sobreformados admiticos son
  400k, 800k, 1680k (y 3660k) con más sectores y 1494k (en lugar de 
  1200k) y 1743k (y 3486k) con más pistas también. Más pistas no
  funcionarán en todas las unidades, utilícelas bajo su propio riesgo.

  Advertencia: las versiones anteriores de DOS solo pueden usar
  sobreformatos con un controlador. 720k en 1440k necesita 720k de 
  medios. Use una unidad de 360k para formatear 360k. Para el formato
  FAT32, puede usar el modificador /A para forzar la alineación 4k.
  FORMAT tiene los siguientes EXITCODES (ERRORLEVEL):
    0        Bien
    1        Sectores defectuosos detectados (solo si /d o /y o /z:
             opción seriamente activa)
    2        Pantalla de ayuda mostrada o error de sintaxis
    3        Abortado por Usuario
    4        Error de formateo
    5        El usuario no confirmó
  Los niveles de error 1 y 2 son extensiones más allá del estilo MS 
  FORMAT.
  Si usa el indicador /D, los niveles de error serán diferentes.
  Serán específicos para la versión de estilo de FORMAT.

  En 0.91t, por ejemplo:
    0        Bien
    1        Sectores defectuosos detectados
    2        Ayuda mostrada o error de sintaxis
    10..13   Error de usuario (no confirmado / error de sintaxis)
    20..22   Error de Createfs (creación del sistema de archivos)
    25..29   dDrive I/O error,
    30..31   Error de grabadoBC (análisis de sistema de archivos
             existente)
    35..36   Error de formato (formato incondicional)
    38..39   Error de Savefs (SafeFormat/Espejo)
    40..52   Error de disquete
    55..59   Error de Disco Duro
    60..62   Error Principal: tipo de unidad inadecuado, p. ej. CD-ROM
   128..255  Error crítico (int de usuario: (código de error de DOS o 
             BIOS) | 128)

Ejemplos:

  format a: /f:1440   formatea un disquete de 1,4 MB
  format c: /v:my_hd  formatea un disco duro, etiqueta: my_hd

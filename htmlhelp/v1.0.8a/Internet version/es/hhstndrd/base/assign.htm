Comando: assign

  ASSIGN reasigna operaciones de disco de una unidad a otra.

Sintaxis:

  ASSIGN [/SHADOW] [/STATUS] [/?]
  ASSIGN [/SHADOW] [/M] [/U] [{unidad1[:]=unidad2[:]}]
         unidad1:  letra de unidad que se reemplazará, por ejemplo C:
         unidad2:  letra de unidad a la que apunta, por ejemplo D:

Opciones:

  /SHADOW  Hace que la presencia de ASSIGH en la memoria sea difícil
           de detectar. Los usos posteriores de ASSIGH emitirán una
           advertencia si la opción /SHADOW no se usa (pero aún así
           realiza el comando).
  /STATUS  Muestra todas las letras de unidad asignadas. Si ASSIGN aún
           no está cargado, un mensaje lo refleja, pero ASSIGN no está
           cargado.
  /M       Intenta utilizar la memoria superior, en lugar de la memoria
           convencional, para almacenar el ASSIGN.
  /U       Desistala un ASSIGN instalada previamente.
  /?       Muestra la ayuda.

Comentarios:

  Los caracteres ':' son opcionales, entre las letras de unidad y el
  signo igual '=' no se permiten espacios en blanco. Si no se nombran
  unidades y no se usa /SHADOW o /U, se asigna reinstalar (borrando todas
  las asignaciones de unidades anteriores). ASSIGN tiene los siguientes
  EXITCODES (ERRORLEVEL):
    0 éxito/ayuda
    3 abortado por el usuario (por ejemplo, Ctrl-C)
    4 error de sintaxis
  No utilice asignar en combinación con los siguientes comandos:
    diskcomp
    diskcopy
    format
    label
    print
    restore
    subst

Ejemplos:

  assign B:=A:
  assign B=A

Ver también:

  join
  subst
  swsubst
  truename

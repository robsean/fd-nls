Comando: cdd

  CDD muestra el nombre o cambia el directorio actual y la unidad.

Sintaxis:

  CDD [unidad][ruta]
  CDD [..][/?]
      unidad  La letra de la unidad, p.ej. C:
      ruta    El directorio, por ejemplo \ejemplo\

Opciones:

  ..  Especifica que se desea cambiar al directorio principal.
  -   Si la característica "último directorio" está habilitada, cambie al
      último directorio.
  /?  Muestra la ayuda.

Comentarios:

  Si se especifica unidad: se cambia la unidad de trabajo actual; esto es
  lo único diferente de la orden "CHDIR"
  Escriba CDD sin parámetros para visualizar la unidad y el directorio 
  actuales.
  CDD es un comando interno de command.com y no necesita otro archivo para
  que funcione.

Ejemplos:

    cdd c:\fdos     Cambia al directorio c:\fdos
    cdd ..          Cambia de c:\fdos a c:\
    cdd \           Cambia de c:\anydirectory\anysubdirectory a c:\
  Si estás en c:\ :
    cdd fdos        Cambia al directorio c:\fdos
    cdd bin         Cambia al directorio c:\fdos\bin
    cdd d:\example  Cambia el directorio en d: a d:\example, Y cambia
                    la letra de la unidad para manejar d:

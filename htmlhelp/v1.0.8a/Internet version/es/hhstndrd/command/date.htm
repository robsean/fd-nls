Comando: date

  DATE muestra o establece la fecha.

Sintaxis:

  DATE [nueva_fecha]
  DATE [/D] [/?]
       Escriba DATE sin parámetros para mostrar la configuración de fecha
       actual y un aviso para uno nuevo. Presione ENTER para mantener la
       misma fecha.

Opciones:

  nueva_fecha  mm-dd-yy o mm-dd-yyyy
               mm/dd/yy o mm/dd/yyyy
               mm.dd.yy o mm.dd.yyyy
               La nueva fecha para tu sistema.
               Valores para el día (dd), mes (mm) y año (aa o aaaa) pueden
               estar separados por puntos, guiones o barras. Se puede usar
               el año de 4 o 2 dígitos.
  /D       Muestra la fecha sin solicitar una nueva configuración.
  /?       Muestra la ayuda.

Comentarios:

  Si se especifica un año de 2 dígitos, se supone los años de 00 a 79 para
  simplificar 2000-2079. Se supone que los años 80-99 significan 1980-1999.
  Cuando se le solicite una nueva hora, presione intro para aceptar la hora
  actual. Configurando sin cambiarlo.
  DATE es un comando interno de command.com y n necesita otro archivo para
  que funcione.

Ejemplos:

  date 04-22-2007 o date 04-22-07
  date 04/22/2007 o date 04/22/07
  date 04.22.2007 o date 04.22.07

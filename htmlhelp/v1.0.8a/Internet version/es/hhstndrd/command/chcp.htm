Comando: chcp

  CHCP muestra o establece el número de página de códigos activo.

Sintaxis:

  CHCP [nnn] [/?]

Opciones:

  nnn  Especifica el número de página de códigos.
       437  US
       850  Multi (Latín I)
       852  Eslavo (Latín II)
       860  Portugués
       863  Canadiense-Francés
       865  Nórdico
  /?   Muestra la ayuda.

Comentarios:

  Escriba CHCP sin un parámetro para mostrar el número de página de
  códigos activo. Cualquier programa que comenzó antes de que se asignara
  la nueva página de códigos pude continuar usando la página de códigos
  original. (Esto excluye COMMAND.COM, que será cnsciente de los cambios
  en la página de códigos).
  Para usar CHCP:
    1. especifique la ubicación de COUNTRY.SYS con el comando COUNTRY.
    2. cargue el programa NLSFUNC (usando INSTALL).
    3. use CHCP en cualquier momento para cambiar la página de códigos.
  CHCP es un comando interno de command.com y no necesita ningún otro
  archivo para que funcione.

Ejemplos:

  CHCP      Muestra el número de página de códigos activo
  CHCP /?   Muestra la ayuda
  CHCP 850  Establece la página de códigos número 850

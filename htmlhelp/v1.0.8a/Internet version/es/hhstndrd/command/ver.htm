Comando: ver

  VER Muestra la información de la versión de FreeDOS. De forma
  predeterminada, sólo se notifica la versión del shell de comandos
  (FreeCOM).

Sintaxis:

  VER [/R] [/W] [/D] [/C]

Opciones:

  /R  Muestra los números de versión del núcleo de DOS.
  /W  Muestra la garantía del shell d comandos de FreeDOS.
  /D  Muestra la infromación de distribución del shell de comandos de
      FreeDOS.
  /C  La sección de confirmaciones de FreeCOM. Enumera los colaboradores.

Comentarios:

  VER es un comando interno de command.com y no necesita otro archivo para
  poder ejecutarse.

Ejemplos:

  ver /R
  ver /W
  ver /D
  ver /C

Ver tamvién:

  callver
 (setver)

Comando: path

  PATH muestra o establece una ruta de búsqueda para archivos ejecutables.
  PATH es un comando para Archivo por lotes / AUTOEXEC.BAT.
  También se puede utilizar en la línea de comandos.

Sintaxis:

  PATH [[unidad]ruta[;...]]
  PATH [lista_rutas]
  PATH ;
  PATH [/?]

  unidad    la letra de la unidad, por ejemplo C:
  ruta      El directorio, por ejemplo \ejemplo\

Opciones:

  lista_rutas  Establece la lista de rutas. Separa las rutas con punto y
               coma (;).
  ;            Borra todas las configuraciones de la ruta de búsqueda y
               dirige el comando shell para buscar solo en el directorio
               actual.
  /?           Muestra la ayuda.

Comentarios:

  Escriba "PATH ;" para borrar todos los ajustes de la ruta de búsqueda y
  dirigir el comando shell para buscar solo en el directorio actual.
  Escriba "PATH" sin parámetros para mostrar la ruta actual.
  Algunos comandos son internos a FreeDOS. Sin embargo, muchos están en el
  \bin o el directorio \freedos\bin, por lo que puede desear tener este
  directorio en la lista de rutas (como en el primer ejemplo a 
  continuación).
  Las rutas de los programas de uso frecuente se pueden colocar en la 
  lista de rutas por conveniencia.
  Cuando se ingresa un programa o comando externo en el símbolo del
  sistema, FreeDOS lo busca en el directorio actual y luego un cada uno de
  las rutas especificadas por el comando path.
  Si el mismo nombre de programa está en directorios de ruta diferentes,
  el programa se iniciará desde la ruta que se menciona por primera vez en
  la lista de rutas.
  PATH también se puede utilizar fuera de un archivo por lotes.
  PATH es un comando interno de command.com y no necesita otro archivo
  para que funcione.

Ejemplos:

  EN UN ARCHIVO .BAT / AUTOEXEC.BAT:
    path c:\fdos\bin;c:\util
    path %path%;c:\games
    El segundo ejemplo agrega c:\games a la lista existente, preservando
    el previamente establecido en PATH.

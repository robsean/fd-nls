Comment utiliser la souris dans l'aide :

  FreeDOS utilise le pilote de souris CTMOUSE (Cutemouse).
  * Les liens des sujets sont surlignés en jaune (dans l'aide de FreeDOS).
    Sur Internet ils sont marqués comme d'habitude.
  * Cliquez sur un lien pour voir davantage sur ce sujet.
  * Cliquez sur l'ascenseur pour faire défiler la page vers le haut
    ou le bas.
  * Quittez l'aide en cliquant sur "Sortie" en haut de l'écran. (dans
    l'aide de FreeDOS)
  * Davantage d'options de navigation fournies en bas de l'écran.

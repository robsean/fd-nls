KPDOS v3.0 - Paquet de dispositions de clavier pour les pages de code DOS

  KPDOS contient les fichiers de bibliothèques à utiliser avec KEYB.
  L'outil officiel de FreeDOS pour faire fonctionner les fichiers
  ci-dessous avec l'une de ses pages de codes possibles est MODE.
  Pour utiliser MODE, DISPLAY doit déjà fonctionner.
  Tous les fichiers CPI sur cette liste sont ceux qui sont disponibles
  dans le paquet CPIDOS. Les fichiers sources peuvent être compilés
  avec KC, qui, à son tour peut être inclus dans les fichiers de
  bibliothèques personnalisées avec KLIB.
  Davantage de détails sont disponibles dans le paquet KC, qui inclut
  également KLIB.

  Dispositions de clavier officielles de FreeDOS, v3.0
  ====================================================

  Ceci est la liste de tous les noms des dispositions de clavier
  disponibles pour FreeDOS. Lorsque le nom est au format zz999, cela
  signfie qu'il existe un nom ET un identifiant qui DOIT être spécifié,
  sinon KEYB chargera la disposition d'un autre clavier pour ce pays.

  Dans ce cas, l'utilisateur devra exécuter : "KEYB ZZ /ID:999".

  Lorsque le nom est au format "aa, bb", cela signifie qu'il existe
  en fait (au moins) deux noms qui peuvent être utilisés pour charger
  la disposition de ce pays/cette langue. (Vous pouvez trouver
  d'autres noms et/ou identifiants dans la section [GENERAL] des
  fichiers KEY.)

  Veuillez vous référer à la section Remarques à la fin de ce fichier
  chaque foi qu'il y a un numéro à droite du nom de votre clavier.

  Utilisation de claviers à double script
  =======================================

  Il est nécessaire d'appuyer sur une combinaison de touches pour
  accèder à la couche non latine lors de l'utilisation de claviers
  destinés (ou qui ont été améliorés pour) utiliser plus d'un
  alphabet/script. Veuillez vous référer aux tableaux ci-dessous.
  Claviers à double script - arabe, arménien, azéri, biélorusse,
  bulgare, cherokee, géorgien, grec, hébreu, kazakh, kirghiz,
  macédonien, mongole*, russe, serbe, tadjik, ukrainien et ouzbek :

      Description de la couche     Combinaison de touches
      ------------------------     ----------------------
      Latin                        [ALT]+[MAJ_GAUCHE]
      Alphabet national            [ALT]+[MAJ_DROITE]

  *Alphabet cyrillique.

  Claviers italiens, lors de l'utilisation de cp869 (grec) :

      Description de la couche     Combinaison de touches
      ------------------------     ----------------------
      Latin                        [ALT]+[MAJ_GAUCHE]
      Grec phonétique              [ALT]+[MAJ_DROITE]

  Clavier letton lv455 :
  (Couche cyrillique disponible lors de l'utilisation de cp3012)

      Description de la couche     Combinaison de touches
      ------------------------     ----------------------
      QWERTY latin                 [ALT]+[MAJ_GAUCHE]
      UGJRMV latin                 [ALT]+[MAJ_DROITE]
      Cyrillique phonétique        [ALT]+[CTRL_GAUCHE]

  Clavier lituanien lt456 :
  (Couche cyrillique disponible lors de l'utilisation de cp771 ou cp772)

      Description de la couche     Combinaison de touches
      ------------------------     ----------------------
      QWERTY latin                 [ALT]+[MAJ_GAUCHE]
      AZERTY latin                 [ALT]+[MAJ_DROITE]
      Cyrillique phonétique        [ALT]+[CTRL_GAUCHE]

  Claviers norvégien et suédois/finlandais, lors de l'utilisation
  de cp30000 : (cp30000 : langues sames)

      Description de la couche     Combinaison de touches
      ------------------------     ----------------------
      QWERTY latin                 [ALT]+[MAJ_GAUCHE]
      ASERTT latin                 [ALT]+[MAJ_DROITE]

  Clavier américain, lors de l'utilisation de cp30034 :

      Description de la couche     Combinaison de touches
      ------------------------     ----------------------
      Latin                        [ALT]+[MAJ_GAUCHE]
      Cherokee                     [ALT]+[MAJ_DROITE]

  Clavier turkmène, lors de l'utilisation de cp59234(*) :

      Description de la couche     Combinaison de touches
      ------------------------     ----------------------
      Latin                        [ALT]+[MAJ_GAUCHE]
      Cyrillique phonétique        [ALT]+[MAJ_DROITE]

  (*) Le turkmène peut être écrit avec l'alphabet latin lorsqu'il est
      utilisé avec cp852. Dans ce cas, la couche cyrillique ne sera
      pas disponible.

  Le schéma phonétique cyrillique ci-dessous s'applique aux éléments
  suivants :

  1) Claviers lettons (tous sauf lv455), lors de l'utilisation de cp3012
     ("RusLat").
  2) Claviers lituaniens (tous sauf lt456), lors de l'utilisation de
     cp771 ou cp772.
  3) Claviers polonais lors de l'utilisation de cp848 (ukrainien) ou
     cp849 (biélorusse).
  4) Claviers roumains lors de l'utilisation de cp848 (ukrainien) ou
     cp30010 (gagaouze/roumain cyrillique).

      Description de la couche     Combinaison de touches
      ------------------------     ----------------------
      Latin                        [ALT]+[MAJ_GAUCHE]
      Cyrillique phonétique        [ALT]+[MAJ_DROITE]

Utilitaires :

  Ces UTILITAIRES ne font pas partie de la distribution FreeDOS standard,
  mais ils fonctionnent bien entendu avec FreeDOS. Pour les faire
  fonctionner, vous aurez besoin d'un fichier exécutable (fichier
  *.com ou *.exe) existant et FreeDOS doit déjà être démarré.
  Vous pouvez les exécuter en tapant leur nom, par exemple "tee" ou
  "foxcalc". Le programme doit se trouver dans le répertoire courant,
  sinon vous devez auparavant utiliser la commande CD ou PATH.
  Dans cette section vous ne trouverez qu'une petite liste d'outils
  qui sont souvent utilisés avec FreeDOS. Si vous avez besoin d'un
  outil spécial, veuillez vous rendre sur Internet et effectuer une
  recherche.

Commande : devload

  DEVLOAD charge des pilotes de périphérique depuis le shell.

Syntaxe :

  DEVLOAD [/H] [/Q] [/V] [/A] nomfichier [params]
    Émule : DEVICE=nomfichier [params] dans CONFIG.SYS / FDCONFIG.SYS
  DEVLOAD [/?]
          nomfichier  Le nom du fichier à charger
          params      Les paramètres du fichier à charger

Options :

  /H  Tente de charger le pilote dans l'UMB.
  /Q  Mode silencieux.
  /V  Mode verbeux.
  /A  Mode auto (force à rester chargé).
  /?  Affiche l'aide.

Commentaires :

  DEVLOAD permet à l'utilisateur de charger les pilotes alors que
  FreeDOS est en train de fonctionner plutôt que mettre à jour
  config.sys / fdconfig.sys et de redémarrer.

Exemples :

  devload C:\FDOS\DRIVERS\UIDE.SYS /S125 /D:MYDVD

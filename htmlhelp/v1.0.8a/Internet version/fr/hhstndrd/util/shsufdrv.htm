Commande : shsufdrv

  SHSUFDRV simule une disquette ou un disque dur en utilisant un fichier
  d'image.

Syntaxe :

  SHSUFDRV [/F[:][?]]image[,lecteur]... [/W] [/V] [/U] [/Q[Q]]
           /F:    Le chemin vers l'image.
           image    Nom de fichier de l'image disque brut ; '?'
                    ignorera une image invalide.
           lecteur  La lettre de lecteur à utiliser (la valeur par
                    défaut est la première disponible).

Options :

  /W      Autoriser les images à être écrites.
  /V      Afficher l'utilisation de la mémoire (seulement lors de
          l'installation).
  /U      Décharger.
  /Q      Silencieux - ne pas afficher la bannière d'identification.
  /QQ     Vraiment silencieux - ne rien afficher.

Commentaires :

  Les fichiers d'image peuvent être créés avec DISKCOPY.
  Aucune lettre de lecteur ne sera affectée plus bas que la première.
  Par exemple : img1,E img2,D affectera à E: et F:, *pas* E: et D:.
  Seul le lecteur (sur la première partition d'un disque dur) est
  simulé, pas le disque lui-même et seulement une taille de secteur
  de 512 octets.
  SHSUFDRV / SHSURDRV a les  CODES DE SORTIE (NIVEAUX D'ERREUR)
  suivants :
    0     Désinstallé, l'aide affichée
    1-32  Première lettre de lecteur attribué (A: = 1) lors d'une
          installation réussie ;
    255   Pas installé, pas désinstallé

Exemples :

  SHSUFDRV /F:?C:\test\diskimg.img,H /W /V

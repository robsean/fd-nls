Commande : dosfsck

  DOSFSCK vérifie et répare les systèmes de fichiers DOS.

Syntaxe :

  dosfsck [-aAflrtvVwy][-d chemin -d ...][-u chemin -u ...] périphérique

Options:

  -a        Réparer automatiquement le système de fichiers.
  -A        Bascule vers le format de système de fichiers Atari.
  -d chemin Supprimer ce fichier.
  -f        Récupérer les chaînes inutilisées dans les fichiers.
  -l        Lister les noms de chemin.
  -n        No-op, vérifier non interactivement sans effectuer de
            changements.
  -r        Réparer interactivement le système de fichiers.
  -t        Tester à la recherche de mauvais clusters.
  -u chemin Tenter de récupérer ce fichier (pas répertoire).
  -v        Mode verbeux.
  -V        Effectuer une passe de vérification.
  -w        Écrire les modifications immédiatement sur le disque.
  -y        Identique à -a, pour la compatibilité avec d'autres *fsck.
  --help    Affiche l'aide.

Commentaires :

  Utilisez DOSFSCK si vous avez des problèmes avec des fichiers / 
  partitions.
  DOSFSCK prend en charge les noms de fichiers longs et le FAT32.

Exemples :

  dosfsck -av c:

Voir également :

  chkdsk

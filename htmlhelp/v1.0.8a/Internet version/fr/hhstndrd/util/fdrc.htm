Commande : fdrc

  FreeDOS Resident Calculator (FDRC) est une calculatrice résidente pour
  les programmeurs.

Syntaxe :

  fdrc [options]

Options :

  FDRC prend en charge les paramètres de ligne de commande suivants
  (vous pouvez utiliser '/' au lieu de '-') :
  -h         Aide courte.
  -?         Identique à -h.
  -H         Aide sur le clavier et l'utilisation des opérateurs.
  -a         Calculer le résultat après l'appui sur n'importe quelle
             touche, pas seulement après l'appui sur 'Entrée'.
             Remarque : la chaîne actuelle sera stockée dans
             l'historique seulement suite à un appui sur 'Entrée',
             même dans ce mode.
  -c         Effacer la ligne de saisie lors de l'appui sur une touche
             alphanumérique _immédiatement_ après le popup. L'appui sur
             'Suppr', 'Retour arrière' ou sur une de ces touches
             n'efface pas la ligne de saisie.
  -e         Autoriser à charger une seconde, troisième... copie de
             FDRC dans la mémoire.
  -n         Mode non résident.
  -w         Empêcher le chargement de FDRC dans l'UMB. Par défaut FDRC
             se charge dans l'UMB si disponible.
  -u         Décharger. Si INT 09h et/ou INT 10h est intercepté par un
             autre programme, FDRC affichera un avertissement et
             quittera.
  -D         Désactive temporairement FDRC.
  -E         Active FDRC.
  -q         Mode silencieux. Si vous essayez d'appeler FDRC depuis un
             programme de mode graphique, FDRC émettra deux bips. Si
             vous atteignez la première ou dernière entrée dans
             l'historique, FDRC émettra un bip. Le commutateur -q
             désactive tous ces sons.
  -l:nombre  Colonnes du côté gauche de la fenêtre du FDRC.
  -t:nombre  Rangées du côté supérieur de la fenêtre du FDRC.
             nombre -- nombre décimal >= 0.
  -k:nombre  Scancode de la touche d'activation (hexadécimal).
             Cf 'rbil_i09.txt' pour les détails. Valeur par défaut
             -- 35h ('/').
  -s:nombre  Code des touches Maj (shift) (hexadécimal).
               Maj droite :                 01
               Maj gauche :                 02
               Ctrl (n'importe lequel) :    04
               Alt (n'importe lequel) :     08
               Arrêt Défil :                10
               Verr Num :                   20
               Verr Maj :                   40
               Insertion :                  80
               Valeur par défaut -- 04 (Ctrl). Par exemple :
               MajDroite+Alt ==== 1 + 8 == 9.
  -y:nombre  Taille du tampon de l'historique (décimal). Valeur par
             défaut -- 0.
             La valeur minimale est 200, maximale est 16000 (peut être
             modifiée dans 'config.inc').

Commentaires :

  FDRC prend en charge les opérations suivantes :
    + -- addition              & -- bit à bit ET
    - -- soustraction          | -- bit à bit OU
    * -- multiplication        ^ -- bit à bit XOR
    / -- division              < -- décalage logique vers la gauche
    % -- reste                 > -- décalage logique vers la droite
    \ -- racine carrée         { -- décalage arithmétique vers la gauche
    ~ -- bit à bit NON         } -- décalage arithmétique vers la droite
    ` -- négation

  Vous pouvez utiliser les touches suivantes :
  Chiffres et lettres -- entrer des chiffres.
  " -- si vous entrez un symbole de guillemets anglais doubles, le
  deuxième de ce symbole sera entré automatiquement.
  Touches de curseur droite et gauche -- déplacer le curseur.
  Touches de curseur haut et bas -- chaîne précédente / suivante de
  l'historique Page Up et Page down -- premières et dernières chaînes
  de l'historique.
  Home et Fin -- déplacer le curseur.
  Entrée -- calculer le résultat et le stocker dans le tampon de
  l'historique.
  Insertion -- basculer entre le mode d'insertion et surimpréssion.
  Retour arrière -- effacer le caractère avant le curseur.
  Suppr  -- effacer le caractère sous le curseur.
  Ctrl-Y -- effacer la chaîne.
  Ctrl-K -- effacer le texte jusqu'à la fin de la chaîne.
  Vous pouvez déplacer la fenêtre de FDRC avec les raccourcis de
  WordStar (valeur par défaut) :
    Ctrl-S -- gauche
    Ctrl-D -- droite
    Ctrl-E -- haut
    Ctrl-X -- bas
  ou avec les raccourcis Vi (si vous avez défini VI_KEYS constant dans
  config.inc):
    Alt-H -- gauche
    Alt-L -- droite
    Alt-K -- haut
    Alt-J -- bas
  Alt-S -- basculer entre la représentation décimale signée et non
  signée du résultat.
  Alt-D -- mettre la représentation décimale du résultat dans le
  tampon du clavier et quitter.
  Alt-H -- mettre la représentation hexadécimale du résultat dans
  le tampon du clavier et quitter.
  Alt-1 .. Alt-4 -- mettre la partie correspondante de la représentation
  binaire du résultat dans le tampon du clavier et quitter.
  Échap -- quitter.

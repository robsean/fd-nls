Commande : locate

  LOCATE est un outil pour rechercher des fichiers sous DOS.

Syntaxe :

  LOCATE [filespec] [options]
         Filespec peut contenir une lettre de lecteur, un nom de
         répertoire ou les deux à la fois. Si une lettre de lecteur
         est spécifiée, la recherche ne sera effectuée que sur ce
         lecteur. Si aucun lecteur n'est spécifié, Locate décidera
         seul les lecteurs sur lesquels la recherche sera effectuée.
         Tout ce qui précède la dernière barre oblique inverse dans la
         spécification de fichier est supposé être le nom d'un
         répertoire dans lequel la recherche sera effectuée ; Locate
         recherchera dans ce répertoire et les sous-répertoires qu'il
         contient. Si un nom de répertoire est spécifié sans lettre de
         lecteur, le répertoire sera supposé être sur le lecteur actuel,
         la recherche ne sera effectuée que sur le lecteur actuel.
         Les jokers ne sont pas autorisés dans le nom de répertoire.

Options :

  Que rechercher :
    /D              À la fois les fichiers et les sous-répertoires (la
                    valeur  par défaut).
    /D-             Seulement les fichiers, pas les sous-répertoires
                    (valeur par défaut avec /B /K /X /S: /0).
    /D+             Les sous-répertoires seulement, pas les fichiers.
    /H              Les éléments cachés ou système seulement.
    /X              Les fichiers exécutables seulement (.COM .EXE .BAT).
    /0              Les fichiers vides (0 octet) seulement.
    /S:[min][,max]  Les fichiers dans une plage de taille.
    /D:[min][,max]  Les éléments dans une plage de dates.
    /D:dowlist      Les éléments datés de jours spécifiques de la
                    semaine.
    /T:[min][,max]  Les éléments dans une plage de temps.
    /A:[a+][a-]     Masquer selon les attributs.
  Où rechercher :
    /T              Répertoire courant et chemin de recherche seulement.
    /NR             Ne pas rechercher récursivement dans les
                    sous-répertoires.
    /R              Les disques fixes locaux seulement.
    /M              Inclure les mappages de lecteurs réseau en double.
    /E:var          Les répertoires listés dans une variable
                    d'environnement.
  Comment afficher les éléments trouvés :
    /P              Affichage de style Peter, avec les dates, horaires,
                    attributs et tailles.
    /I              Avec les numéros d'index, dates, heures, attributs,
                    tailles.
    /W              Affichage large.
    /N              Affichage brut (liste de fichiers).
    /B:"commande"   En tant que fichier batch.
    /L              En utilisant les noms de fichiers longs de
                    Windows 95.
    /O:"chaîne"     Sortie formatée à l'aide de macros.
    /S              Informations de résumé seulement.
    /US             Utiliser le format de date et d'horaire des
                    États-Unis (Tue May 31 1994).
    /UK             Utiliser le format de date et d'horaire du
                    Royaume-Uni (Tue 31 May 1994).
    /UJ             Utiliser le format de date et d'horaire nippon
                    (ISO) (Tue 1994-05-31).
    /12             Format d'heure de 12 heures.
    /24             Format d'heure de 24 heures.
    /NP             Pas de pagination.
  Quoi faire avec les éléments trouvés :
    /G              Aller au répertoire.
    /K              Tuer (kill, effacer) les éléments trouvés.
    /C:"chaîne"     Exécuter des commandes, en utilisant les mêmes
                    macros que /O.
    /F:n            S'arrêter après les premiers n éléments.
  Other options:
    /Y              Yes, Oui -- ne pas demander confirmation pour
                    /K ou /C.
    /NV             Pas d'appels vidéo BIOS.
    /V              Échanger la barre d'espace et la touche Entrée
                    lors de la pagination.
    /NA             Ne pas romaniser les lettres à des fins de
                    comparaison.

Commentaires :

  Veuillez lire locate.txt pour davantage d'informations.

Examples:

  locate           trouver tous les fichiers et répertoires ("éléments")
  locate *.SYS     trouver tous les éléments avec une extension .SYS
  locate /H        trouver tous les éléments cachés ou système
  locate *.SYS /H  trouver les éléments cachés/système avec
                   l'extension .SYS

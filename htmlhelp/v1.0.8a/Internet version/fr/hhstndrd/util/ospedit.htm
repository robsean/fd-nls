Commande : ospedit

  OSPEDIT est un éditeur de texte libre.

Syntaxe :

  ospedit [[lecteur][chemin]fichier]
          lecteur  La lettre du lecteur, par exemple C:
          chemin   Le répertoire, par exemple \exemple\
          fichier  Le fichier, par exemple test.txt.
                   Vous pouvez également ouvrir des nouveaux fichiers
                   depuis le programme.

Options :

  - aucune -

Commentaires :

  Un éditeur tel qu'OSPEDIT peut être utilisé pour créer des fichiers
  textuels simples. Il peut aussi être utilisé pour modifier les
  fichiers comme CONFIG.SYS / FDCONFIG.SYS et AUTOEXEC.BAT.

Exemples:

  ospedit c:\exemple.txt   ouvre c:\exemple.txt pour écrire dans le
                           fichier (le fichier exemple.txt doit
                           déjà exister)
  ospedit c:\autoexec.bat
  ospedit c:\fdconfig.sys

Commande : tree / pdtree

  TREE / PDTREE affiche graphiquement la structure du répertoire
  [dossier] d'un lecteur ou chemin donné. Il inclut éventuellement
  une liste de fichiers à l'écran.

Syntaxe :

  TREE [d:][chemin] [/F] [/A] [/?]
  PDTREE [d:][chemin] [/F] [/A] [/V] [/S][/P] [/?]
       d:chemin  Le lecteur et le chemin à partir duquel démarrer. Si
       aucun chemin n'est entré, tree/pdtree démarrera à partir du
       répertoire courant.

Options :

   /F  Afficher les noms des fichiers dans chaque répertoire.
   /A  Utiliser les caractères ASCII pour représenter visuellement
       la structure des répertoires.
   /?  Afficher l'aide et quitter.

 Options supplémentaires de pdTree de FreeDOS : (indisponibles dans le
 Tree standard.)
   /V  Afficher les informations de version et quitter.
   /S  Noms courts uniquement (utilise un nom de fichier court,
       c'est-à-dire alternatif, lorsqu'il existe).
       Version DOS seulement, désactive l'utilisation de l'API LFN
       pour trouver le premier/le suivant.
   /P  Effectuer une pause après chaque page. (Remarque : utilise
       actuellement un écran 80x25 par défaut).
       Pour la version Win32 seulement, ignore l'option pause si la
       sortie est redirigée vers un fichier ou envoyé vers un autre
       programme (pipe), avec par exemple tree | more ; tente
       également d'utiliser la taille actuelle de la console (TODO
       (à faire) pour la version DOS).

Commentaires :

  PDTREE prend en charge NLS (national language support, prise en
  charge des langues nationales).

Exemples :

  Pour afficher la structure hiérarchique complète des répertoires du
  lecteur C effectuer une pause après chaque écran, lancez tree en
  spécifiant C: en tant que chemin de départ et transmettez sa sortie
  à la commande MORE :
  tree C:\ | more
  Lors de l'utilisation de pdTree, l'exemple ci-dessus peut être obtenu
  en utilisant :
  pdtree C:\ /p
  Pour afficher la hiérarchie d'un sous-répertoire et obtenir la sortie
  uniquement avec des caractères standards ASCII (par exemple pour
  imprimer ou l'envoyer à quelqu'un qui pourrait le visionner sans la
  prise en charge des caractères de DOS pour dessiner des lignes et des
  boîtes, spécifiez le sous-répertoire avec le commutateur /A :
    tree ASUBDIR /a
  Pour afficher la structure d'un répertoire et les fichiers dans chaque
  répertoire, ajoutez le commutateur /F :
    tree ASUBDIR /f | more

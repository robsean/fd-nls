Commande : memory

  MEMORY affiche la mémoire interne utilisée par FreeCOM
         (command.com) en interne.

Syntaxe :

  MEMORY [/?]

Options :

  /?  Affiche des informations au sujet de la commande.

Commentaires :

  MEMORY affiche combien de mémoire est utilisée par les alias,
  l'historique, la pile des répertoires, le tampon du dernier dossier,
  swapinfo, etc.
  MEMORY est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  memory /?
  memory

Commande : alias

  ALIAS crée, affiche ou enlève un alias à utiliser sur la ligne de
  commande.

Syntaxe :

  ALIAS [variable[=][chaîne]]
  ALIAS [/?]

Options :

  variable  Spécifie le nom de l'alias.
  chaîne    Spécifie une série de caractères à attribuer à l'alias.
  /?        Affiche l'aide.

Commentaires :

  Tapez ALIAS sans paramètres pour afficher les alias actuels.
  Tapez "ALIAS variable=" (sans la chaîne) pour effacer l'alias.
  ALIAS est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  "Alias"             affiche une liste des alias actuels, s'ils
                      existent
  "Alias texte=edit"  si vous tapez "texte" + ENTER l'éditeur sera
                      lancé.
  "Alias texte="      la variable texte est effacée. Si vous tapez
                      "texte", l'éditeur ne sera pas lancé, vous
                      obtiendrez un message d'erreur : la commande
                      "texte" est introuvable !

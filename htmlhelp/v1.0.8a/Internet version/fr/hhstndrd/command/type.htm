Commande : type

  TYPE affiche le contenu des fichiers textuels.

Syntaxe :

  TYPE [lecteur][chemin]nomfichier
  TYPE [/?]
       lecteur     Spécifie la lettre du lecteur sur lequel se trouve
                   le fichier, par exemple C:
       chemin      Spécifie le chemin vers le fichier, par
                   exemple \exemple\
       nomfichier  Spécifie le fichier à afficher, par exemple test.txt

Options :

  /?  Affiche l'aide.

Commentaires :

  Type affiche le texte complet. Si celui-ci est trop long, il
  s'arrêtera à la fin. Vous pouvez empêcher cela en utilisant :
  type | more xy.txt
  TYPE est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  type C:\autoexec.bat
  type | more C:\autoexec.bat
  type | more C:\config.sys
  type | more C:\exemple\test.txt

Voir également :

  more
  sort

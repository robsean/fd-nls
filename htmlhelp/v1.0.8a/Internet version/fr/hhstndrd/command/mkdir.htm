Commande : mkdir

  MKDIR crée un répertoire ou un sous-répertoire.

Syntaxe :

  MKDIR [lecteur][chemin]nomchemin
  MKDIR [/?]
        lecteur     La lettre du lecteur sur lequel vous voulez créer
                    un répertoire, par exemple C:
        chemin      Le nom de chemin qui existe déjà, par exemple si
                    vous êtes déjà dans un répertoire.
        nomchemin   Le nom du répertoire que vous voulez créer, par
                    exemple \exemple. Cela peut aussi être le nom
                    d'un sous-répertoire.

Options :

  /?  Affiche l'aide

Commentaires :

  Vous pouvez aussi utiliser MD au lieu de MKDIR.
  Si vous tapez simplement "MKDIR NOMCHEMIN" le dossier sera créé
  dans le répertoire du moment. Signification : vous créez un
  sous-dossier.
  Sous DOS pur, le nom du répertoire ne doit pas dépasser 8 lettres.
  MKDIR est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  mkdir c:\test
  mkdir c:\test\testx
    ou si vous êtes dans c:\test :
  mkdir test2 (crée c:\test\test2)

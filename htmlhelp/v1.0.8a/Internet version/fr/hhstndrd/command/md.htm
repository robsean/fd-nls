Commande : md

  MD crée un répertoire ou un sous-répertoire.

Syntaxe :

  MD [lecteur][chemin]nomchemin
  MD [/?]
     lecteur     La lettre du lecteur sur lequel vous souhaitez créer
                 un répertoire, par exemple C:
     chemin      Le nom de chemin qui existe déjà, par exemple si
                 vous êtes déjà dans un répertoire.
     nomchemin   Le nom du répertoire que vous souhaitez créer, par
                 exemple \exemple. Cela peut aussi être le nom
                 d'un sous-répertoire.

Options :

  /?  Affiche l'aide

Commentaires :

  Vous pouvez aussi utiliser MKDIR au lieu de MD.
  Si vous tapez simplement "MD NOMCHEMIN" le dossier sera créé
  dans le répertoire du moment. Signification : vous créez un
  sous-dossier.
  Sous DOS pur, le nom du répertoire ne doit pas dépasser 8 lettres.
  MD est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  md c:\test
  md c:\test\testx
    ou si vous êtes dans c:\test :
  md test2 (crée c:\test\test2)

Voir également :

  cd
  cdd
  chdir
  mkdir

Commande : copy

  COPY copie un ou plusieurs fichiers sur un autre emplacement.

Syntaxe :

  COPY [lecteur][chemin]nomfichier [/A | /B]
       + [lecteur][chemin]nomfichier [/A | /B]
       [+ [...]] [lecteur][chemin]nomfichier [/A | /B] [/V] [/Y | /-Y]
       [/?]
       lecteur     La lettre du lecteur, par exemple C:
       chemin      Le répertoire, par exemple \exemple\
       nomfichier  Le nom du fichier, par exemple test.txt

Options :

  source       Spécifie le ou les fichiers à copier
               (= premier [[lecteur][chemin][fichier]]).
  /A           Indique un fichier textuel ASCII.
  /B           Indique un fichier binaire.
  destination  Spécifie le répertoire et/ou le nom de fichier pour le
               ou les nouveaux fichiers (=dernier [[lecteur][chemin]
               [fichier]]).
  /V           Vérifie que les nouveaux fichiers ont été écrits
               correctement.
  /Y           Supprime la demande de confirmation avant d'écraser
               un fichier de destination existant.
  /-Y          Invite à confirmer que vous souhaitez remplacer un
               fichier de destination existant.
  /?           Affiche l'aide.

Commentaires :

  COPY ne copiera pas les dossiers ou fichiers qui ont une taille nulle.
  Pour les copier, veuillez utiliser XCOPY.
  La variable d'environnement COPYCMD peut être définie pour faire de /Y
  la valeur par défaut :  SET COPYCMD=/Y
  Pour annuler /Y pour une commande copy particulière, utilisez /-Y sur
  la ligne de commande. La variable COPYCMD affecte aussi la commande
  XCOPY.
  Pour concaténer des fichiers, spécifiez un seul fichier de destination
  mais de multiples fichiers pour la source (avec les jokers ou le
  format fichier1+fichier2+fichier3).
  Explication détaillée des options :
    source       La source se compose de "lecteur" "chemin" et "fichier".
    lecteur      La lettre du lecteur, par exemple "C:" ou bien "D:".
    chemin       Le répertoire, par exemple "\exemple\"
    fichier      Le om du fichier à copier, par exemple "test.txt".
                 En entier :  C:\exemple\test.txt .
    destination  Le nom du répertoire ou des fichiers de destination.
                 Si le nom de fichier de destination contient a *, cela
                 sera remplacé par cette partie du nom de fichier avec
                 la même partie que la source.
    /A           Copie un fichier ASCII. Si cela précède un nom de
                 fichier sur une liste de fichiers, alors à partir de
                 ce fichier la copie sera effectuée en fichiers ASCII.
                 par exemple:
                 copy /A ASCI1.txt+ASCI2.txt+ASCI3.txt  /B+BIN1.bin
                 CIBLE
    /B           Copie un fichier binaire. C'est la valeur par défaut.
                 Si cela précède un nom de fichier sur une liste de
                 fichiers, alors à partir de ce fichier, la copie sera
                 effectuée en binaire, par exemple :
                 copy /B BIN1.bin+BIN2.bin+BIN3.bin /A+ASCI1.txt CIBLE
    /V           Vérifie que les fichiers sont écrits correctement.
    /Y           Supprime le message de confirmation avant d'écraser ;
                 les fichiers avec le même nom sur la destination
                 seront écrasés.
  COPY est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  copy c:\command.com a:
  copy c:\*.exe d:\exemple
  copy c:\fdos\bin\edit.* a:

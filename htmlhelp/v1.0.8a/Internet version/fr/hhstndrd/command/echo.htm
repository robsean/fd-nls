Commande : echo (commande command.com)

  ECHO affiche des messages ou active ou désactive l'écho des commandes.
  Il existe également une commande ECHO CONFIG.SYS / FDCONFIG.SYS.
  ECHO est une commande de FICHIER BATCH / AUTOEXEC.BAT.
  Elle peut aussi être utilisée sur la ligne de commande.

Syntaxe :

  ECHO [ON | OFF]
  ECHO [message]

Options :

  ON       Le mode ÉCHO est activé. Si vous utilisez ECHO dans un fichier
           batch, chaque ligne de commande sera affichée lors de son
           exécution.
  OFF      Le mode ÉCHO est désactivé. Si vous utilisez ECHO dans un
           fichier batch, la ligne de commande ne sera pas affichée lors
           de son exécution.
           Cela n'a pas d'effet sur la sortie de la commande elle-même.
  message  Le message que vous souhaitez afficher sur l'écran.

Commentaires :

  Les commandes peuvent aussi être rendues silencieuses en les commençant
  par le symbole @.
  Tapez ECHO sans paramètre pour afficher le réglage actuel d'ECHO.
  ECHO peut aussi être utilisée en dehors d'un fichier batch.
  ECHO est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner. Il existe également une commande
  ECHO CONFIG.SYS / FDCONFIG.SYS avec des options différentes.

Exemples :

  DANS UN FICHIER .BAT / AUTOEXEC.BAT:
    ECHO Bonjour,
    ECHO. comment allez-vous ?
    @ECHO Bien ?
    @ECHO. Ça fait plaisir d'entendre ça

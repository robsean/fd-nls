Commande : doskey

  Dans certaines versions de DOS, DOSKEY étend la fonctionnalité
  de base du clavier de l'interpréteur de la ligne de commande.
  Il n'y a pas besoin de DOSKEY dans FreeDOS, ses fonctionnalités
  étant intégrées dans l'interpréteur de commandes.

Syntaxe :

  - aucune -

Options :

  touche HAUT      affiche l'historique
  touche BAS       affiche l'historique
  touche GAUCHE    pour naviguer sur la ligne de commande
  touche DROITE    pour naviguer sur la ligne de commande
  touche FIN       pour naviguer sur la ligne de commande
  touche HOME      pour naviguer sur la ligne de commande
  touche INSÉR     bascule entre le mode surimpression et le mode
                   d'insertion
  touche TAB       complète le mot actuel comme nom de fichier
  touche TAB 2x    affiche tous les fichiers correspondants

Commentaires :

  L'utilitaire externe DOSKEY a été incorporé dans FreeCOM. Utilisez
  curseur HAUT et BAS pour afficher l'historique et HISTORY pour une
  historique complète. Utilisez curseur GAUCHE et DROITE et les
  touches FIN ainsi que HOME pour naviguer sur la ligne de commande
  et appuyez sur INSÉR pour basculer entre les modes de surimpression
  et d'insertion. Utilisez la touche TAB pour compléter le mot actuel
  en tant que nom de fichier ; appuyez deux fois pour afficher tous
  les fichiers correspondants.
  DOSKEY est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  - aucune - testez simplement les touches décrites.

Commande : cd

  CD affiche le nom ou modifie le répertoire courant.

Syntaxe :

  CD [lecteur] [chemin]
  CD [..] [-] [\] [/?]
     lecteur  La lettre du lecteur, par exemple, C:
     chemin   Le répertoire, par exemple \exemple\

Options :

  ..      Spécifie que vous souhaitez aller au répertoire parent.
  -       Si la fonctionnalité "dernier répertoire" est activée, va
          au dernier répertoire.
  \       Va au répertoire racine.
  /?      Affiche l'aide.

Commentaires :

  Tapez CD lecteur: pour afficher le répertoire actuel sur le lecteur
  spécifié. Tapez CD sans paramètre pour afficher le lecteur et dossier
  actuels. Si aucun lecteur n'est spécifié, alors le lecteur actuel est
  supposé. Si aucun chemin n'est spécifié, le répertoire courant est
  affiché. Vous pouvez aussi utiliser CHDIR au lieu de CD.
  CD est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

    cd c:\fdos     va au répertoire c:\fdos
    cd ..          retourne de c:\fdos à c:\
    cd \           retourne de c:\undossier\unsousdossier au
                   répertoire racine, c'est-à-dire c:\
  si vous êtes sur c:\ :
    cd fdos        va au répertoire c:\fdos
    cd bin         va au répertoire c:\fdos\bin
    cd d:\exemple  change le répertoire sur d: en d:\exemple, mais
                   ne va PAS sur le lecteur d:

Commande : fcbs

  FCBS définit le nombre de blocs de contrôle de fichiers.
  FCBS est une commande  CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  fcbs=nnn

Options :

  nnn    Nombre de blocs de contrôle de fichiers, compris entre 1 et 255.
         La valeur par défaut est 4.

Commentaires :

  FCBS définit le nombre de blocs de contrôle de fichiers pour réserver
  de la place.
  Comme les blocs de contrôle de fichiers ont été remplacés par des
  descripteurs de fichiers (voir  FILES ) et FreeDOS simule
  dynamiquement FCBS à partir des données d'indicateur selon les besoins,
  nnn est simplement ignoré par FreeDOS.
  FCBS est une commande interne à kernel.sys et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS:
    fcbs=4

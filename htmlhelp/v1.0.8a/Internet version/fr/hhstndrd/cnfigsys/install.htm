Commande : install

  INSTALL exécute un programme TSR (un programme qui reste dans la
  mémoire), sans environnement, au lieu d'utiliser quelque chose comme
  lh ou loadhigh.
  INSTALL charge un programme TSR dans la mémoire basse ou
  conventionnelle, tandis que INSTALLHIGH charge le programme TSR dans
  la mémoire haute.
  INSTALL est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  install=[lecteur][chemin]nomfichier [options]
          lecteur     La lettre du lecteur, par exemple C:
          chemin      Le répertoire, par exemple \exemple\
          nomfichier  Le nom du fichier du tsr à exécuter, par exemple
                      test.sys

Options :

  options  Il s'agit là d'option qui seront passées au TSR. Il ne
           s'agit pas d'options pour la commande 'install'.

Commentaires :

  INSTALL charge le programme spécifié par lecteur:\chemin\nomfichier.
  Il est généralement utilisé pour charger des programmes TSR (terminate
  and stay resident) avec un bloc d'environnement minimal. Les options
  sont pour le programme ; veuillez vous référer à la documentation de
  votre logiciel pour les options prises en charge et leur utilisation.
  INSTALL est une commande interne à kernel.sys et ne nécessite
  aucun autre fichier pour fonctionner.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS:
    install=C:\fdos\bin\ctmouse.exe

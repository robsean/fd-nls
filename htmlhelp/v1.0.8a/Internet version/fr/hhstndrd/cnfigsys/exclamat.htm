Commande : !

  Avec "!" et "?" config.sys peut être rendu interactif.
  Les lignes Config.sys commençant par "!" seront traitées en une
  seule étape (c'est-à-dire même en cas de single stepping).
  '!' est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  ! d'autres commandes config.sys telles que DEVICE=...

Options :

  - aucune -

Commentaires :

  Les lignes de commande dans config.sys commençant par '!' seront
  exécutées en toutes circonstances.
  '!' est une commande interne à kernel.sys et ne nécessite
  aucun autre fichier pour fonctionner.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS:
    !FILES=20
    !DOS=HIGH,UMB
    !BUFFERS=30
    !DEVICE=MyNetWorkDriver.sys

Voir également :

  ;
  ?

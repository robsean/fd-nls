Commande : ;

  ';' marque une ligne de config.sys en tant que commentaire.
  ';' a le même effet que 'REM'.
  ';' est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  ; [commentaire]
  REM [commentaire]

Options :

  commentaire  tout commentaire que vous voudriez ajouter pour vous
               souvenir de pourquoi vous avez fait ceci ou cela ; peut
               aussi être une commande qui ne doit pas être exécutée
               (peut-être pour des tests).

Commentaires :

  ';' et 'REM' fournissent la capacité de placer des commentaires dans
  un fichier de configuration. Le texte suivant ';' ou 'REM' jusqu'à ce
  que la fin de la ligne soit atteinte sera ignoré. Cela peut également
  être utilisé pour désactiver temporairement le chargement d'un
  périphérique ou d'une autre option.
  ';' est une commande interne à kernel.sys et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS :
    REM DOS=HIGH,UMB  a le même effet !
    ;DOS=HIGH,UMB     a le même effet !

Voir également :

  !
  ?
  config.sys
  fdconfig.sys
  rem

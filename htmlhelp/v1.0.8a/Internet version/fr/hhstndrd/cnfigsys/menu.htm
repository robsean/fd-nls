Commande : menu

  MENU affiche un menu durant l'exécution de config.sys / fdconfig.sys.
  MENU est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

   menu [texte]

Options :

  texte  Le texte que vous souhaitez afficher durant l'exécution de
  config.sys
    Utilisation :
      Utilisez menudefault pour définir une option par défaut temporisée.
      Les lignes commençant par 1? ne seront traitées que si
      l'utilisateur appuie sur 1. Les lignes commençant par 23? ne
      seront chargées que si l'utilisateur appuie sur 2 ou 3. Les
      options différentes de 0 ne peuvent être choisies que si au moins
      une ligne "chiffres?" les utilise, comme montré dans l'exemple
      ci-dessous.
    Utilisation avec autoexec.bat:
      La variable d'environnement CONFIG est définie sur la valeur de
      l'option du menu. Donc %CONFIG% peut être utilisé dans autoexec.bat
      en conjonction avec la commande if. Par exemple :
        Dans config.sys "1" a été choisi :
      Dans autoexec.bat:
        IF "%config%"=="1" SHSUCDX /D:FDCD001

Commentaires :

  MENU affiche un menu lors de l'exécution de config.sys / fdconfig.sys.
  MENU est une commande interne à kernel.sys et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS:
    MENU Veuillez choisir la configuration :
    MENU
    MENU Option 0    des choses basiques seulement
    MENU Option 1    CD-ROM
    MENU Option 2    TROUSERS
    MENU Option 3    CD-ROM ET TROUSERS
    0? ECHO Avertissement : seulement des pilotes standards !
    13?DEVICE=uide.sys /S125 /D:FDCD001
    23?DEVICE=TROUSERS.SYS
  DANS AUTOEXEC.BAT:
    IF "%config%"=="1" SHSUCDX /D:FDCD001
    IF "%config%"=="3" SHSUCDX /D:FDCD001

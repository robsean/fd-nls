Commande : version

  VERSION indique à FreeDOS la version de DOS à déclarer.
  VERSION est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  version=x.y
    Où x et y sont des nombres (ils peuvent chacun avoir un ou plusieurs
    chiffres). La valeur par défaut est 5.0.

Options :

  - aucune -

Commentaires :

  VERSION est une commande spécifique à FreeDOS qui spécifie la version
  de DOS à déclarer. Pour un kernel avec prise en charge du FAT32, 7.10
  est un bon choix. Sinon, 5.0 et 6.22 sont des valeurs habituelles.
  FreeCOM command.com 'ver /r' affiche à la fois les numéros de version
  déclarés et ceux, internes, de la version de FreeDOS. VERSION ne
  modifie pas le numéro de version de FreeCOM indiqué en exécutant 'ver'.
  Cependant, le numéro de version DOS est modifié, comme indiqué dans
  'ver /R'.
  VERSION est nécessaire pour faire croire à des vieux programmes DOS
  qu'ils fonctionnent avec la version correcte de DOS.
  VERSION est une commande interne à kernel.sys et ne nécessite
  aucun autre fichier pour fonctionner.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS :
    version=6.22

Commande : screen

  SCREEN définit le nombre de lignes de texte à l'écran.
  SCREEN est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  screen=xx

Options :

  xx (Codes) :
    0x11  police 8x14 (28 lignes VGA, 25 lignes EGA)
    0x12  police 8x8  (50 lignes VGA, 43 lignes EGA)
    0x14  police VGA par défaut (25 lignes VGA)

Commentaires :

  SCREEN bascule dans le mode vidéo xxx (INT10/11xx/000).
  Certaines cartes graphiques peuvent ne pas avoir les polices 8x14
  dans le BIOS. Dans ce cas, un pilote peut être chargé afin de
  charger une police appropriée dans la RAM, mais SCREEN=0x11 ne
  devrait pas être utilisé. Si xx est inférieur à 15 (0x0f), il sera
  traité comme un nombre de mode d'écran, par exemple 1 pour du texte
  monochrome 40x25 CGA.
  SCREEN est une commande interne à kernel.sys et ne nécessite
  aucun autre fichier pour fonctionner.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS :
    screen=0x12

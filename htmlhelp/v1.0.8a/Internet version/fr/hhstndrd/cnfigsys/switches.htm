Commande : switches

  SWITCHES ajuste le traitement durant l'amorçage.
  SWITCHES est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  switches=/E[:xxx] /F /K /N /W

Options :

  /F   ignore la vérification de délai pour les touches F5/F8 avant de
       traiter config.sys [équivalent à SYS CONFIG skipconfigseconds=0]
       F5 et F8 sont traitées seulement si elles sont enfoncées avant
       l'amorçage de DOS mais après la configuration du clavier - le
       bon moment est facile à manquer.
  /N   désactive la prise en charge de F5/F8 [équivalent à kernel
       config (SYS CONFIG, exécutez SYS CONFIG /? pour les détails)
       skipconfigseconds=-1]
       F5 (sauter la config) et F8 (config.sys en une seule étape)
       sont ignorés.
  /K   force le traitement du clavier comme un clavier à 86 touches,
       pas à 102/105 touches. Peut être utile pour les BIOS ou pilotes
       qui gèrent mal ou pas du tout les claviers à 102/105 touches.
  /E   active le déplacement de l'EBDA (Extended BIOS Data Area), une
       taille en kilooctets peut optionnellement être spécifiée
       [xxx, dans la plage 48-1024]
  /W   n'est PAS pris en charge par FreeDOS. Cette option dans MS DOS
       définit un indicateur pour Windows 3.0 afin que ce dernier ignore
       le chargement de wina20.386 depuis le répertoire racine.

Commentaires :

  SWITCHES est une commande interne à kernel.sys et ne nécessite
  aucun autre fichier pour fonctionner.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS :
    switches=/E[:48] /F /K /N /W

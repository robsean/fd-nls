Commande : echo (commande config.sys)

  ECHO affiche des messages. Il existe également une version
  BATCH-FILE/AUTOEXEC.BAT/COMMAND.COM.
  ECHO est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  ECHO [message]

Options :

  message  Le message que vous souhaitez afficher à l'utilisateur.

Commentaires :

  ECHO affiche (fait écho) ses arguments sur la console pendant le
  traitement de CONFIG.SYS / FDCONFIG.SYS lorsque les pilotes de
  périphériques sont chargés (quand les lignes DEVICE= sont exécutées).
  ECHO est une commande interne à kernel.sys et ne nécessite aucun autre
  fichier pour fonctionner. Il existe également une commande ECHO
  de command.com avec des options différentes.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS:
    ECHO chargement du pilote1
    device=Driver1.sys
    ECHO pilote1 chargé avec succès

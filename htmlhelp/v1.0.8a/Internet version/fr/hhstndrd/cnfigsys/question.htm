Commande : ?

  Avec '!' et '?' config.sys peut être rendu interactif.
  Pour les lignes de config.sys commençant par '?' l'utilisateur sera
  interrogé : une confirmation lui sera demandée avant qu'une ligne de
  config.sys qui commence par '?' ne soit exécutée.
  '?' est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  ?  une autre commande config.sys telle que DEVICE?=...

Options :

  - aucune -

Commentaires :

  '?' - demande TOUJOURS si la ligne seule doit être exécutée.
  '?' est une commande interne à kernel.sys et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS:
    FILES?=20
    DOS?=HIGH,UMB
    BUFFERS?=30
    DEVICE?=MyNetWorkDriver.sys

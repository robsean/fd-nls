Visionneur d'aide HTML 5.3.2 - Balises prises en charge :

  Ce document décrit le sous-ensemble du HTML pris en charge par l'aide.
  <!-- -->
  Commentaire. Toutes les balises et le texte sont ignorés par l'aide
  HTML.
  <A>
    L'aide supporte complètement les attributs HREF et NAME, sauf que :
      * Les liens vers les autres fichiers ne peuvent être que relatifs,
        pas absolus. Donc
        HREF="..\foo.htm" fonctionne mais pas HREF="C:\foo.htm"
      * Les guillemets anglais "" doivent toujours entourer les noms
        de fichier et les signets, par ex. HREF="foo.htm" mais pas
        HREF=foo.htm
      * les références aux liens web et adresses e-mail sont ignorées.
      * Des noms de fichiers de style DOS (8.3) devraient être utilisés.
  <B>
    Du texte en gras est affiché en blanc clair.
  <BODY>
    Indique le début du texte à afficher.
  <BR>
    Retour à la ligne.
  <CODE>
    Du code est affiché tout comme du texte en gras (en blanc clair).
  <EM>
    Du texte souligné comme du texte en italique, est affiché
    avec la couleur magenta (un ton du violet).
  <H1>
    L'aide affichera cet entête 1 en gras et mettra une ligne épaisse
    en dessous.
  <H2>, <H3>
    Les entêtes 2 et 3 sont affichés en gras et un nouveau paragraphe
    est démarré.
  <HEAD>
    Entête HTML.
  <HR>
    Créé une règle horizontale le long de l'écran. Seule une règle
    horizontale sans aucun attribut est pris en charge à l'heure
    actuelle.
  <HTML>
    Dénote un fichier HTML.
  <I>
    Du texte en italique est affiché avec la couleur magenta (un ton
    du violet).
  <LI>
    Un objet de liste.
  <OL>
    À l'heure actuelle, les listes ordonnées sont affichées comme des
    listes non ordonnées. Les listes incorporées ne sont pas prises
    en charge.
  <P>
    Nouveau paragraphe.
  <PRE>
    Du texte préformaté
  <STRONG>
    Du texte Strong, tout comme du texte en gras est
     affiché en blanc clair.
  <TITLE>
    Ne fait rien - pour le moment.
  <TT>
    Du texte TT, tout comme du texte en italique, est affiché
    avec la couleur magenta (un ton du violet).
  <UL>
    Les listes incorporées ne sont actuellement pas prises en charge.

Les caractères supportés par l'aide HTML :

  Plus de 250 caractères sont supportés, y compris tous les caractères
  Latin-1.
  Les NCR (références numériques de caractères) et l'UTF-8 sont également
  pris en charge. Cependant, l'aide ne peut qu'afficher les caractères
  qui sont disponibles dans la page de code DOS active. Pour une liste
  des pages de code dont l'aide est 'au courant', tapez "HELP /C" sur
  la ligne de commande.

Notes générales sur l'aide HTML :

   1. Le texte passe sur la ligne suivante si elle ne tient pas sur
      une ligne.
   2. Les balises sont  insensibles à la casse, donc <A HREF="FOO.HTM">
      est équivalent à <a href="foo.htm">.
   3. La taille maximale pour un fichier est de 64 Ko.
   4. Seuls les noms de fichier courts (8.3) sont pris en charge.
   5. L'aide distingue les types de fichier selon l'extension du nom
      de fichier.
      L'extension .htm est donc recommandée pour les fichiers HTML.

Commande : diskcomp

  DISKCOMP compare le contenu de deux disquettes, piste par piste.

Syntax:

  diskcomp [lecteur [lecteur]] [/1] [/8] [/4[0]] [/b] [/?]
           lecteur La lettre du lecteur pour une disquette.
           Si une seule lettre est indiquée, on supposera que
           c'est le lecteur courant (si c'est un lecteur de disquettes.)
           Si aucune lettre n'est indiquée, on supposera que les
           deux sont le lecteur courant (s'il s'agit de disquettes).
           Si les deux lecteurs sont le même, diskcomp lira une
           une disquette à la fois et vous devrez les échanger.

Options :

  /1   Compare seulement le premier côté des disques, même
       s'il s'agit de disques double-face.
  /8   Compare seulement les 8 premiers secteurs par piste.
  /40  Compare seulement les 40 premières pistes (raccourci : /4).
  /b   Force diskcomp à utiliser le BIOS pour accéder au lecteur de
       disquettes. (Le BIOS est plus lent, mais peut être nécessaire
       pour la compatibilité sur certaines machines.)
  /?   Affiche l'aide.

Commentaires :

  Ce programme compare deux disques (max 32 Mo par chacun) et affiche
  la somme de contrôle md5 des contenus. Il indique si les contenus des
  disques sont différents (affiche un message et définit le
  CODE DE SORTIE (ERRORLEVEL) de façon appropriée).
  Si vous indiquez seulement un lecteur, vous devez changer les disques
  après que la première piste ait été lue. Si vous n'indiquez aucune
  option de lecteur, deux disques dans le lecteur courant seront
  comparés.

Exemples :

  diskcomp A: A:  compare les deux disques

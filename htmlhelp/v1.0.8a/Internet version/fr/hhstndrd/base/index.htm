Commandes externes :

  Ces fichiers font partie de la distribution FreeDOS standard.
  Pour les faire fonctionner vous aurez besoin d'un fichier exécutable
  existant (fichier *.com ou *.exe) et FreeDOS doit être déjà démarré.
  Vous pouvez les exécuter en tapant leur nom, par exemple "edit" ou
  "format" et - si nécessaire - le commutateur d'options / etc., cf.
  SYNTAXE.
  Le programme devrait être dans le répertoire courant, autrement vous
  devrez utiliser la commande CD ou PATH auparavant.

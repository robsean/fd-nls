Commande : listxdef

  LISTXDEF liste toutes les attributions de touches Xkeyb sur STDOUT.

Syntaxe :

  listxdef

Options :

  - aucune -

Commentaires :

  LISTXDEF est utilisé pour la sortie (sur STDOUT) du fichier de
  définitions actuellement utilisé par XKEYB (y compris toutes les
  touches définies avec KEYMAN). Il est le plus souvent utilisé pour
  enregistrer un fichier de définitions (c'est-à-dire créé en chargeant
  l'un des fichiers de définitions standard puis en assignant des
  macros supplémentaires en utilisant KEYMAN).
  Cela est fait en utilisant la commande :
    KEYMAN >xyz.KEY
  Veuillez consulter le manuel XKEYB.TXT pour davantage d'informations.
  COMMANDES CONNEXES :
    XKEYB     Version principale de XKEYB
    XKEYBRES  Version "allégée, lite" de XKEYB
    KEYMAN    Enregistreur de macro XKeyb
  LISTXDEF est fourni avec le paquet XKEYB.

Exemples:

  listxdef > monfichier.key sauvegarde toutes vos définitions de macros.

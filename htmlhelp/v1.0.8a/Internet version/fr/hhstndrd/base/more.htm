Commande : more

  MORE affiche un fichier texte ou la sortie d'une commande écran
  par écran.

Syntaxe :

  more < [lecteur][chemin]fichier
  commande | MORE [/Tn]
  MORE [/Tn] fichier
  MORE [/Tn] < file
  MORE [/?]
       lecteur     Spécifie la lettre du lecteur où se trouve le
                   fichier, par exemple C:
       chemin      Spécifie le chemin pour atteindre le fichier, par
                   exemple \exemple\
       nomfichier  Spécifie le fichier que vous souhaitez afficher, par
                   exemple test.txt
       commande    Une commande dont la sortie sera transmise au
                   programme more.

Options :

  Tn  Définit la taille du tabulateur à n, n devant être entre 1 et 9.
  /?  Affiche l'aide.
  Touches :
    Espace Page suivante (seulement lors de la visualisation d'un
    fichier)
    N n    Fichier suivant (seulement lors de la visualisation d'un
           fichier)
    Q q    Quitter le programme (seulement lors de la visualisation
           d'un fichier)

Commentaires :

  FreeDOS MORE prend en charge les jokers (ou caractères génériques
  * ou ?) dans les noms de fichiers.
  MORE prend en charge NLS (support des langues nationales).

Exemples :

  more < c:\config.sys
  fdisk /dump | more
  more /T4 *.sys
  more fdconfig.sys autoexec.bat

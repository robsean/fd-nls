Commande : klib

  KLIB est un bibliothécaire pour les fichiers KL : il regroupe les
  fichiers KL dans un seul fichier de bibliothèque. KLIB vous permet
  de créer des bibliothèques, de lister, d'ajouter et d'enlever des
  fichiers KL dans une bibliothèque et enfin de rendre cette dernière
  non éditable (de la fermer).

Syntaxe :

  KLIB  lib_nomfichier [{+|-}KL_nomfichier] [/B] [/Z] [/M=fabricant]
        [/D=Desc]
  KLIB  /?

Options :


  lib_nomfichier Le nom du fichier de bibliothèque
  KL_nomfichier  Nom du fichier KL qui doit être ajouté ou enlevé
                 Utilisez + pour AJOUTER le fichier KL (le fichier
                 doit exister) Utilisez - pour ENLEVER le fichier
                 KL (il doit être dans la bibliothèque)
  /B             Évite de créer des fichiers de sauvegarde
  /Z             Après l'opération la bibliothèque est fermée et ne
                 peut plus être éditée
  Fabricant      Nom du fabricant de la bibliothèque (création seulement)
  Desc           Description de la bibliothèque (création seulement)

Commentaires :

  Veuillez lire la documentation jointe pour en savoir plus sur le
 langage KEY et les formats de sortie.
  KLIB a les  CODES DE SORTIE (ERRORLEVEL) suivants :
    0                             = OK
    numéro d'erreur de syntaxe    = numéro d'erreur de syntaxe
    numéro de code critique + 100 = numéro de code d'erreur critique
  KLIB est fourni avec le paquet KC200.

Exemples :

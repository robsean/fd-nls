Commande : mscdex

  MSCDEX est le pilote de lecteur MS CD-ROM.
  FreeDOS utilise la commande SHSUCDX au lieu de MSCDEX.

Syntaxe :

  MSCDEX [/E/K/S/V] [/D:périphérique] [/L:lecteur] [/M:tampons]

Options :

  /E          Au cas où le système utiliserait de la mémoire
              d'extension, cette dernière devrait être utilisée.
  /K          Les structures de fichier KANJI sont utilisées au lieu
              des structures de fichier normales.
  /S          Le lecteur de CD-ROM peut être utilisé par les systèmes
              réseau MS.
  /V          Affiche des informations sur l'utilisation de la RAM.
  /D:pérpih.  Le nom du pilote de périphérique qui est installé pour
              contrôler le lecteur de CD-ROM.
  /L:lecteur  La lettre du lecteur qui doit être utilisée par le
              lecteur CD-ROM.
  /M:tampons  Le nombre de tampons qui sont utilisés pour stocker
              la table des chemins d'un lecteur de CD-ROM.

Commentaires :

  FreeDOS utilise la commande SHSUCDX au lieu de MSCDEX.

Exemples :

  - aucun -

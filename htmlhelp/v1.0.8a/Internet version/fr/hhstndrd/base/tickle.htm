Commande : tickle

  TICKLE est un read-ahead (lecture anticipée) pour LBACACHE. Cela
  augmente la vitesse des lecteurs de disquettes.

Syntaxe :

  tickle [/CHS] [/LBA] [/?]

Options :

  /CHS  Lecture anticipée pour anciens disques durs activée
        (mode CHS - Cylinder/Head/Sector).
  /LBA  Lecture anticipée pour nouveaux disques durs activée
        (mode LBA - Logical Block Adressing).
  /?    Affiche l'aide.

Commentaires :

  TICKLE lit toute la piste de la disquette en une fois lorsque
  vous lisez quelque chose sur cette piste - c'est plus rapide car
  la disquette doit tourner moins souvent.
  La lecture anticipée pour les disquettes est toujours activée.
  Chargez d'abord LBACACHE avec l'option FLOP activée !
  N'utilisez pas LOADHIGH avec TICKLE, risque de perte de
  données / problèmes DMA !

Exemples :

  lbacache FLOP
  tickle /lba

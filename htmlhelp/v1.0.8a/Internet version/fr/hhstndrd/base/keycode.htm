Commande : keycode

  KEYCODE vous aide à analyser la sortie du pilote de clavier.

Syntaxe :

  - aucune -

Options :

  - aucune -

Commentaires :

  Le compilateur KC est fourni avec un petit programme appelée KEYCODE,
  qui peut être utilisé pour déterminer la paire de scancode/caractère
  qui est produite par un programme d'amélioration du clavier (tel que
  FD-KEYB) vers le tampon clavier du BIOS. Pour l'exécuter, entrez
  simplement KEYCODE sur la ligne de commande. Appuyez sur les touches
  ou les combinaisons de touches que vous voulez analyser. Pour
  quitter, appuyez sur Échap ou F1.
  KEYCODE est fourni avec le paquet KC200.

Exemples :

  keycode

Commande : chkdsk

  CHKDSK vérifie un lecteur à la recherche d'erreur et tente
  optionnellement de les corriger. Il détaille aussi la taille du
  disque et sa structure.

Syntaxe :

  Chkdsk [volume] [/f] [/d fichiers] [/r] [/s] [/v] [/?]

Options :

  /f  tenter de corriger les erreurs trouvées.
  /d  afficher les fichiers indiqués et montrer le facteur de
      fragmentation.
  /r  analyser l'espace de données et tenter de récupérer les données
      illisibles
  /s  afficher seulement le résumé du lecteur.
  /v  afficher le nom du fichier lors de sa vérification.
  /?  affiche l'aide.

Commentaires :

   volume    Le disque, le lecteur ou la partition que vous souhaitez
             vérifier, par ex. : chkdsk a: vérifie le lecteur A. Si
             aucun volume n'est indiqué, le lecteur actuel est supposé.
   /f        Dit à  chkdsk de tenter de corriger les erreurs.
   /d fichie Cela  listera le niveau de fragmentation pour chaque
             fichier Si de nombreux fichiers sont très fragmentés,
             alors les opérations de disque peuvent être ralenties. Vous
             pouvez défragmenter votre lecteur de disque avec defrag.
             Cette option liste également les attributs de fichier pour
             chaque fichier, voyez attrib pour plus d'informations sur
             les attributs de fichier.
   /r        Analyse et tente de récupérer les données. Cela peut prendre
             longtemps !
   /s        Indique à chkdsk de ne pas vérifier le lecteur, mais
             seulement d'afficher son l'analyse.
   /v        Cela liste les fichiers lors de leur vérification.
             Notez que cela peut ralentir les choses, particulièrement
             s'il existe de nombreux fichiers.
  CHKDSK a les CODES DE SORTIE (ERRORLEVEL) suivants :
    0    lecteur en bon état

Exemples :

  chkdsk c: /f

Commande : diskcopy

  DISKCOPY copie le contenu d'une disquette vers une autre seconde
  disquette.
  Créé, copie et restaure également des fichiers d'images de disquette.
  Les images peuvent être utilisées par SHSUFDRV.

Syntaxe :

  Diskcopy [source][destination][/a][/v][/m][/i][/o][/1][/x][/d][/r]
           [/f][/t]
  Diskcopy [/?]
           source:      lecteur ou fichier d'image duquel effectuer une
                        copie.
           destination: lecteur ou fichier d'image vers lequel copier.

Options :

  /a  donner un avertissement sonore pour les actions de l'utilisateur.
  /v  vérifie les lectures et écritures.
  /m  utilise seulement la mémoire pour la copie de disques.
  /i  affiche l'utilisation de la mémoire (informatif).
  /o  écrase la destination, même s'il existe déjà (dans le cas d'un
      fichier d'image).
  /x  quitter toujours automatiquement.
  /d  suppose que le disque est déjà dans le lecteur.
  /r  passer en mode de récupération des erreurs de disque.
  /f  effectue une copie rapide (seuls sont copiés les secteurs occupés).
  /t  ne pas demander de disque cible si la copie du fichier image se
      fait sur le même disque.
  /1  no-op inclus pour la compatibilité MS-DOS. Sur d'autres versions
      de DOS, cela ne copiait que le premier côté.
  /?  Affiche l'aide.

Commentaires :

  Un signe moins après une option la désactive.
  Si la source et/ou la destination est un lecteur, par exemple "a:"
  alors elle est traitée comme une disquette, sinon elle est traitée
  comme un fichier image.
  La source et la destination peuvent être le même lecteur ; DISKCOPY
  copiera d'abord depuis la source, puis vous demandera de changer les
  disquettes. Ensuite il écrira sur la destination.
  DISKCOPY utilise le format Linux rawrite pour ses fichiers image.
  Donc les fichiers image de DISKCOPY peuvent être utilisés par Rawrite
  et Rawrite et réciproquement.
  DISKCOPY ne peut lire et écraser les disquettes GNU/Linux.
  Il peut, cependant, écrire une image disquette GNU/Linux sur un disque
  DOS. DISKCOPY prend en charge le NLS (support des langues nationales).

Exemples :

  diskcopy a: a:            Copie d'une disquette vers une autre
  diskcopy a: c:\image.img  Copie la disquette vers une image
                            sur C:\image.img
  diskcopy c:\image.img a:  Copie l'image vers une disquette

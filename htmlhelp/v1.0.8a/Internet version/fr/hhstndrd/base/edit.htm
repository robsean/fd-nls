Commande : edit

  EDIT est l'éditeur de texte de FreeDOS.

Syntaxe :

  edit [/B][/I][/H][/R][/?] [[lecteur][chemin]fichier]
       lecteur La lettre du lecteur, par exemple C:
       chemin  Le répertoire, par exemple \exemple\
       fichier Le fichier, par exemple test.txt. Les jokers peuvent être
               utilisés. Vous pouvez aussi ouvrir des fichiers depuis le
               programme.

Options :

  /B     Utilise un affichage noir et blanc (monochrome).
  /I     Utilise le schéma de couleur inversé.
  /H     Utilise 43/50 lignes sur EGA/VGA (la résolution vidéo/texte la
         plus élevée disponible).
  /R     Ouvre tous les fichiers en lecture seule.
  /?     Affiche l'aide.

Commentaires :

  Un éditeur comme EDIT peut être utilisé pour créer des fichiers texte
  simples.
  Il peut aussi être utilisé pour modifier les fichiers comme
  CONFIG.SYS / FDCONFIG.SYS et AUTOEXEC.BAT.
  EDIT possède un système incorporé d'aide complet.
  EDIT ne peut actuellement pas ouvrir les fichiers supérieurs à 64Ko
  en taille.

Exemples :

  edit c:\exemple.txt   ouvre c:\exemple.txt pour y écrire (le fichier
                        exemple.txt doit exister)
  edit c:\autoexec.bat
  edit c:\fdconfig.sys

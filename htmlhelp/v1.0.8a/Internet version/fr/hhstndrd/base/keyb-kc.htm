Commande : kc

 KC est le compilateur de fichiers de description de clavier du langage
 KEY.
 KC prend un fichier source de description de langue du clavier (en
 langage KEY) et le convertir en un fichier nu KeybCB ou un fichier
 KL destinés à être utilisés avec FD-KEYB.

Syntaxe :

  KC  nomfichiersource  [nomfichiersortie] [/B] [/?]
      nomfichiersource  Le nom du fichier source qui doit être ouvert.
                        L'extension KEY est ajoutée si aucune extension
                        n'est trouvée.
      nomfichiersortie  Le nom du fichier cible compilé. S'il est omis,
                        le même nom de fichier avec l'extension KL est
                        créé.

Options :

  /B  Crée des fichiers KeybCB nus, sans en-tête KL.
  /?  Affiche l'aide.

Commentaires :

  Veuillez lire la documentation jointe pour en savoir plus sur le
  language KEY et les formats de sortie.
  KC est fourni avec le paquet KC200.
  Les CODES DE SORTIE (ERRORLEVEL) sont 0 pour OK, le numéro d'erreur
  de syntaxe pour une erreur de syntaxe, et ajouter 100 au code
  d'erreur critique, pour les codes d'erreur critiques de KC.

Examples:

  KC FR                Compile FR.KEY en FR.KL
  KC SP.KEY SP.KCB /B  Compile SP.KEY dans un bloc de contrôle de
                       clavier nu SP.KCB

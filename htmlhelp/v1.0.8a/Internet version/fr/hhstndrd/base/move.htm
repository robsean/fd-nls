Commande : move

  MOVE déplace un ou plusieurs fichiers ou un répertoire vers un autre
  endroit.

Syntaxe :

  MOVE [/Y | /-Y] [/V] [/?] source1[, source2[,...]] destination

Options :

  source       Le nom du fichier ou du dossier que vous voulez déplacer
               (renommer).
  destination  L'endroit où vous souhaitez déplacer le ou les fichiers.
  /Y           Supprime la demande de confirmation pour écraser un
               fichier de destination existant.
  /-Y          Fait qu'une demande de confirmation lors de l'écrasement
               d'un fichier de destination existant sera affichée.
  /V           Vérifie chaque fichier lors de l'écriture sur la
               destination pour s'assurer que les fichiers de
               destination sont identiques aux fichiers sources.
  /?           Affiche l'aide.

Commentaires :

  Vous pouvez aussi déplacer des répertoires avec cet outil.
  MOVE prend en charge NLS (support des langues nationales).

Exemples :

  move C:\command.com D:\exemple\kommand.com

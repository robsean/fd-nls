Command : format

  FORMAT formate un disque dur ou une disquette. Cela prépare le média
  à être utilisé avec FreeDOS.

Syntaxe :

  FORMAT lecteur [/V[:label]] [/Q] [/U] [/F:taille] [/S] [/D]
  FORMAT lecteur [/V[:label]] [/Q] [/U] [/T:pistes /N:secteurs] [/S] [/D]
  FORMAT lecteur [/V[:label]] [/Q] [/U] [/4] [/S] [/D]
  FORMAT [/?]
         lecteur  La lettre du lecteur, par exemple C:

Options :

  /1            Formate un disque simple face (160 ko/180 ko).
  /4            Formate une disquette 160 ko/360 ko dans un lecteur de
                disquettes à 1,2 Mo.
  /8            Formate un disque 5-1/4 pouces avec 8 secteurs par piste
                (160 ko/320 ko, DOS 1.x).
  /A            Force l'alignement 4 ko pour le formatage FAT32.
  /B            Réserve de l'espace pour créer un disque amorçable (c'est
                factice et ne peut être combiné avec /s (sys).
  /D            Utilise le mode très verbeux et affiche la sortie de
                débogage. Pour les rapports de bugs.
  /F:taille     Spécifie la taille de la disquette à formater. Les
                tailles normales sont: 360, 720, 1200, 1440, or 2880
                (unité: kilooctets). 160, 180, 320 devrait aussi
                fonctionner. /F:0
                affiche la liste des tailles autorisées.
  /N:secteurs   Spécifie le nombre de secteurs par piste sur une
                disquette.
  /Q            Effectue un formatage rapide du disque. S'il n'est pas
                combiné avec /U, peut être UNFORMATé et préserve les
                marques de clusters défectueux (/Q /U ne le fait pas)
  /S            Appelle  SYS  pour rendre le disque amorçable et pour
                ajouter des fichiers système
  /T:pistes     Spécifie le nombre de pistes sur la disquette.
  /U            Formate inconditionnellement le disque. Effectue un
                formatage de bas niveau s'il s'agit d'une disquette.
                NOTE : Cette option peut durer TRÈS TRÈS longtemps !
                Appuyez sur échap et attendez pour quitter /U.
  /V:label      Spécifie un LABEL de volume pour le disque, stocke
                son horodatage (ne convient pas aux disques 160 ko/
                320 ko).
  /Y            Ignore la demande de confirmation.
  /Z:longhelp:  Donne des informations détaillées techniques sur
                l'utilisation.
  /Z:miroir     Sauvegarde les "données MIROIR pour UNFORMAT", sans
                formater le disque.
  /Z:seriously  Supprime la demande de confirmation lorsque vous utilisez
                format avec un disque dur.
  /Z:unformat   Annule le formatage du disque si les données MIROIR sont
                récentes.
  /?            Affiche l'aide.

Commentaires :

  Avertissement : bien que format supporte FAT32, le lancer sous Win9x
  pour créer des disques FAT32 ne fonctionne pas bien.
  Les modes pour les DISQUETTES sont : tente d'utiliser le formatage
  sûr rapide. Utilisez le formatage de bas niveau seulement quand c'est
  nécessaire. Le formatage rapide sûr sauvegarde les données miroir
  pour UNFORMAT.
  Les modes pour les DISQUES DURS sont : tente d'utiliser le formatage
  rapide sûr. Utilisez le formatage rapide complet que si c'est
  nécessaire. Le formatage rapide complet ne fait que remettre à zéro
  le système de fichiers. Si vous souhaitez forcer un formatage de bas
  niveau (disquettes) ou souhaitez analyser toute la surface et effacer
  toutes les données (disques durs), utilisez /U.
  FORMAT /Q /U effectue un formatage rapide complet (pas de formatage bas
  niveau / analyse / effacement !)
  FORMAT /Q effectue un formatage rapide (sauvegarde les données de
  miroir si possible)
  Les données de miroir écraseront toujours la fin de l'espace de
  données !
  FORMAT sélectionne automatiquement un mode (cf. ci-dessus) si vous ne
  choisissez ni /Q ni /U.

  Les types de FAT pris en charge sont : FAT12, FAT16, FAT32, tous avec
  les données miroirs / UNFORMAT.
  Les tailles de disquette prises en charge sont : 160 ko, 180 ko,
  320 ko, 360k 360 ko et 1200 ko pour les 5,25 pouces et 720 ko et
  1440 ko (2880 ko jamais testée jusqu'à présent) pour les lecteurs 3,5
  pouces. Les surformats pris en charge sont 400 ko, 800 ko, 1680 ko
  (et 3660 ko) avec plus de secteurs et 1494 ko (au lieu de 1200 ko)
  et 1743 (et 3486 ko) avec plus de pistes, aussi.
  Plus de pistes ne fonctionneront pas dans tous les lecteurs, utilisez
  à vos risques et périls.
  Avertissement : les versions anciennes de DOS ne peuvent utiliser les
  surformats qu'avec un pilote.
  720 ko dans 1440 ko nécessite un média de 720 ko. Utilisez un lecteur
  360 ko pour formater du 360 ko. Pour le formatage FAT32, vous pouvez
  utiliser le commutateur /A pour forcer l'alignement 4 Ko.
  FORMAT a les CODES DE SORTIE (ERRORLEVEL) suivants :
    0        OK,
    1        Secteurs défectueux détectés (seulement si les options /d
             ou /y ou /z:seriously sont actives)
    2        Écran d'aide affiché ou erreur de syntaxe,
    3        Annulation par l'utilisateur,
    4        Erreur de formatage,
    5        L'utilisateur n'a pas confirmé.
  Les niveaux d'erreur 1 et 2 sont des extensions dépassant le style
  MS FORMAT. Si vous utilisez l'indicateur /D, les niveaux d'erreur
  seront différents. Ils seront spécifiques à la version particulière
  de FORMAT.

  Pour 0.91t, par exemple :
    0        OK,
    1        Secteurs défectueux détectés,
    2        Aide affiché ou erreur de syntaxe,
    10..13   Erreur interaction utilisateur (pas de confirmation,
             erreur de syntaxe)
    20..22   Erreur Createfs  (création de système de fichiers),
    25..29   Erreur E/S du lecteur de disques,
    30..31   Erreur Recordbc (analyse du système de fichier existant),
    35..36   Erreur Uformat (formatage inconditionnel),
    38..39   Erreur Savefs (SafeFormat / Miroir)
    40..52   Erreur de disquette,
    55..59   Erreur de disque dur,
    60..62   Erreur principale : type de lecteur inadapté, par
             exemple CD-ROM,
   128..255  Erreur critique (userint: (code d'erreur DOS ou BIOS) | 128)

Exemples :

  format a: /f:1440   formate une disquette de 1,4 Mo
  format c: /v:mon_dd formate un disque dur, label: mon_dd

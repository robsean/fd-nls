Commande : graphics

  GRAPHICS permet à la touche Impr Écr (Impression d'écran) d'imprimer
  des écrans graphiques.
  GRAPHICS fonctionne en mode de compatibilité avec MS GRAPHICS.

Syntaxe :

  GRAPHICS type [options]

Options :

  /B             Active le traitement de la palette de couleurs de
                 fond 0 CGA.
  /I             Impression inversée (utile s'il y a beaucoup de zones
                 sombres
                 sur l'écran, pour économiser de l'encre / du toner :
                 imprimera
                 en noir là où l'écran est blanc, etc.).
  /LCD           ignorée (pas encore prise en charge).
  /LOW           Ne charge pas haut (pas d'option MS).
  /PRINTBOX:LCD  ignorée (pas encore prise en charge).
  /PRINTBOX:STD  ignorée (pas encore prise en charge).
                 (essayez d'utiliser /LCD pour afficher une explication).
                 Ces options vous permettront de définir un rapport
                 largeur / hauteur d'écran différent de 4: 3, qui est
                 actuellement toujours supposé.
  /R             Imprime le noir en tant que noir et le blanc en tant
                 que blanc.
  Types d'imprimantes prises en charge :
    HPPCL (PCL), EPSON (ESC/P), POSTSCRIPT (PS)
  Vous pouvez utiliser les éléments suivants pour le type :
    Compatible HP PCL :
      HPDEFAULT          (compatible HP PCL)
      DESKJET            (compatible HP PCL)
      RUGGEDWRITER       (compatible HP PCL)
      RUGGEDWRITERWIDE   (compatible HP PCL)
      THINKJET           (compatible HP PCL)
    Compatible EPSON :
      EPSON              De nombreuses imprimantes à impact / à ruban
                         fonctionnent avec "EPSON"
    Compatible POSTSCRIPT :
      LASERJET           (compatible POSTSCRIPT)
      LASERJET II        (compatible POSTSCRIPT)
      QUIETJET           (compatible POSTSCRIPT)
      QUIETJETPLU        (compatible POSTSCRIPT)
    Certains d'entre eux sont équivalents, tels que POSTSCRIPT et PS.
  Types d'imprimantes non prises en charge :
    Imprimante à ruban couleur IBM PC (couleur1, couleur4, couleur8)
    Imprimante thermique convertible IBM (thermique)
    Imprimante graphique IBM (graphics[wide])
    IBM proprinter
    IBM quietwriter

Commentaires :

  HP (GRAPH-HP - PCL) permet les options supplémentaires suivantes :
    /1  Utilise LPT1 (valeur par défaut)
    /2  Utilise LPT2
    /3  Utilise LPT3
    /C  Pour 300 dpi au lieu de 600 dpi
    /E  Mode d'économies
    /R  Pour le tramage aléatoire au lieu du tramage ordonné
    /?  Affiche l'aide.
  PS (GRAPH-PS - Postscript) permet les options supplémentaires
  suivantes :
    /1  Utilise LPT1 (valeur par défaut)
    /2  Utilise LPT2
    /3  Utilise LPT3
    /C  Mode de compatibilité (HP Laserjet)
    /E  Mode d'économies (seulement  50 % de densité)
    /?  Affiche l'aide.
  PIN (GRAPHPIN - EPSON) permet les options supplémentaires suivantes :
    /1  Utilise LPT1 (valeur par défaut)
    /2  Utilise LPT2
    /3  Utilise LPT3
    /9  Mode à 9 aiguilles (120x72, 60x72 avec /C)
    /C  Mode de compatibilité (120x60dpi)
    /E  Mode d'économies (seulement 50 % de densité)
    /R  Utilise le tramage aléatoire, pas le tramage ordonné
    /X  Envoie des données d'impression pour les lignes entières
        au lieu de les envoyer sous forme de blocs
    /?  Affiche l'aide.
  Après avoir chargé le pilote vous pouvez imprimer des captures
  d'écran en utilisant 'maj - Imp Écr' y compris en mode graphique.

Exemples :

  graphics HPDEFAULT /low /I /E /1

Voir également :

  graphics-ps...

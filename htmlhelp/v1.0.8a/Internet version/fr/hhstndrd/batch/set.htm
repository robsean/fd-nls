Commande : set (commande command.com)

  SET affiche, définit et efface les variables d'environnement.
  SET est une commande de FICHIER BATCH / AUTOEXEC.BAT.
  Elle peut également être utilisée sur la ligne de commande.

Syntaxe :

  SET [/C] [/P] [/E] [/U] [variable[=[chaîne]]]
  SET [/?]
  variable  Spécifie le nom de la variable d'environnement.
  chaîne    Spécifie une série de caractères à attribuer à la variable.
  * Si aucune chaîne n'est spécifiée, la variable est effacée de
    l'environnement.
  Tapez SET sans paramètres pour afficher les variables d'environnement
  actuelles. Tapez SET VAR pour afficher la valeur de VAR.

Options :

  /C  Force à conserver la casse de la variable ; par défaut la variable
      est en majuscules, si elle ne se trouve pas déjà dans
      l'environnement, sinon la casse est conservée (ne fonctionne
      que pour les nouvelles variables).
  /P  Invite l'utilisateur avec une chaîne, et affecte l'entrée
      utilisateur à la variable.
  /E  Définit une variable donnée à la première ligne de sortie de
      la commande pointée par [chaîne].
  /U  Modifie la casse de [chaîne] en majuscules.
  /?  Affiche l'aide.

Commentaires :

  SET peut aussi être utilisée en dehors d'un fichier batch.
  SET est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  DANS UN FICHIER .BAT / AUTOEXEC.BAT:
    set             affiche la liste des variables d'environnement
                    actuels
    set DRIVE=Z     définit la variable d'environnement DRIVE à la
                    valeur donnée Z
    set DRIVE=      efface la variable DRIVE de l'environnement

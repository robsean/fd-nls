Commande : cls

  CLS efface l'écran.
  CLS est une commande de FICHIER BATCH / AUTOEXEC.BAT.
  Elle peut également être utilisée sur la ligne de commande.

Syntaxe :

  cls

Options :

  - aucune -

Commentaires :

  CLS enlève tout le texte sur l'écran.
  CLS peut aussi être utilisée en dehors d'un fichier batch.
  CLS est une commande interne à command.com et n'a besoin d'aucun
  autre fichier pour fonctionner.

Exemples :

  DANS UN FICHIER .BAT / AUTOEXEC.BAT OU sur la LIGNE DE COMMANDE :
    cls

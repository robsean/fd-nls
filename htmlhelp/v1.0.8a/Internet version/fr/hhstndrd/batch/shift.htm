Commande : shift

  SHIFT modifie la position des paramètres remplaçables dans un fichier
  batch.
    (Utilisée pour décaler les paramètres remplaçables des fichiers
    batch).
  SHIFT est une commande de FICHIER BATCH / AUTOEXEC.BAT.

Syntaxe :

  SHIFT [DOWN]

Options :

  DOWN décale la fenêtre d'argument vers le début (%0) ; sinon vers
  la fin.

Commentaires :

  Avec down, %0 prend la valeur que %1 avait, et ainsi de suite.
  Sinon, %2 prend la valeur que %1 avait, etc.
  SHIFT est une commande interne à command.com et ne nécessite aucun
  autre aucun autre fichier pour fonctionner.

Exemples :

  - aucun -

Commande : whatis

  Affiche les descriptions des programmes fondées sur les fichiers
  Appinfo (.LSM).
  Utilisez APROPOS pour trouver des programmes appropriés pour le
  sujet donné.
  Utilisez WHATIS avec un nom de programme pour obtenir une description
  en une ligne dudit programme.
  Utilisez WHATIS sans nom de programme pour obtenir une description en
  une ligne de tous les programmes et utilitaires externes de votre
  installation de FreeDOS.
  Les commandes internes (commandes de command.com) ne seront pas
  affichées.

Syntaxe :

  APROPOS [options] sujet
          Rechercher des commandes en rapport avec le sujet donné
  WHATIS [options] [commande]
          Affiche une description sur une ligne de la commande donnée.
          Si aucune commande n'est spécifiée, toutes les commandes et
          leur description sur une ligne sont listées
  WHATIS  /B [options]
          Reconstruit la base de données whatis pour être utilisée avec
          WHATIS et APROPOS

Options :

  /S [chemin]  Chemin non défini par défaut vers les fichiers lsm
               appinfo.
  /D [chemin]  Chemin non défini par défaut pour l'emplacement de
               la base de données whatis.
  /?           Affiche l'aide.

Commentaires :

  Ce sont les clones des commandes GNU/Linux "apropos et whatis"
  APROPOS et WHATIS fondent leur aide sur les fichiers lsm stockés dans
  le répertoire APPINFO.
  La première fois qu'elle est exécutée, APROPOS / WHATIS construit une
  base de données et la stocke dans le répertoire d'aide de
  l'installation de FreeDOS. Cette base de données peut être recréée
  en utilisant :
    WHATIS /B

Exemples :

  apropos edit  affichera une liste d'éditeurs
  apropos disk  affichera une liste d'utilitaires de disque
  whatis edit   affichera une courte information au sujet d'edit
  whatis        affichera une courte information sur les commandes DOS

Hilfe zum Thema "Syntax":

  Bei den meisten Hilfethemen werden Sie etwas ähnliches wie das hier
  vorfinden:

    command required [optional] [/switch] [/A|/B]

  Hier eine Erklärung der unterschiedlichen Teile dieser Zeile:

    command
  Das ist der Name des Befehls.
   
    required
  Alles was dem Befehlsnamen folgt und nicht in eckigen Klammern ist, 
  [und] ist ein erforderlicher Parameter.
  Der Befehl wird einen Fehler ausgeben, wenn er diese Information 
  nicht erhält.
   
    [optional]
  Optionen werden in eckigen Klammern geschrieben [und]. Diese 
  verändern die Art und Weise wie der Befehl läuft. Die eckigen Klammern 
  müssen nicht eingetippt werden, wenn der Befehl benutzt wird.

    [/switch]
  Optionen, die mit einem / beginnen werden Schalter (switch) genannt. 
  Das / Symbol muß normalerweise eingetippt werden, wenn der Schalter 
  benützt wird.

    [/A|/B]
  Wenn ein | Symbol angezeigt wird, bedeutet dies, daß es zwei 
  Möglichkeiten gibt.
  Entweder ist /A oder /B erlaubt, aber nicht beide.
  Das | muß nicht eingetippt werden, wenn der Befehl benutzt wird.

Netzwerken unter FreeDOS - CIFS (auch: NetBEUI)

  Das Common Internet File System (CIFS = gemeinsames Internetdatei-
  system) ist eine Methode für zwei oder mehr Computer, um Daten in
  einem Netzwerk miteinander zu teilen. Es ist abgeleitet von dem
  Server Message Block (SMB) Protokoll, welches auf das NetBIOS folgte.
  NetBIOS war ursprünglich eine Erweiterung der API des PC BIOS. Es
  wurde von Microsoft und anderen für mehr Verwendbarkeit und ver-
  schiedene Betriebssysteme erweitert. Microsoft lieferte einmal einen
  Client für DOS, aber dieser ist seitdem auferstanden und verschwun-
  den. Jacco de Leeuw liefert Details über Netzwerksoftware für DOS,
  wo man sie finden kann und wie man sie installiert.
  Unglücklicherweise ist ein Samba Open Source Client für DOS bisher
  nicht entwickelt worden und auch wenn der MS Client funktioniert, ist
  er nicht besonders kompatibel mit anderen CIFS Systemen. Siehe auch
  die unten angegebenen Links für weitere Infos.

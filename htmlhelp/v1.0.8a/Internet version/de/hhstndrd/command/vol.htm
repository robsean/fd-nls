Befehl: vol

  VOL zeigt das Laufwerk-Volumelabel und, falls vorhanden, die
  Seriennummer an.

Syntax:

  VOL [Laufwerk]
  VOL [/?]
      Laufwerk  Spezifiziert den Laufwerksbuchstaben des anzuzeigenden
                Volumelabels, z.B. C:

Optionen:

  /?  Zeigt die Hilfe

Kommentar:

  Jeder Laufwerksbuchstabe kann einen Laufwerksnamen haben. Sie können
  ihn mit dem Befehl LABEL erstellen. Der Name darf nicht länger als
  11 Buchstaben sein.
  VOL ist ein interner Befehl, d.h. der Befehl ist in der command.com
  enthalten. Es wird zur Ausführung des Befehls kein zusätzlicher
  Befehl (Datei) benötigt.

Beispiel:

  vol c:
    wenn Sie auf Laufwerk C: sind:
  vol

Befehl: cd

  CD zeigt den Namen oder wechselt das aktuelle Verzeichnis.

Syntax:

  CD [Laufwerk][Pfad]
  CD [..] [-] [/] [/?]
     Laufwerk  Der Laufwerksbuchstabe, z.B: C:
     Pfad      Der zu wählende Pfad, z.B.: \Beispiel\

Optionen:

  ..       Spezifiziert, daß Sie zum übergeordneten Verzeichnis 
           wechseln möchten.
  -        Falls das "vorheriges Verzeichnis"-Feature aktiviert 
           ist, Wechsel zum vorherigen Verzeichnis.
  \        Wechselt in das Stammverzeichnis.
  /?       Zeigt die Hilfe

Kommentar:

  Geben Sie CD Laufwerk: ein, um das aktuelle Verzeichnis des 
  angegebenen Laufwerks anzuzeigen.
  Geben Sie CD ohne Parameter ein, um das aktuelle Laufwerk und 
  Verzeichnis anzuzeigen.
  Wenn kein Laufwerk angegeben wird, wird vermutet, daß das derzeit 
  verwendete Laufwerk gemeint ist.
  Wenn kein Pfad angegeben wird, wird der derzeit verwendete Pfad
  angezeigt.
  Sie können statt CD auch CHDIR verwenden. 
  CD ist ein interner Befehl, d.h. der Befehl ist in der command.com
  enthalten. Es wird zur Ausführung des Befehls kein zusätzlicher 
  Befehl (Datei) benötigt.

Beispiel:

    cd c:\fdos      wechselt in das Verzeichnis c:\fdos
    cd ..           wechselt von c:\fdos zurück nach c:\
    cd \            wechselt aus c:\ irgendeinverzeichnis
                    \irgendeinunterverzeichnis zurück nach c:\
  wenn sie auf c:\ sind:
    cd fdos         wechselt in das Verzeichnis c:\fdos
    cd bin          wechselt in das Verzeichnis c:\fdos\bin
    cd d:\beispiel  wechselt den Pfad auf d: nach d:\beispiel; 
                    wechselt aber nicht grundsätzlich von c: nach d:

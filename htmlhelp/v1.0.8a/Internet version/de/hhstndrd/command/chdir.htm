Befehl: chdir

  CHDIR zeigt den Namen oder wechselt das aktuelle Verzeichnis.

Syntax:

  CHDIR [Laufwerk][Pfad]
  CHDIR [..] [-] [/] [/?]
        Laufwerk  Der Laufwerksbuchstabe, z.B: C:
        Pfad      Der zu wählende Pfad, z.B.: \Beispiel\

Optionen:

  ..       Spezifiziert, daß Sie zum übergeordneten Verzeichnis 
           wechseln möchten.
  -        Falls das "vorheriges Verzeichnis"-Feature aktiviert 
           ist, Wechsel zum vorherigen Verzeichnis.
  \        Wechselt in das Stammverzeichnis.
  /?       Zeigt die Hilfe

Kommentar:

  Geben Sie CHDIR Laufwerk: ein, um das aktuelle Verzeichnis des 
  angegebenen Laufwerks anzuzeigen.
  Geben Sie CHDIR ohne Parameter ein, um das aktuelle Laufwerk und 
  Verzeichnis anzuzeigen.
  Wenn kein Laufwerk angegeben wird, wird vermutet, daß das derzeit 
  verwendete Laufwerk gemeint ist.
  Wenn kein Pfad angegeben wird, wird der derzeit verwendete Pfad
  angezeigt.
  Sie können statt CHDIR auch CD verwenden. 
  CHDIR ist ein interner Befehl, d.h. der Befehl ist in der command.com
  enthalten. Es wird zur Ausführung des Befehls kein zusätzlicher 
  Befehl (Datei) benötigt.

Beispiel:

    chdir c:\fdos     wechselt in das Verzeichnis c:\fdos
    chdir ..          wechselt von c:\fdos zurück nach c:\
    chdir \           wechselt aus c:\ irgendeinverzeichnis
                      \irgendeinunterverzeichnis zurück nach c:\
  wenn sie auf c:\ sind:
    chdir fdos        wechselt in das Verzeichnis c:\fdos
    chdir bin         wechselt in das Verzeichnis c:\fdos\bin
    cd d:\beispiel    wechselt den Pfad auf d: nach d:\beispiel; 
                      wechselt aber nicht grundsätzlich von 
                      c: nach d:

Befehl: which

  WHICH sucht die ausführbare Datei für jeden angegebenen Befehl
  und zeigt ihn (incl. Pfad) an.

Syntax:

  which Befehl [/?]

Optionen:

  Befehl  Der zu suchende Befehl
  /?      Zeigt die Hilfe

Kommentar:

  WHICH ist ein Tool aus der UNIX-Welt, das es Benutzern erlaubt,
  ausführbare Dateien zu finden, die in der PATH Umgebungsvariablen
  aufgelistet sind.
  WHICH ist ein interner Befehl, d.h. der Befehl ist in der command.com
  enthalten. Es wird zur Ausführung des Befehls kein zusätzlicher
  Befehl (Datei) benötigt.

Beispiel:

  which xcopy  sucht die ausführbare Datei für den xcopy Befehl. 
               Die Ausgabe lautet: "xcopy   C:\FDOS\BIN\xcopy.exe"
  which copy   Die Ausgabe lautet: "copy"   (copy ist ein command.com
                                             Befehl)

Befehl: ver

  VER zeigt die FreeDOS Versionsnummer an. Standardmäßig wird
  nur die Version der Command Shell (FreeCOM) ausgegeben.

Syntax:

  VER [/R] [/W] [/D] [/C]

Optionen:

  /R  Zeigt die Kernel- und DOS-Versionsnummer an.
  /W  Zeigt die FreeDOS Command Shell Gewährleistung an.
  /D  Zeigt die FreeDOS Command Shell Weitergabeinformationen an.
  /C  Der FreeCOM Anerkennungsbereich. Er listet die Unterstützer auf.

Kommentar:

  VER ist ein interner Befehl, d.h. der Befehl ist in der command.com
  enthalten. Es wird zur Ausführung des Befehls kein zusätzlicher
  Befehl (Datei) benötigt.

Beispiel:

  ver /R
  ver /W
  ver /D
  ver /C

Siehe auch:

  callver

Befehl: ctty

  CTTY wechselt das Terminalgerät, das zur Steuerung Ihres 
  Systems verwendet wird.

Syntax:

 CTTY Gerät
 CTTY [/?]

Optionen:

  Gerät  Das Terminalgerät das Sie verwenden möchten.
          Gueltige Geräte sind: prn, lpt1, lpt2, lpt3, con,
          aux, com1, com2, com3, com4, NUL.
  /?      Zeigt die Hilfe

Kommentar:

  CTTY ist ein interner Befehl, d.h. der Befehl ist in der command.com
  enthalten. Es wird zur Ausführung des Befehls kein zusätzlicher 
  Befehl (Datei) benötigt.

Beispiel:

  CTTY lpt1
  CTTY /?

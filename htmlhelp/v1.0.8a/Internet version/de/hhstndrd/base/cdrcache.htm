Befehl: cdrcache.sys

  CDRCACHE.SYS ist ein CD-Laufwerk Zwischenspeicherprogramm. Es ver-
  bessert die Laufwerksperformance. Es setzt XMS Speicher und mindestens
  einen 386er Computer voraus.
  CDRCACHE.SYS muß in der CONFIG.SYS / FDCONFIG.SYS geladen werden.
  Alternativ kann CDRCACHE.SYS während des Betriebes mit DEVLOAD
  nachträglich geladen werden.

Syntax:

  In der config.sys entweder:
    device=cdrcache.sys Gerät Name cachesize
  oder:
    devicehigh=cdrcache.sys Gerät Name cachesize

Optionen:

  Gerät     Der gleiche Name für das Gerät, das Sie vorher mit Ihrem 
             CD-ROM-Gerätetreiber geladen haben. 
  Name       Ein neuer Name, den Sie an shsucdx in der autoexec.bat
             anstelle von Gerät weitergeben werden.
  cachesize  Gibt die Cachepuffergröße in Kilobytes an. Der
             Cache wird im XMS Speicher abgelegt.
             Minimaler Wert ist 128.
             Maximaler Wert ist 65280.
             Werte unter 100 sind aus Kompatibilitätsgründen
             zu vorhergehenden Versionen von CDRCACHE.SYS gültig.
             Bei Werten unter 100 wird die Größe als Mehrfaches
             von 0,25 MB interpretiert.
  Nach dem Laden können Sie den Treiber wie folgt kontrollieren:
  Syntax:
    echo Befehl > Name
  Befehl:
    F     Leert den Cache.
    S     Zeigt die Statistik an.
    I     Zeigt die Statistik mit zusätzlichen technischen Informationen.
    C     Loescht die Statistik.
    Q     Stiller Modus.
    N     Normale Ausgabe.
    V     Umfangreiche Ausgabe / Spurmodus.
    0     Beendet das Cachen.
    1     Setze das Cachen fort (z.B. nach einem Stop).
    ?     Zeigt den Hilfetext.
    Name  Der Parameter, der an CDRCACHE.SYS weitergegeben wurde
          (siehe Beispiel).

Kommentar:

  Laden Sie CDRCACHE.SYS NACHDEM Sie Ihren CD-ROM-Gerätetreiber geladen
  haben (in der Regel uide.sys).
  Nachdem CDRCACHE.SYS (2003) auf LBACACHE.SYS basiert, lesen Sie
  bitte die LBAcache.sys Dokumentation für zusätzliche technische /
  programmiererische Details.

Beispiel:

  In der config.sys / fdconfig.sys:
    DEVICE=c:\fdos\bin\xcdrom.sys /D:FDCD0000
    DEVICE=c:\fdos\bin\CDRCACHE.SYS FDCD0000 CDRCACH$ 2048
  dann in der autoexec.bat:
    c:\fdos\bin\shsucdx /D:CDRCACH$
  Dies weist einen 2.048 KB (2 MB) großen Cachepuffer für ein
  einzelnes IDE CD-Laufwerk zu.
    echo F > cdrcach$    (leert den Cache)
    echo SCF > cdrcach$  (Zeigt die Statistik, löscht danach die
                         Statistik und leert danach den Cache)
    echo S > cdrcach$    (zeigt nur die Statistik)

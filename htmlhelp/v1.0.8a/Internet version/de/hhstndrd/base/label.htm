Befehl: label

  LABEL erzeugt, verändert oder löscht den Laufwerksnamen eines Mediums 
  (Diskette, Festplatte usw.).

Syntax:

  LABEL [Laufwerk][Bezeichnung]
  LABEL [/?]
        Laufwerk  Gibt den Laufwerksbuchstaben an, den Sie benennen wollen,
                  z.B. C:. Wenn nichts angegeben wird, wird das derzeit 
                  aktive Laufwerk genommen.

Optionen:

 Bezeichnung  Spezifiziert den neuen Namen, mit dem Sie das Laufwerk
              benennen wollen. Wenn nichts angegeben wird, frägt Sie
              label nach einem Namen.
  /?          Zeigt die Hilfe

Kommentar:

  Das Label (der Name) darf nicht länger als 11 Buchstaben sein.

Beispiel:

  label C:
  label C: PARTITON-C
  label D: my_files

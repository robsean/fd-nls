Command: scankbd

  SCANKBD scans current keyboard layout (XKEYB or other keyboard driver)
  into the configuration file, generating a .KEY compatible file.

Syntax:

  scankbd

Options:

  - none -

Comments:

  LISTXDEF lists the XKEYB layout and all the macro definitions
  to the console. Most probably you would like to do:
    LISTXDEF > MYFILE.KEY
  to save all your macro definitions.

Examples:

  scankbd
  listxdef > german.key

See also:

  keyb
  keyman
  listxdef
  mkeyb
  xkeyb

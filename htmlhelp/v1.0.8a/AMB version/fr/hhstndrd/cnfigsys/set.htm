Commande : set (commande config.sys)

  SET définit une variable d'environnement à la valeur donnée.
  SET est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  SET [variable[=[chaîne]]]
      variable  Spécifie le nom de la variable d'environnement.
      chaîne    Spécifie une série de caractères à attribuer à
                la variable.

Options :

  - aucune -

Commentaires :

  SET définit la variable d'environnement à la valeur donnée.
  SET est une commande interne à kernel.sys et ne nécessite
  aucun autre fichier pour fonctionner.
  Il existe aussi la commande SET de COMMAND.COM.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS :
    set home=C:\FDOS\bin

Commande : menudefault

  MENUDEFAULT définit une option par défaut temporisée pour un menu
  config.sys / fdconfig.sys.
  MENUDEFAULT est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  menudefault=optionpardéfaut,délai

Options :

  délai           durée du délai en secondes
  optionpardéfaut l'option standard utilisée si le délai est écoulé

Commentaires :

  MENUDEFAULT définit une option par défaut temporisée pour un menu
  config.sys / fdconfig.sys : si aucune touche n'est enfoncée pendant
  les secondes du 'délai', le choix par défaut est activé. Enfoncer
  une touche arrête le compte à rebours, le kernel attendra indéfiniment
  que l'utilisateur fasse un choix.
  Notez que certaines machines virtuelles comme Bochs peuvent avoir un
  chronométrage erroné (trop rapide).
  MENUDEFAULT est une commande interne à kernel.sys et ne nécessite
  aucun autre fichier pour fonctionner.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS :
    menucolor 15,1
    MENU Choisissez avec sagesse :
    MENU
    MENU 0)    pilote normal (himem et emm386)
    MENU 1)    pilote alternatif (himemx et jemm386)
    MENUDEFAULT=0,5
    0?DEVICE=A:\himem.exe
    1?DEVICE=A:\himemx.exe
    0?DEVICE=A:\emm386.exe
    1?DEVICE=A:\jemm386.exe
  Le menu attendra 5 secondes. Si aucune touche n'est enfoncée pendant
  ce délai, l'option 0 sera automatiquement sélectionnée. Menucolor
  15,1 modifie la couleur en du texte blanc sur un arrière-plan bleu.
    0 choisit himem.exe et emm386.exe,
    1 choisit himem.exe et jemm386.exe

Commande : eecho

  EECHO affiche des messages.
  EECHO est une commande CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  EECHO [message] avec une séquence d'échappement ANSI

Options :

  message  Le message que vous souhaitez afficher à l'utilisateur.

Commentaires :

  EECHO permet de faire écho aux séquences d'échappement ANSI (redéfinit
  la saisie au clavier par exemple). Utilisez un signe dollar ($) pour
  représenter le caractère d'échappement ANSI. Remarque : nécessite un
  pilote ANSI comme NANSI chargé avant utilisation.
  EECHO est une commande interne à kernel.sys et ne nécessite
  aucun autre fichier pour fonctionner.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS:
    EECHO je me sens $[33;44;1m blue :-)

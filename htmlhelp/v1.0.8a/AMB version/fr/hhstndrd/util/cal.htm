Commande : cal

  CAL est un petit calendrier en ligne de commande.

Syntaxe :

  cal
  cal mm/aaaa
  cal aaaa
  cal /?
      mm    le mois que vous souhaitez voir (2 chiffres)
      aaaa  l'année que vous souhaitez voir (4 chiffres)

Options :

  /?  Affiche l'aide.

Commentaires :

  CAL est un petit (900 octets) calendrier en ligne de commande. Il
  peut afficher le mois actuel, un mois spécifique, ou un calendrier
  annuel. Cal prend en charge le format de calendrier grégorien pour
  les années 1753-2399.

Exemples :

  cal          affiche le mois actuel
  cal 04/1959  affiche avril 1959
  cal 1959     affiche tous les mois de l'année 1959

Commande : fdpkg

  FDPKG installe des paquets FreeDOS à la fois au format spécialisé
  et au format standard.

Syntaxe :

  FDPKG [/INSTALL] [/REMOVE] [/CHECK] [/CONFIGURE] [/E] PKGX.ZIP ...

Options :

  /INSTALL       Mode d'installation ; installer le paquet (valeur
                 par défaut).
  /REMOVE        Mode d'effacement ; enlever le paquet.
  /CHECK         Mode de vérification ; vérifier les dépendances
                 du paquet.
  /CONFIGURE     Mode de configuration ; configurer le paquet.
  /DESC[:CHAMP]  Afficher la description depuis un LSM. Le champ
                 (optionnel) est le champ du LSM à afficher. Les
                 LSM sont recherchés d'abord dans le répertoire courant,
                 sinon dans %DOSDIR%\APPINFO
  /DISPLAY       Lister tous les paquets installés correspondant à
                 l'argument.
  /E             Extraire seulement, ne pas vérifier les dépendances
                 (sauf si déjà installé), ne pas configurer.
  /S[:|=]ARG     Basculer les barres obliques inverses d'ARG en
                 barres obliques.
  /[-]F|/[-]Y    Forcer ; ne pas poser de questions (PEUT ÊTRE
                 DANGEREUX).
  PKGX.ZIP       Le paquet à installer, des noms multiples / jokers sont
                 acceptés.
  /?[E]|/H[E]    Affiche l'aide. /?E ou /HE affichent l'aide étendue.

Commentaires :

  FDPKG prend en charge NLS (National Language Support, prise en
  charge des langues nationales).

Exemples :

  FDPKG /INSTALL PKGX.ZIP
  FDPKG /REMOVE PKGX.ZIP

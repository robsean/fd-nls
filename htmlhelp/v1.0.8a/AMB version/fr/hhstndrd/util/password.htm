Commande : password

  PASSWORD est un outil de mot de passe pour DOS qui gère plusieurs
  utilisateurs.

Syntaxe :

  password [/adduser] [/login] [/remuser] [/?]

Options :

  /adduser  Ajoute un nouvel utilisateur au programme. Doit être exécuté
            en premier.
  /login    Démarre la procédure d'identification. Si vous entrez un
            utilisateur existant créé avec /adduser et son mot de
            passe, vous pourrez démarrer FreeDOS.
  /remuser  Enlève un utilisateur du programme.
  /?        Affiche l'aide.

Commentaires :

  Vous devez ajouter "PASSWORD" au début d'AUTOEXEC.BAT.
  Vous devriez ajouter "SWITCHES=/F" au début de CONFIG.SYS pour
  empêcher un utilisateur de contourner CONFIG.SYS / AUTOEXEC.BAT ET le
  mot de passe en tapant "F5"."
  Vous devriez aussi modifier les réglages du BIOS : amorcer depuis le
  disque dur en premier (n'oubliez pas de définir un mot de passe pour
  le BIOS). PASSWORD prend en charge NLS (national language support,
  prise en charge des langues nationales).

Exemples :

  SUR LA LIGNE DE COMMANDE :
    password /adduser  Vous devez d'abord entrer les utilisateurs
                       possibles et leur mot de passe, puis :
  DANS CONFIG.SYS / FDCONFIG.SYS :
    switches=/f        Ajoutez cela au début de config.sys pour empêcher
                       un utilisateur de contourner config.sys /
                       autoexec.bat avec la touche "F5".
  DANS AUTOEXEC.BAT:
    password /login    L'utilisateur doit désormais entrer son nom
                       d'utilisateur et son mot de passe.
  Lors du prochain démarrage il vous sera demandé votre nom
  d'utilisateur et votre mot de passe.

Command: foxcalc

  FOXCALC est une calculatrice simple pour DOS avec une interface
  textuelle. Le programme prend en charge la souris, donc prenez
  soin de lancer votre pilote de souris avant d'exécuter FOXCALC ;
  elle sera bien plus facile de l'utiliser.

Syntaxe :

  - aucune -

Options :

  Si vous ne pouvez vraiment pas utiliser une souris, vous trouverez
  ci-dessous une liste de touches de raccourci. Cette liste devrait
  apparaître tout seule si FoxCalc ne peut trouver un pilote de souris.
  Touche         Fonction

  0-9            0-9 (chiffres)
  "+"            +   (addition)
  "-"            -   (soustraction)
  "=" or ENTER   =   (retour chariot)
  "."            .   (comma)
  "*"            *   (multiplication)
  "/"            /   (division)
  "%"            %   (percent)
  "]"                (racine carrée)
  "["                (carré)
  "^"                (exponentiation)
  "!"            x!  (factorisation)
  ":"            MOD (modulo)
  "m"            M+  (addition à la mémoire)
  "M"            M-  (soustraction de la mémoire)
  "r"            MR  (rappel de la mémoire)
  "c"            MC  (effacement de la mémoire)
  "\"            +/- (changer le signe)
  ESC            OFF (quitter)
  DEL            AC  (tout effacer)
  BackSpace      C   (effacer)
  "s"            snd (activer ou désactiver le son)
  "?"            ?   (au sujet du programme)

Commentaires :

  CALCUL
  FoxCalc utilise la bibliothèque GMP (GNU Multi-Precision) et la
  bibliothèque MPFR pour tous les calculs. Cela le rend précis et
  très fiable. Toutes les opérations sont effectuées avec une précision
  de 60 chiffres. Les nombres en virgule flottante qui ne rentrent
  pas dans la fenêtre sont tronqués sont aucun arrondi. Lors de
  l'utilisation de la fonctionnalité MOD dans FoxCalc, vous pouvez
  obtenir des résultats différents des autres calculatrices ou langages
  de programmation. Ce n'est pas un bug. L'opération modulo sur les
  nombres négatifs peut être effectué selon plusieurs méthodes.
  Généralement, les ordinateurs traitent une opération "X MOD Y" en
  utilisant les valeurs absolues de X et Y et changent le signe du
  résultat selon le signe de X. C'est la façon la plus facile, et
  c'est ainsi que les processeurs Intel fonctionnent. FoxCalc se
  comporte différemment. Donc, l'opération "7 mod -3" peut être
  calculée comme 1 sur les autres calculatrices, alors que FoxCalc
  retournera un résultat de "-2". Les deux résultats sont corrects.
  PAGE DE CODE & TRADUCTIONS :
  Comme il existe des touches qui sont décrites avec des caractères
  ASCII supérieurs à ASCII #127, FoxCalc détecte la page de code du
  système et modifie la description des touches si la page de code ne
  peut les afficher correctement. Par exemple, la touche carré est
  "¹" sur CP437 et "SQR" sur CP850. Les séparateurs (séparateur décimal
  et séparateur de milliers) sont copiés des réglages du système. Si
  vous souhaitez les modifier, modifiez simplement les réglages de votre
  système (généralement en utilisant COUNTRY.SYS).

Exemples :

  foxcalc

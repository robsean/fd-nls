Commande : print

  PRINT imprime un fichier en arrière-plan alors que vous exécutez
  d'autres commandes DOS.

Syntaxe :

   print [options] [[lecteur:][chemin]fichier]
         [[lecteur:][chemin]fichier] [...]
           lecteur:chemin\fichier  Le fichier que vous voulez imprimer.

Options :

  /D:périph  Spécifie le nom du périphérique sur laquelle l'impression
             aura lieu. Si l'option est omise, Print utilisera le
             périphérique LPT: par défaut.
  /B:taille  Définit la taille du tampon, en octets, entre 512 et 16384.
             La valeur par défaut est 512.
  /U:ticks   Le nombre de ticks d'horloge, entre 1 et 255, que Print doit
             attendre pour que l'imprimante devienne disponible avant
             d'imprimer le travail.
  /M:ticks   Le nombre maximal de ticks pour envoyer un seul caractère à
             l'imprimante, compris entre 1 et 255.
  /S:ticks   Le nombre de ticks d'horloge, compris entre 1 et 255, pour
             le processus de spouleur d'arrière-plan. La valeur par
             défaut est 2.
             De grands nombres vont accélérer l'impression mais
             ralentiront les autres programmes DOS.
  /Q:nn      Le nombre de fichiers qui seront autorisés dans la file
             d'attente d'impression, entre 1 et 32. La valeur par
             défaut est 10.
  /T         Enlève tous les fichiers dans la file d'attente
             d'impression. Arrête le travail d'impression en cours.
  /C         Annule des travaux dans la file d'attente d'impression.
             Si cette option est utilisée avec une liste de noms de
             fichier, seuls ces fichiers sont enlevés de la file
             d'attente. Si l'option est utilisée seule, tous les travaux
             sont annulés (mais le travail en cours n'est pas annulé.)
  /P         Ajoute un fichier à la file d'attente d'impression. Si
             cette option précède une liste de fichiers, seuls ces
             fichiers seront imprimés. Vous pouvez utiliser /P et /C
             sur la même ligne de commande.

Commentaires :

  Si aucune option n'est donnée à PRINT mais seulement des fichiers, /P
  est supposé. Si aucune option ni fichier n'est donné, print affichera
  le contenu de la file d'attente d'impression.
  Ce PRINT ici prend seulement /1 /2 /3 en tant qu'options équivalentes à
  "D:lpt1" etc. ou un nom de fichier pour imprimer un fichier. Veuillez
  utiliser l'outil libre  PRINTQ pour "print fichier", "print /t"
  (effacer la file d'attente y compris le travail actuel), "print /c"
  (nettoyer la file d'attente) et "print /p fichier" (ajouter le fichier
  à la file d'attente).

Exemples :

  print c:\foo.txt c:\test.txt

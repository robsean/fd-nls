Commande : scankbd

  SCANKBD analyse la disposition de clavier actuelle (XKEYB ou autre
  pilote de clavier) dans un fichier de configuration, générant un
  fichier compatible .KEY.

Syntaxe :

  scankbd

Options :

  - aucune -

Commentaires :

  LISTXDEF liste la disposition XKEYB et toutes les définitions
  de macros sur la console. Très probablement, vous aimeriez faire :
    LISTXDEF > MONFICHIER.KEY
  pour sauvegarder toutes vos définitions de macros.

Exemples :

  scankbd
  listxdef > allemand.key

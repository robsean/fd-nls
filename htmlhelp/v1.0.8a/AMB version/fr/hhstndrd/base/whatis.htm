Commande : whatis

  WHATIS affiche la description des programmes fondée sur les fichiers
  Appinfo (.LSM).
  Utilisez APROPOS pour trouver des
  programmes qui conviennent à un sujet donné.
  Utilisez WHATIS avec un nom de programme pour obtenir une description
  en une ligne de ce programme.
  Utilisez WHATIS sans nom de programme pour obtenir une description
  en une ligne de tous les programmes et utilitaires externes sur votre
  installation FreeDOS. Les commandes internes (les commandes de
  command.com) ne sont pas affichées.

Syntaxe :

  APROPOS [options] sujet
          Trouver des commandes liées au sujet donné
  WHATIS [options] [commande]
          Affiche une description en une ligne de la commande donnée.
          Si aucune commande n'est spécifiée, toutes les commandes et
          leurs descriptions en une ligne sont listées
  WHATIS  /B [options]
          Reconstruit la base de données whatis pour utilisation avec
          WHATIS et APROPOS

Options :

  /S [chemin]  Chemin d'accès qui n'est pas celui par défaut pour les
               fichiers lsm appinfo.
  /D [chemin]  Chemin d'accès qui n'est pas celui par défaut pour
               l'emplacement de la base de données whatis.
  /?         Affiche l'aide.

Commentaires :

  Ce sont les clones FreeDOS des commandes GNU/Linux "apropos et whatis"
  APROPOS et WHATIS fondent leur aide
  sur les fichiers lsm stockés dans le répertoire APPINFO.
  Lors de la première exécution, APROPOS / WHATIS construit une base
  de données et la stocke dans le répertoire d'aide de l'installation
  FreeDOS. Cette base de données peut être recréée en utilisant :
    WHATIS /B

Exemples :

  apropos edit  affichera une liste d'éditeurs
  apropos disk  affichera une liste d'utilitaires de disque
  whatis edit   affichera une courte information au sujet d'edit
  whatis        affichera une courte information au sujet des
                commandes DOS

Commande : defrag

  DEFRAG optimise le temps de chargements depuis un disque en rassemblant
  les fragments de fichiers.

Syntax:

  defrag [/C] [lecteur] [{/F|/U|/DF|/FF|/FD}] [/Sordre[-]] [/B] [/X] [/A]
         [/FO] [/?]
         drive  C'est la lettre du lecteur pour le lecteur que vous
                souhaitez défragmenter. Par exemple : defrag c:

Options :

  /C   Utilise seulement la version en ligne de commande. Defrag
       n'utilisera pas son interface utilisateur pilotée par les menus
       si vous choisissez cette option. Il communiquera à travers la
       ligne de commande. Lorsque vous utilisez le commutateur /C,
       vous devez utiliser l'option [lecteur] comme décrit ci-dessus,
       par exemple defrag /c c:
  /F   Optimise entièrement le lecteur spécifié. Avec cette option,
       defrag effectuera une optimisation complète, en s'assurant que
       tous les fichiers seront au début du disque, sans écart entre eux.
  /U   Avec cette option, defrag défragmentera les fichiers mais des
       écarts peuvent subsister entre ces fichiers.
  /FF  Optimise entièrement le lecteur, déplaçant tous les fichiers
       d'abord.
  /DF  Optimise entièrement le lecteur, déplaçant tous les répertoires
       d'abord.
  /FD  Optimise entièrement le lecteur, déplaçant ensemble tous les
       fichiers d'un même répertoire.
  /S   Trier les fichiers par ordre spécifié.
       ordre: N  par Nom (alphabétique)
              E  par Extension (alphabétique)
              D  par Date et temps (les plus anciens en premier)
              S  par taille (les plus petits en premier)
              -  suffixe pour trier par ordre descendant.
  /B   Redémarre l'ordinateur après l'optimisation.
  /X   Force defrag à quitter dès qu'il aura terminé l'optimisation.
  /A   À utiliser avec l'option /C, defrag donnera un avertissement
       sonore avant d'effectuer des actions.
  /FO  À utiliser avec l'option /C, cela indique à defrag de donner une
       sortie d'informations plus complète.
  /?   Affiche l'aide.

Commentaires :

  Tapez DEFRAG /no-ops pour une liste de no-ops (/SKIPHIGH, /LCD, /BW,
  /GO). Tapez DEFRAG /contrib pour une liste des contributeurs.
  Il est recommandé d'utiliser DEFRAG avec l'interface utilisateur
  (sans : /C). Qu'est-ce que la fragmentation du disque ?
  Lorsque des fichiers sont effacés, ou rendus plus petits, cela laisse
  des écarts entre les données stockées sur le disque. Si un fichier est
  créé, ou si sa taille augmente, il sera placé dans ces espaces, mais
  si cet espace n'est pas assez grand, le fichier sera continué ailleurs
  sur le disque. À la suite de nombreuses opérations de ce type, les
  données peuvent être très éparpillées. Cette fragmentation ralentit
  l'accès au disque. La fragmentation est un processus normal sur les
  disques DOS.

  Quand dois-je utiliser defrag ?
  Utilisez defrag occasionnellement, ou après l'installation de gros
  programmes ou après la création de nombreux (petits) fichiers.
  DEFRAG a les CODES DE SORTIE
  (ERRORLEVEL) suivants :
    0  succès
    1  erreur interne
    2  disque trop plein pour être défragmenté
    3  annulé par l'utilisateur
    4  erreur générale
    5  erreur de lecture
    6  erreur d'écriture
    7  erreur d'allocation FAT
    8  erreur de la mémoire
    9  mémoire insuffisante

Exemples :

  defrag c: /F

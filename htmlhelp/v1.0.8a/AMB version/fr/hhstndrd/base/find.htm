Commande : find

  FIND affiche des lignes contenant une chaîne dans un ou plusieurs
  fichiers textuels.

Syntaxe :

  FIND [ /C ] [ /I ] [ /N ] [ /V ] "chaîne" [[lecteur][chemin]fichier1]
  FIND [/?]
       lecteur   la lettre du lecteur, par exemple C:
       chemin    le répertoire, par exemple \exemple\
       fichier1  les fichiers dans lesquels vous souhaitez effectuer la
                 recherche, par exemple test.exe
       Note :  Si vous ne voulez pas spécifier un fichier, find prendra
               son entrée de la console. Tapez Ctrl-Z pour terminer.

Options :

  /C        Affiche uniquement le nombre de lignes correspondantes.
  /I        Ignore la casse pendant la comparaison.
  /N        Numérote les lignes affichées, commençant à 1.
  /V        Inverse la recherche : affiche les lignes qui ne contiennent
            PAS la chaîne.
  "chaîne"  La chaîne de signes que vous souhaitez trouver, par exemple
            "eau" ou "0123". La chaîne doit être entourée de guillemets
            anglais.
  /?        Affiche l'aide.

Commentaires :

  FIND prend en charge le NLS (le support des langues nationales).
  FIND a les CODES DE SORTIE (ERRORLEVEL) suivants :
    0  trouvé
    1  rien trouvé
    2  recherche non terminée

Exemples :

  find /c loadhigh c:\autoexec.bat
  find /c devicehigh c:\config.sys

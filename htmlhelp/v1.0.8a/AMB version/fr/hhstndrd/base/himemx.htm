Commande : himemx

  HIMEMX est le gestionnaire de mémoire haute.
  HIMEMX doit être chargé dans CONFIG.SYS / FDCONFIG.SYS.

Syntaxe :

  HIMEMX [/MAX=####] [/METHOD:xxx] [/HMAMIN=n] [/NUMHANDLES=m]
        [/TESTMEM:ON|OFF] [/VERBOSE] [/NOABOVE16] [/X] [/LOG]

Options :

  /MAX=#####       Limite la mémoire contrôlée par XMM à ##### Ko.
  /METHOD:xxx      Spécifie la méthode à utiliser pour la gestion A20.
                   Valeurs possibles pour xxx :
                   ALWAYSON    Suppose que la ligne A20 est toujours
                               allumée
                   BIOS        Utilise le BIOS pour basculer la ligne A20
                   FAST        Utilise le port 92h, contourne le test
                               INT 15h
                   PS2         Utilise le port 92h, contourne le test
                               PS/2
                   KBC         Utilise le contrôleur de clavier
                   PORT92      Utilise toujours le port 92h
  /HMAMIN=n        Spécifie le nombre minimal de Ko de HMA qu'un
                   programme doit demander pour obtenir l'accès au
                   HMA (par défaut : 0Ko).
  /NOX2MAX32       Pas de limite pour les rapports de mémoire XMS 2.0
                   libre/disponible (valeur par défaut).
  /NUMHANDLES=m    Spécifie le nombre d'indicateurs XMS disponibles
                   (déf : 48).
  /TESTMEM:ON|OFF  Effectue ou ignore le test de mémoire étendue
                   (déf: OFF).
  /VERBOSE         Donne des informations supplémentaires pendant
                   l'installation.
  /NOABOVE16       N'utilise pas la fonction E801h d'INT 15h pour
                   détecter > 64Mo.
  /X               N'utilise pas la fonction E820h d'INT 15h pour
                   détecter > 64Mo.
  /X2MAX32         Limite le rapport de mémoire XMS 2.0 libre/dispo.
                   à 32Mo-1Ko
  /LOG             Affiche l'activité du pilote sur l'écran.
  /?               Affiche l'aide.

Commentaires :

  HIMEMX nécessite un processeur 80386+. JEMM386 peut être chargé
  après ce pilote. Chargez HIMEMX comme DEVICE=HIMEMX.EXE dans CONFIG.SYS
  (bien que ce soit un fichier ".exe" !)

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS:
    device=c:\fdos\bin\himemx.exe
    device=c:\fdos\bin\jemm386.exe

Commande : apropos

  APROPOS affiche la description des programmes en se fondant sur les
  fichiers Appinfo (.LSM).
  Utilisez APROPOS pour trouver des programmes appropriés pour un
  sujet donné.
  Utilisez WHATIS avec un nom de programme pour obtenir une description
  sur une ligne du programme.
  Utilisez WHATIS sans nom de programme pour obtenir une descrioption
  sur une ligne de tous les programmes externes de votre installation de
  FreeDOS. Les commandes internes ne sont pas affichées.

Syntaxe :

  APROPOS [options] sujet
          Trouver des commandes relatives au sujet donné.
  WHATIS [options] [commande]
          Affiche une description sur une ligne de la commande donnée.
          Si aucune commande n'est spécifiée, toutes les commandes et
          leurs description sur une ligne sont listées.
  WHATIS  /B [options]
          Reconstruit la base de données de WHATIS et APROPOS

Options :

  /S [path]  Chemin d'accès (non par défaut) aux fichiers lsm appinfo.
  /D [path]  Chemin d'accès (non par défaut) pour la base de données
             whatis.
  /?         Affiche l'aide.

Commentaires :

  Il s'agit là de clones FreeDOS des commandes Linux "apropos et whatis"
  APROPOS et WHATIS fondent leur aide sur les fichiers lsm
  stockés dans le répertoire APPINFO.
  La première fois qu'il est exécuté, APROPOS / WHATIS construit une
  base de données et le stocke dans le répertoire d'aide de FreeDOS.
  Cette base de données peut être recréé en utilisant :
    WHATIS /B

Exemples :

  apropos edit  affichera une liste d'éditeurs
  apropos disk  affichera une liste d'utilitaires de disques
  whatis edit   affichera une courte information au sujet d'edit
  whatis        affichera une courte information au sujet des
                commandes DOS

Commande : atapicdd.sys

  ATAPICDD.SYS est un pilote générique de CD-ROM pour les CD-ROM IDE
  (ATAPI). Il est toujours en développement. Veuillez utiliser UIDE.SYS à
  la place. ATAPICDD.SYS doit être chargé dans CONFIG.SYS / FDCONFIG.SYS.
  Lorsque FreeDOS fonctionne déjà, vous pouvez charger ATAPICDD.SYS plus
  tard avec DEVLOAD.
  ATAPICDD.SYS est périmé. Vous devriez utiliser UIDE.SYS à la place.

Syntaxe :

   device=[lecteur][chemin]atapicdd.sys [options]
          lecteur  la lettre du lecteur, par ex. C:
          chemin   le répertoire, par ex. \exemple\

Options :

  /D:nom  Définit le nom de l'appareil en tant que "nom". Valeur par
           défaut :
           FDCD0000. Ce nom doit aussi être mentionné dans autoexec.bat
           lors du chargement de shsucdx / mscdex.
  /N:x     Définit le nombre d'appareils à rechercher sur x.
           Par défaut : 1

Commentaires :

  ATAPICDD.SYS est chargé dans CONFIG.SYS / FDCONFIG.SYS.
  L'option /D: xxxxyyyy doit être identique à celui dans SHSUCDX
  (MSCDEX) qui est chargé dans AUTOEXEC.BAT.

Exemples :

  IN CONFIG.SYS / FDCONFIG.SYS:
    device=c:\fdos\bin\atapicdd.sys /d: fdc0000 /n:2

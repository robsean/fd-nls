Commande : comp

  COMP compare deux fichiers.

Syntaxe :

  COMP [/options] [lecteur][chemin]fichier1 [lecteur][chemin]fichier2
       où [fichier1] et [fichier2] sont les noms (avec éventuellement
       des jokers) de fichier ou de dossiers. Si [fichier2] est omis,
       le répertoire actuel est utilisé.
       lecteur  la lettre du lecteur, par exemple C:
       chemin   le répertoire, par exemple \exemple\
       fichier1 le premier fichier à comparer, par exemple test1.txt
       fichier2 le second fichier à comparer, par exemple test2.txt

Options :

  #       Ne pas afficher plus de # erreurs par fichier. Si # = 0, ne
          pas limiter le nombre d'erreurs. La valeur par défaut est 0
          si [fichier1] fait référence à un seul fichier, 10 s'il est
          question de plusieurs.
  /d      Affiche les différences au format décimal. Par défaut, c'est
          hex.
  /a      Affiche les différences en utilisant des caractères.
  /L      Affiche le numéro de ligne pour les différences qui se
          produisent.
  /n=nnn  Compare les nnn premières ligne des deux fichiers, même si
          les fichiers sont de taille différente.
  /c      Ignore la casse (la valeur par défaut est de ne pas l'ignorer).
  /?      Affiche l'aide.

Commentaires :

  - aucun -

Exemples :

  comp c:\program\help.txt c:\program\help1.txt

Voir également :

  diskcomp
  fc
  md5sum

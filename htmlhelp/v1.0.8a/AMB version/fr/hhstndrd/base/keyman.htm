Commande : keyman

  KEYMAN est un enregistreur de macros simple qui est installé en tant
  que programme résident dans la mémoire et qui permet de simples
  (re)définitions des touches.

Syntaxe :

 [loadhigh] keyman

Options :

  - aucune -

Commentaires :

  KEYMAN ne peut être chargé que si XKEYB est déjà résident en mémoire.
  Il est activé par un raccourci clavier (F11 par défaut). Après
  activation KEYMAN affichera une fenêtre sur l'écran. Vous pouvez
  maintenant soit appuyer sur une touche à laquelle vous voulez attribuer
  une nouvelle macro, ou bien appuyer sur le raccourci clavier à nouveau
  pour le redéfinir.
  Si vous avez appuyé sur une touche qui avait déjà une macro qui lui
  était attribuée, la macro actuelle sera affichée sur la fenêtre. Cette
  macro peut être éditée en utilisant la touche RETOUR ARRIÈRE pour
  effacer les appuis de touches affichés (pour inclure un appui sur
  RETOUR ARRIÈRE en tant que partie intégrante de la macro, appuyez sur
  Alt et sur "8" sur le pavé numérique).
  Après un autre appui sur le raccourci clavier, la macro sera attribuée
  et la fenêtre disparaîtra.
  Note :
  Si la touche sur laquelle vous avez appuyé n'avait PAS déjà une macro
  qui lui était attribuée, cette touche sera considérée comme le premier
  appui de la macro (c'est-à-dire que si vous avez appuyé sur CTRL-D,
  la macro consistera initialement de l'appui sur "CTRL-D").
  Si la nouvelle macro comportait plus d'un caractère, un XString sera
  créé automatiquement et sera attribué à la touche (si assez d'espace
  Xstring avait été alloué avec le commutateur /Xnn lors de
  l'installation de XKEYB).
  Si vous avez appuyé sur le raccourci clavier après l'activation de
  KEYMAN, ce dernier vous demandera d'appuyer sur le nouveau raccourci
  clavier. Après avoir appuyé sur une touche, la fenêtre disparaîtra et
  la touche choisie deviendra le nouveau raccourci clavier.
  Si vous souhaitez enregistrer une attribution de touches créée
  interactivement, vous pouvez le faire avec la commande LISTXDEF
  > xyz.KEY
  KEYMAN est fourni avec le paquet XKEYB.

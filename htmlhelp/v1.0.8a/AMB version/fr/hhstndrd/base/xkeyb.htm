Commande : xkeyb

  XKEYB est un pilote de clavier avec une certaine ressemblance avec
  le bien connu KEYB.

Syntaxe :

  xkeyb fichier de configuration [/Xnn] [/U] [/Q]
    où fichier de configuration est un code de pays ("GR.key"
    ou "US.key")

Options :

  /Xnn  Cette option devrait être utilisée pour définir explicitement la
        quantité d'espace de stockage à réserver pour XStrings (où nn
        est la taille de mémoire indiquée en octets).
  /U    (Uninstall), enlève la copie résidente d'Xkeyb de la mémoire
        (si elle est installée).
  /Q    (Quiet), supprime la sortie du paragraphe LIST dans les fichiers
        de configuration.

Commentaires :

  Comme avec KEYB, XKEYB est aussi un pilote de clavier pour les
  ordinateurs MS-DOS, qui est installé comme un programme résident et
  qui peut être configuré pour différents pays au moyen de paramètres.
  Cependant XKEYB autorise bien plus d'options de configuration que KEYB.
  Ainsi, vous pouvez configurer XKEYB non seulement pour divers pays,
  mais aussi pour vos préférences personnelles.
  D'un côté, XKEYB permet l'attribution libre de toutes les touches à
  tous les niveaux (normalement avec MAJ, CTRL, ALT, and CTRL-ALT), et
  d'un autre côté il offre la flexibilité d'attribuer des chaînes de
  caractères à certaines touches à certains niveaux.
  De plus, pour chaque touche il est possible de déterminer séparément
  si elle est influencée par VERR MAJ, VERR NUM ou ARRÊT DÉFIL, et si
  les touches MAJ, CTRL, et ALT déterminent les valeurs émises.
  Il est offert au programmeur une IP (Interface de Programme) pour
  XKEYB. Elle permet l'accès à de nombreuses fonctionnalités d'XKEYB.
  XKEYB est fourni avec le paquet XKEYB.

Exemples :

  XKEYB GR /X480
  XKEYB /X480 GR.key

Commande : cdrcache.sys

  CDRCACHE.SYS est un programme de cache de lecteur CD. Il améliore les
  performances du lecteur. Nécessite la mémoire XMS, et au moins un
  ordinateur 386. CDRCACHE.SYS doit être chargé dans CONFIG.SYS /
  FDCONFIG.SYS.
  Lorsque FreeDOS fonctionne déjà, vous pouvez charger CDRCACHE.SYS plus
  tard avec DEVLOAD.

Syntaxe :

  In config.sys, soit :
    périphérique=cdrcache.sys nom du périphérique taille du cache
  soit :
    devicehigh=cdrcache.sys nom du périphérique taille du cache

Options:

  périphér   Le même nom que le périphérique que vous avez déjà chargé
             avec votre pilote de lecteur de CD-ROM.
  nom        Un nouveau nom que vous transmettrez à shsucdx dans
             autoexec.bat
  cachesize  Spécifie la taille du cache tampon en Ko. Le cache est
             stocké dans la mémoire XMS.
             La valeur minimale est 128
             La valeur maximale est 65280
             Les valeurs sous 100 sont valides pour la compatibilité
             avec les versions précédentes de CDRCACHE.SYS. Sous 100,
             les tailles sont interprétées comme étant des multiples
             de 0.25 Mo.
  Contrôle du pilote après le chargement :
    Utilisation :
  echo commande > nom
  Commande :
    F     Vide le cache.
    S     Affiche les statistiques.
    I     Affiche les statistiques avec des informations techniques.
    C     Efface les statistiques.
    Q     Mode silencieux.
    N     Verbosité normale.
    V     Mode verbeux/ de traçage.
    0     Arrête la mise en cache.
    1     Continue la mise en cache (après un arrêt par exemple).
    ?     Affiche l'aide.
    nom   Est le paramètre transmis à CDRCACHE.SYS (cf. l'exemple).

Commentaires :

  Chargez CDRCACHE.SYS après le chargement de votre pilote de CD-ROM
  device (généralement uide.sys).
  Comme CDRCACHE.SYS (2003) est basé sur LBACACHE.SYS, veuillez lire
  la documentation LBAcache pour des détails techniques et de
  programmation additionnels.

Exemples :

  Dans config.sys / fdconfig.sys:
    DEVICE=c:\fdos\bin\xcdrom.sys /D:FDCD0000
    DEVICE=c:\fdos\bin\CDRCACHE.SYS FDCD0000 CDRCACH$ 2048
  puis dans autoexec.bat:
    c:\fdos\bin\shsucdx /D:CDRCACH$
  Cela alloue un cache tampon de 2048KB (2MB) pour un seul lecteur
  CD IDE CD.
    echo F > cdrcach$    (vider le cache)
    echo SCF > cdrcach$  (afficher les statistiques, puis les effacer
                         et vider le cache)
    echo S > cdrcach$    (afficher seulement les statistiques)

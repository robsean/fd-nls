Commande : label

  LABEL crée, modifie ou efface l'étiquette (label) de volume d'un disque.

Syntaxe :

  LABEL  [lecteur][label]
  LABEL  [/?]
         lecteur Spécifie à quel lecteur vous souhaitez attribuer une
         étiquette, par exemple C: S'il est omis, le lecteur courant est
         supposé.

Options :

  label  Spécifie la nouvelle étiquette que vous souhaitez attribuer au
         lecteur. S'il est omis, label vous le demandera.
  /?     Affiche l'aide.

Commentaires :

  Le label (étiquette ou nom) d'une partition peut avoir jusqu'à 11
  lettres.

Exemples :

  label C:
  label C: PARTITION-C
  label D: mes_fichiers

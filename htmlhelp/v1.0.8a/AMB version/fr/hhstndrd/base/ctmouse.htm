Commande : ctmouse (Cutemouse)

  CuteMouse (CTMOUSE) est le pilote de souris de FreeDOS.

Syntaxe :

  ctmouse [options]
  LH ctmouse [options]
  LOADHIGH ctmouse [options]

Options :

  /3        force le mode 3 boutons (pour les souris Microsoft et PS/2).
  /B        Annule le lancement si les services de souris sont déjà
            présents.
  /L        Intervertit les boutons gauche et droit.
  /M        Essaye les systèmes de souris *anciens* / Genius pour les
            souris non-PnP.
  /N        Charge CuteMouse en tant que nouveau TSR, même si CuteMouse
            est déjà chargé (utile pour les fichiers batch qui déchargent
            CuteMouse de la mémoire à la fin).
  /O        Active la détection PS2 et BIOS de la molette (peut planter).
  /P        Force le mode de souris PS/2.
  /R[h[v]]  Résolution horizontale/verticale : h,v = 1-9, ou 0 pour auto.
            (aucune valeur = utilise la valeur par défaut : automatique
            pour h, ou "comme h" pour v).
  /S[c[i]]  Force le mode souris série sur le port COM c (c=1-4, la
            valeur par défaut est COM1) avec la ligne IRQ i (i=2-7,
            la valeur par défaut est IRQ4 pour COM 1/3 et IRQ3 pour
            COM 2/4).
  /U        Désinstalle le pilote, enlève le TSR de la mémoire.
  /V        Recherche inversée : trouve la souris PS/2 après la souris
            série.
  /Y        Force la recherche de souris série mais contrairement à 
            l'option /S, ne désactive pas le support disable PS/2.
  /W        N'autorise pas CuteMouse à se déplacer dans l'UMB.
  /?        Affiche l'aide.

Commentaires :

  CTMOUSE ne prend pas en charge les souris USB.
  CTMOUSE peut être chargé dans CONFIG.SYS / FDCONFIG.SYS ou dans
  AUTOEXEC.BAT ou à la ligne de commande.
  CTMOUSE2 est une nouvelle branche de CuteMouse qui supporte la molette.
  Il fournit une API pour la molette afin que les programmes FreeDOS
  puissent utiliser la molette de la souris.
  Visitez le site de CuteMouse sur http://cutemouse.sourceforge.net
  CTMOUSE a les CODES DE SORTIE (ERRORLEVEL) suivants :
    0  Souris PS/2, Microsoft ou Logitech trouvé et CTMOUSE installé ;
       déchargé avec succès ; option /? utilisée.
    1  CTMOUSE installé en mode Systèmes de Souris ;
       échec du déchargement car CTMOUSE n'était pas chargé.
    2  La partie résidente a basculé en mode PS/2, Microsoft ou Logitech;
       échec du déchargement car les interruptions du pilote ont été
       interceptées.
    3  La partie résidente a basculé en mode Systèmes de Souris.
    4  Les services de souris sont déjà présents (retourné pour
       l'option /B).
    5  Souris non trouvée ; utilisation d'une option invalide.

Exemples :

  DANS CONFIG.SYS / FDCONFIG.SYS:
    devicehigh=C:\fdos\bin\ctmouse  charge ctmouse en mémoire haute
  DANS AUTOEXEC.BAT OU SUR LA LIGNE DE COMMANDE :
    lh ctmouse  charge ctmouse en mémoire haute
    ctmouse /u  décharge le pilote de souris de la mémoire
    ctmouse /?  Affiche l'aide (les options)

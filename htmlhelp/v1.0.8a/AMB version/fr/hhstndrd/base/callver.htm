Command: callver

  CALLVER définit le numéro de version de l'interpréteur de commandes
  FreeCOM.
  CALLVER lance un programme (dans un nouveau shell, en utilisant
  COMSPEC) avec un faux numéro de version de DOS, ou modifie
  globalement (seulement dans FreeDOS) la version de DOS rapportée
  (CALLVER 0.00 revient au point de départ)

Syntaxe :

  CALLVER x.xx [PROGRAMME [ARGUMENTS]][/?]

Options :

  PROGRAMME  Nom du programme que vous souhaitez démarrer.
  ARGUMENTS  Options du programme que vous souhaitez démarrer.
  /?         Affiche l'aide.

Commentaires :

  x.xx est le numéro de version, chaque x peut être un chiffre de 0 à 9.
  Cette commande peut être utilisée pour les anciens programmes qui
  recherchent une version de DOS précise.

Exemples :

  callver 6.20
  callver 6.22 format c: /q
  callver 0.00               -> affiche 7.10

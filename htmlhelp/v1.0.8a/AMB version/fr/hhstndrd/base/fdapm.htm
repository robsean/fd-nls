Commande : fdapm

  FDAPM est un outil avancé de gestion de l'alimentation.

Syntaxe :

  fdapm [Options]

Options :

 (aucune)   Si aucune option n'est donnée, la valeur par défaut est
            d'afficher à la fois les sorties INFO et STATS (si le shell
            est chargé, sinon lorsqu'il est chargé depuis INSTALL, la
            valeur par défaut est APMDOS).
  APMoff    Désactive le mode d'économie d'énergie APM du BIOS brut.
            Alias de compatibilité avec la syntaxe MS POWER : "OFF".
  OFF       Identique à APMoff, pour la compatibilité avec MS power.
  APMbios   Active le mode d'économie d'énergie APM du BIOS brut. Alias
            de compatibilité avec la syntaxe MS POWER : "STD".
  STD       Identique à APMbios, pour la compatibilité avec MS power.
  APMDOS    Cela devrait vous permettre de réaliser des économies
            d'énergie maximales. C'est le seul mode qui fait que FDAPM
            reste résident dans la RAM. Pour arrêter, sélectionnez
            APMbios/APMoff.
  ADV:MAX   Économies d'énergie maximales, pour la compatibilité avec
            MS power.
  ADV:MIN   Économies d'énergie minimales, pour la compatibilité avec
            MS power.
  ADV:REG   Économies d'énergies restreintes, pour la compatibilité avec
            MS power.
  INFO      Affiche des informations sur l'état APM. Affiche des choses
            comme le pourcentage de remplissage de la batterie.
  STATS     Affiche des informations sur l'état APM/ APMDOS. Affiche
            des choses comme le pourcentage du temps de processeur
            occupé, si FDAPM ou POWER sont résidents et chargés dans
            mémoire.
  FLUSH     Vide les caches du disque (réécrit les données sur
            le disque).
  SPINUP    Réveille les disques IDE à nouveau.
  SPINDOWN  Vide les caches disque et arrête les disques IDE sur les
            2 premiers contrôleurs IDE. Utilisez à vos risques et
            périls. Les disques vont redémarrer lorsque vous les
            utiliserez à nouveau. Les disques ont "l'intelligence"
            incorporée pour cela. Pour économiser de l'énergie
            automatiquement, vous pouvez souvent utiliser une option
            de configuration du BIOS qui indique au disque au démarrage
            qu'il peut s'arrêter (il possède un minuteur incorporé !)
            dès qu'il n'aura pas été utilisé pendant un certains temps.
  ACPIOFF   Comme POWEROFF mais utilise explicitement l'ACPI au lieu
            de l'APM. Notez que les fonctions classiques utiliseront
            l'ACPI aussi, si aucun support BIOS de l'APM n'est présent.
            Le système est placé en S5 soft-off après avoir vidé les
            caches et arrêté les disques.
            Utile pour comparer la gestion APM / ACPI.
  STANDBY   Vide les caches disque, entre en mode veille. Nécessite
            l'APM ou l'ACPI et une alimentation ATX ou plus récente.
  SUSPEND   Vide les cache disque, arrête (les réveillera
            automatiquement) les disques IDE, et met en veille le
            système. Utile si vous avez un BIOS avec hibernation des
            disques, pour éviter de les activer juste avant que
            l'hibernation n'en ait besoin. Nécessite l'APM ou
            l'ACPI et une alimentation ATX ou plus récente.
  PURESUSP  Comme SUSPEND, mais n'arrête pas d'abord les disques IDE.
            Nécessite l'APM ou l'ACPI et une alimentation ATX ou plus
            récente.
  POWEROFF  Vide les caches disque, arrête les disques IDE, éteint le
            VGA et le système. Nécessite l'APM ou l'ACPI et une
            alimentation ATX ou plus récente.
  PUREOFF   Comme POWEROFF, mais n'arrête pas d'abord les disques IDE.
            Utile si votre BIOS réveillerait autrement les disques juste
            avant de couper l'alimentation (aïe). Nécessite l'APM ou
            l'ACPI et uen alimentation ATX ou plus récente.
  VGAoff    Éteint l'écran EGA/VGA DPMS (pas de réveil automatique !).
            Vous devez taper FDAPM VGAon ou redémarrer pour l'allumer.
            VGAoff n'est pas un économiseur d'écran. Utilisez idledpms
            en tant qu'économiseur d'écran.
  VGAon     Allume à nouveau l'écran EGA/VGA DPMS. Appuyer simplement
            sur une touche après avoir utilisé la fonction VGAoff
            N'activera PAS la vidéo à nouveau, ce n'est pas un
            économiseur d'écran. C'est seulement une façon d'allumer
            et d'éteindre la vidéo sans aucune surveillance du système
            ou de réveil automatique.
  ZapStats  Efface les compteurs STATS.
  SPEEDn    Règle la vitesse du système sur n/8 (n * 12,5%) du maximum
            (0 arrête jusqu'à un appui sur le bouton d'alimentation).
            Utilisez SPEED9 pour afficher le réglage actuel. SPEED9 est
            la méthode recommandée de vérifier si votre système utilise
            l'ACPI d'une manière compatible avec FDAPM. Nécessite l'ACPI.
  SPEED0    Speed 0 mettra le système en veille S1 (appuyez sur le bouton
            d'alimentation ATX ou le bouton veille pour continuer) du
            point de vue du processeur. Aucun autre périphérique ne se
            mettra en veille. Utilisez l'outil PCISLEEP si vous souhaitez
            faire des expériences avec la gestion de l'alimentation PCI.
  COLDboot  Effectue un redémarrage avec les tests BIOS (POST) (après
            avoir vidé les caches disque) en indiquant au contrôleur de
            clavier 8042 d'activer les circuits de remise à zéro du
            matériel. Très similaire à un redémarrage à froid - seuls
            dans de rares cas, appuyer sur le bouton reset ou de
            redémarrer l'ordinateur vous donnera une remise à zéro
            plus "froide".
  WARMboot  Effectue un redémarrage sans les tests BIOS (POST).
            Généralement sûr mais peut planter si vous utilisez des
            logiciels de mode protégé incompatibles comme certaines
            versions d'EMM386. Vide les caches disque avant de
            redémarrer.
  HOTboot   Tente un redémarrage en utilisant int 19h (plantera
             souvent !).
            Avant de tenter le redémarrage, FDAPM indique aux caches
            de se vider (écrire les données sur le disque) et attend
            un moment.
  HELP      Affiche l'aide.
  /?        Affiche l'aide.

Commentaires :

  FDAPM est un alternatif pour MS-DOS Power.
  APM: Advanced Power Management (gestion avancée de l'alimentation).
  ACPI: Advanced Configuration and Power Interface (interface avancée de
        configuration et gestion de l'alimentation).
  ATX: Advanced Technology Extended (technologie avancée étendue).
  DPMS: Display Power Management Signaling (signalisation de gestion de
        l'alimentation de l'affichage).
  Veuillez lire fdapm.txt dans le paquet fdapm pour plus d'informations.
  La casse est sans objet, "-" ou "/" ne sont pas nécessaires.
  Toutes les options d'hibernation/veille/extinction des disques durs
  ajoutent automatiquement un délai pour donner au disque dur le temps
  de changer d'état.
  Toutes ces opérations et toutes les options relative au redémarrage
  indiquent aux caches d'écrire les tampons sales sur le disque avant
  que l'action concrète ne commence.

Exemples :

  fdapm apmdos  vous permet de réaliser des économies d'énergie
                maximales
  fdapm speed2  règle la vitesse sur 2x12,5% = 25 % de la vitesse
                maximale, par exemple pour d'anciens jeux

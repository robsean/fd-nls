Commande : scandisk

  SCANDISK est un programme de MS pour tester les disquettes et les
  disques durs. SCANDISK n'est pas utilisé sous FreeDOS. Veuillez
  utiliser CHKDSK ou DOSFSCK à la place.

Syntaxe :

  - aucune -

Options :

  - aucune -

Commentaires :

  - aucun -

Exemples :

  - aucun -

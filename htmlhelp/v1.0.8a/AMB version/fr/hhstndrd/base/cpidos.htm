CPIDOS -fichiers CPI (Information de page de code) Pack v3.0 - Pages de code DOS

  Ce paquet contient des fichiers d'information de page de code
  compressés avec UPX avec les pages de code DOS, qui sont de type
  DISPLAY. L'outil officiel de FreeDOS pour les préparer et les choisir
  est MODE.
  Les dispositions de clavier disponibles sur KPDOS sont destinés
  à utiliser les fichiers CPI contenus dans ce paquet.
  Voici ci-dessous la liste de ces fichiers et leurs pages de codes :

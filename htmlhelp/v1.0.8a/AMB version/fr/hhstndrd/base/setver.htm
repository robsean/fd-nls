Commande : setver

  FreeDOS utilise CALLVER et VERSION=x.yy. au lieu de SETVER.
  Notez que ces commandes ne reproduisent pas exactement la
  fonctionnalité de SETVER.

Syntaxe :

  - aucune -

Options :

  - aucune -

Commentaires :

  - aucun -

Exemples :

  - aucun -

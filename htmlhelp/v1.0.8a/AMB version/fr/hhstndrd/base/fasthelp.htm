Commande : fasthelp (fdhelp)

  FASTHELP est un clone de fasthelp pour FreeDOS. Affiche l'aide sur un
  sujet, de façon similaire aux pages man de GNU/Linux.

Syntaxe :

  fasthelp [sujet]

Options :

  sujet  Le thème sur lequel vous souhaitez des informations.
    Environnement :
      PAGER :    Un programme de pager pour afficher un fichier texte.
      HELPPATH : Le répertoire qui contient vos fichiers d'aide.
      LANG :     La langue préférée pour vos fichiers d'aide.

Commentaires :

  Utilisez les variables d'environnement HELPPATH et LANG pour spécifier
  respectivement la localisation et la langue des fichiers d'aide. Le
  chemin par défaut est ../help et la langue par défaut est EN (anglais).
  Il existe également la variable d'environnement PAGER qui peut être
  utilisée pour définir un programme alternatif pour l'affichage des
  fichiers d'aide textuels. Les appels à fasthelp renverront les
  fichiers texte au programme PAGER.
  Voyez SET pour plus d'informations sur
  la définition des variables d'environnement.
  Fasthelp est aussi appelé "FSUITE" ou "FSUITE04" ou bien "FS04de"
  (allemand) ou encore "FSUITE04de" (allemand).

Exemples :

  fasthelp       Affiche la liste des thèmes
  fasthelp edit  Affiche des informations au sujet d'edit

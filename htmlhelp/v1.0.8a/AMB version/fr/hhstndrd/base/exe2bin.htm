Commande : exe2bin

  EXE2BIN convertit les programmes EXE au format binaire. C'est un outil
  pour les développeurs de logiciels.

Syntaxe :

  EXE2BIN [options] fichier_exe[.exe] [fichier_bin]
          fichier_exe  le fichier à convertir
          fichier_bin  le nouveau fichier bin

Options :

  /Q        Supprime les messages d'information.
  /H        Affiche l'en-tête exe.
  /R        Affiche les déplacements.
  /L=(seg)  Déplace le fichier_exe vers le segment (seg).
  /X        Comportement étendu, par exemple fichiers > 64Ko.
  /?        Affiche l'aide.

Commentaires :

  - aucun -

Exemples :

  exe2bin c:\fdos\bin\ctmouse.exe c:\ctmouse

Commande : sort

  SORT trie un fichier ou l'entrée à afficher.
  Le fichier ou l'entrée n'est pas modifiée.

Syntaxe :

  sort [/R] [/N] [/+num] [/A] [/?] [lecteur][chemin]nomfichier
  commande | sort [/R] [/N] [/A] [/+num] [/?]
       lecteur     La lettre du lecteur, par exemple C:
       chemin      Le répertoire, par exemple \exemple\
       nomfichier  Le pilote (fichier) à charger, par exemple
                   nonordonné.txt

Options :

  /R     Trie dans l'ordre inverse (Z - A).
  /A     Trie par l'ordre ASCII, ignore COUNTRY.
  /N     Active le NLS / l'ordre de tri basé sur LA PAGE DE CODE /
         COUNTRY (pays).
  /+num  Débute le tri au numéro de colonne donné sur chaque ligne de
         texte. Valeur par défaut=1.
  /?     Affiche l'aide.

Commentaires:

  SORT ne modifie pas le fichier d'entrée, seulement la sortie.
  SORT prend en charge le NLS (support des langues nationales).

Exemples :

  Affiche le répertoire courant dans l'ordre alphabétique :
    dir | sort
  Trie le fichier et l'écrit sur un autre fichier :
    sort nonordonné.txt > ordonné.txt
    sort           -ENTRÉE-   (alternative : sort /+2)
    this           -ENTRÉE-
    is             -ENTRÉE-
    a              -ENTRÉE-
    short          -ENTRÉE-
    sentence       -ENTRÉE-
  (CTRL-Z)  (sort de la commande et l'exécute)
    donne :                trié par la seconde lettre de chaque ligne :
    a                         a
    is                        sentence
    sentence                  this
    short                     short
    this                      is

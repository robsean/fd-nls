Commande : tree / pdtree

  TREE / PDTREE affiche de manière graphique l'arborescence des
  répertoires [dossiers] d'un lecteur ou chemin donné. Il inclut
  éventuellement une liste de fichiers dans l'affichage.

Syntaxe :

  TREE [d:][chemin] [/F] [/A] [/?]
  PDTREE [d:][chemin] [/F] [/A] [/V] [/S][/P] [/?]
       d:chemin  Le lecteur et le chemin desquels commencer. Si aucun
       chemin n'est choisi, tree/pdtree commencera du répertoire courant.

Options :

   /F  Affiche le nom des fichiers dans chaque répertoire.
   /A  Utilise des caractères ASCII pour représenter visuellement la
       structure des répertoires.
   /?  Affiche l'aide et quitte.

 Options supplémentaires de pdTree de FreeDOS:
 (non disponibles dans Tree standard.)
   /V  Affiche des informations de version et quitte.
   /S  Noms courts seulement (utilise les noms de fichiers courts
       c'est-à-dire alternatifs lorsqu'ils existent).
       Version DOS seulement, désactive l'utilisation de l'API LFN pour
       trouver le premier/suivant.
   /P  Fait une pause après chaque page. (Remarque : utilise
       actuellement la valeur par défaut d'un écran 80x25)
       Version Win32 seulement, ignore l'option de pause si la sortie
       est redirigée vers un fichier ou transmis à un autre programme,
       par exemple tree | more ; tente également d'utiliser la taille
       actuelle de la console (TODO (à faire) pour la version DOS).

Comments:

  TREE prend en charge le NLS (support des langues nationales).

Exemples :

  Pour afficher la structure hiérarchique complète des répertoires du
  lecteur C effectuer une pause après chaque écran, lancez tree en
  spécifiant C: en tant que chemin de départ et transmettez sa sortie
  à la commande MORE :
  tree C:\ | more
  Lors de l'utilisation de pdTree, l'exemple ci-dessus peut être obtenu
  en utilisant :
  pdtree C:\ /p
  Pour afficher la hiérarchie d'un sous-répertoire et obtenir la sortie
  uniquement avec des caractères standards ASCII (par exemple pour
  imprimer ou l'envoyer à quelqu'un qui pourrait le visionner sans la
  prise en charge des caractères de DOS pour dessiner des lignes et des
  boîtes, spécifiez le sous-répertoire avec le commutateur /A :
    tree ASUBDIR /a
  Pour afficher la structure d'un répertoire et les fichiers dans chaque
  répertoire, ajoutez le commutateur /F :
    tree ASUBDIR /f | more

Mise en réseau de FreeDOS - Introduction

  Remerciements à M. Ulrich Hansen pour son autorisation à publier cet
  article dans l'aide de FreeDOS !

Résumé

  Ce document décrit comment mettre en réseau un ordinateur sous FreeDOS
  (http://www.freedos.org/). La plupart des informations sont censées
  marcher avec d'autres versions de DOS également.
  Le premier chapitre couvre l'historique réseau de DOS, qui s'est
  essentiellement produit entre 1981 et 1995. Dans le second chapitre
  vous pouvez trouver une vue d'ensemble des pilotes, protocoles et
  applications qui sont toujours là au XXIe siècle. Le troisième et
  quatrième chapitres expliquent comment installer TCP/IP sur une
  machine sous DOS. Il s'agit d'un guide détaillé sur l'installation et
  la configuration des pilotes et d'autres logiciels nécessaires.

Les alternatives

  S'il existe un port LPT disponible sur les deux machines vous pouvez
  aussi utiliser une connexion avec un câble parallèle pour transférer
  des fichiers (par exemple avec MS DOS 6.22 Interlink, Open/DR DOS
  Filelink, Laplink, Winlink ou FileMaven cf. http://www.briggsoft.com/
  fmdos.htm (*01)).

Le matériel

    (Image Suite de tests pour ce wiki : un Compaq Contura Aero 4/33C
    avec une carte PCMCIA)
  Pour vous connecter à un réseau vous aurez besoin d'un appareil pour
  permettre la connexion entre votre PC et le réseau. Cela peut
  se faire :
    * via un Modem, en utilisant la ligne téléphonique ou
    * via une carte réseau Ethernet, en utilisant un câblage direct ou
    * via le Wi-Fi.

  Modem
  La mise en réseau avec un Modem n'est pas encore décrit sur ce Wiki -
  peut-être voudrez-vous contribuer ?

  Ethernet
  Si vous pensez acheter une carte Ethernet pour votre machine sous
  FreeDOS, voyez s'il est fourni avec un pilote de paquets. De nombreuses
  applications TCP/IP pour DOS nécessitent un pilote de paquets pour
  marcher (voyez ici pour plus d'informations. Si la carte est fournie
  avec d'autres pilotes pour DOS ou Windows 3.1 comme des pilotes ODI
  (Novell) ou des pilotes NDIS2 (Microsot) elles fonctionneront également
  sous DOS (elles sont juste un peu plus compliquées à installer).
  Si vous utilisez PCMCIA, les cartes suivantes devraient fonctionner :
    * La carte 3Com Megahertz 3C574 10/100 LAN PC (pas de pilote de
      paquets mais avec l'ODI et le NDIS2),
    * la carte 3Com EtherLink III 3C589 10Mbit (avec un pilote de
      paquets) et
    * la carte D-Link DFE-670TXD Fast Ethernet PC (avec un pilote de
      paquets).
  Les cartes PCMCIA doivent être installées avec un logiciel tierce
  partie comme SystemSofts CardSoft. Il peut être téléchargé depuis
  driverguide en tant que "System Soft Version 3.1, CARDSOFT.ZIP"
  (voyez : http://www2.driverguide.com/uploads/uploads9/26188.html (*02))

  Wi-Fi
  Certaines anciennes cartes Wi-Fi fonctionnent avec FreeDOS. Voyez
  ici pour davantage d'informations.

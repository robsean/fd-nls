Mise en réseau de FreeDOS - les autres

  Bien que les catégories mentionnées sur la mise en réseau des systèmes
  DOS devraient couvrir la plupart des programmes, il existe quelques
  exceptions.
  L'une d'entre elles est "Invisible LAN", c'est-à-dire le réseau local
  invisible (cf.  http://www.invisiblesoft.com/invlan/index.shtml), une
  application DOS similaire à NetWare qui est même fourni avec son propre
  protocole appelé "TransBIOS".
  Une autre solution spéciale pour obtenir rapidement un réseau
  opérationnel est constitué par les disques d'amorçage réseau. Ils sont
  en mesure de fonctionner avec un large éventail de matériel et
  d'utiliser de multiples protocoles. Les disques d'amorçage sont surtout
  utilisés pour la sauvegarde, la restauration et la réparation. Ils
  peuvent être très utiles.
  Les exemples sont :
    * Bart's Network Boot Disk (voyez : http://www.nu2.nu/bootdisk/
      network/ (*01))
    * Drivesnapshot (voyez : http://drivesnapshot.de/en/index.htm (*02))
    * Netbootdisk (voyez : http://www.netbootdisk.com/ (*03))
  La situation légale est malheureuse : certains paquets semblent inclure
  des fichiers depuis le client MS alors que la licence de Microsoft
  n'autorise pas une telle distribution.

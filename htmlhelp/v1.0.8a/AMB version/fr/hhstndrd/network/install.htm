Mise en réseau de FreeDOS - Installation

  La majorité des applications TCP/IP pour DOS utilise l'interface de
  pilote de paquets. Donc pour pouvoir les utiliser, nous devons d'abord
  installer les pilotes appropriés.
  L'installation d'un pilote de paquets générique est la manière la
  plus simple. Mais ce n'est pas toujours possible, car les cartes
  réseau modernes sont souvent fournies qu'avec les pilotes ODI et NDIS.
  Quoi qu'il en soit, les pilotes ODI ou NDIS peuvent également être
  utilisés comme des pilotes de paquets via des convertisseurs spéciaux
  (des "shims"). C'est juste un peu plus compliqué.
  Les chapitres suivants décrivent où obtenir les pilotes et les
  convertisseurs et comment les configurer :
    * [Installation d'un pilote de paquets]
    * [Installation d'un pilote ODI]
    * [Installation d'un pilote NDIS]

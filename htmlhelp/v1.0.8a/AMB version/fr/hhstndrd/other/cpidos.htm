CPIDOS - pack de fichiers CPI (Code Page Information) v3.0 - pages de code DOS

  Ce paquet contient des fichiers d'information de page de code
  compressés avec UPX avec les pages de code DOS, qui sont de type
  DISPLAY. L'outil officiel FreeDOS pur les préparer et les
  sélectionner est MODE. Les dispositions de clavier disponibles chez
  KPDOS sont destinées à utiliser les fichiers CPI contenus dans ce
  paquet. Vous trouverez ci-dessous liste de ces fichiers et les pages
  de codes qu'ils contiennent :

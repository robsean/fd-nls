Autres façons d'obtenir de l'aide

  1. La plupart des commandes décriront leurs fonctionnalités et options
     en utilisant /? sur elles sur la ligne de commande. Par exemple :
     dir /?
  2. La commande FASTHELP (FDHELP ou FSUITE04). C'est un clone de fasthelp
     de MS-DOS. Si vous ne trouvez pas ce que vous recherchez dans
     l'aide, il peut être présent dans fasthelp, ou vice versa.
  3. Les commandes WHATIS et APROPOS.
  4. La commande HELP (HTMLHELP).
  5. Le sous-répertoire doc dans le répertoire DOS contient de nombreux
     manuels détaillés pour divers programmes FreeDOS.
  6. Visitez le site Internet de FreeDOS, http://www.freedos.org

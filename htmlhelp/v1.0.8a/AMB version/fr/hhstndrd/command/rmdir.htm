Commande : rmdir

  RMDIR enlève (efface) un répertoire vide.

Syntaxe :

  RMDIR [lecteur][chemin]nomchemin
  RMDIR [/?]
     lecteur   La lettre du lecteur sur lequel vous souhaitez effacer un
               répertoire, par exemple C:
     chemin    Le nom du chemin qui existe déjà, par exemple si vous êtes
               déjà dans un répertoire.
     nomchemin Le nom du répertoire que vous voulez effacer, par exemple
               \exemple.

Options :

  /?  Affiche l'aide.

Commentaires :

  Pour des raisons de performance, RMDIR écrase seulement la première
  lettre de chaque nom de dossier / fichier par un '?'. Voyez DEL ou
  ERASE ou DELTREE pour plus d'informations. Vous pouvez aussi utiliser
  RD au lieu de RMDIR.
  RMDIR n'effacera que les répertoires vides (pas de fichiers ou
  répertoires à l'intérieur) !
  N'oubliez pas d'enlever les fichiers ou répertoires cachés !
  Sous DOS, les répertoires sont affichés en majuscules et n'ont PAS
  d'extension.
  RMDIR est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  Vous avez un répertoire VIDE C:\test\test2
  rmdir C:\test\test2

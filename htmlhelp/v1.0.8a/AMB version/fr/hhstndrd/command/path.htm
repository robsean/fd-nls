Commande : path

  PATH affiche ou définit un chemin de recherche pour les fichiers
  exécutables. PATH est une commande de FICHIER BATCH / AUTOEXEC.BAT.
  Elle peut également être utilisée sur la ligne de commande.

Syntaxe :

  PATH [[lecteur]chemin[;...]]
  PATH [listedeschemins]
  PATH ;
  PATH [/?]

  lecteur     La lettre du lecteur, par exemple C:
  chemin      Le répertoire, par exemple \exemple\

Options :

  listedeschemins  Définit la liste des chemins. Séparez les chemins
            avec des points-virgules (;).
  ;         Efface tous les réglages de chemin de recherche et indique
            le shell de commande de ne rechercher que dans le répertoire
            courant.
  /?        Affiche l'aide.

Commentaires :

  Tapez "PATH ;" pour effacer tous les réglages de chemin de recherche
  et indiquer au shell de commande de ne rechercher que dans le dossier
  courant.
  Tapez "PATH" sans paramètres pour afficher le chemin actuel.
  Certaines commandes sont internes à FreeDOS. Cependant, beaucoup sont
  dans le dossier \bin ou \freedos\bin vous pouvez donc souhaiter avoir
  ce dossier dans la liste des chemins (comme dans le premier exemple
  ci-dessous). Les chemins des programmes fréquemment utilisés peuvent
  être mis sur la liste des chemins pour des raisons de convenance.
  Lorsqu'un programme ou une commande externe est entré sur l'invite de
  commande FreeDOS le recherche dans le répertoire courant puis dans
  chacun des chemins spécifiés par la commande path.
  Si le même nom de programme se trouve dans différents répertoires de
  path, le programme sera lancé depuis le chemin qui sera mentionné le
  premier sur la liste des chemins de path.
  PATH peut également être utilisée dans un fichier batch.
  PATH est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  DANS UN FICHIER .BAT / AUTOEXEC.BAT :
    path c:\fdos\bin;c:\util
    path %path%;c:\jeux
    Le second exemple ajoute c:\jeux à la liste existante, tout en
   préservant les chemins déjà définis.

Commande : dir

  DIR affiche une liste des fichiers des sous-dossiers dans un
  répertoire.

Syntaxe :

  DIR [lecteur][chemin][nomfichier] [/P] [/W] [/A[[:]attributs]]
      [/O[[:]ordredetri]] [/S] [/B] [/L] [/Y] [/?]
      lecteur     La lettre du lecteur, par exemple C:
      chemin      Le répertoire, par exemple \exemple\
      nomfichier  Le fichier à afficher, par exemple test.txt
                  [lecteur][chemin][nomfichier]
                  Spécifier le lecteur, le dossier et/ou les fichiers
                  à lister. (Peut être une spécification de fichier
                  améliorée ou plusieurs spécifications de fichier.)

Options :

  /P  Affiche une page de la liste à la fois
  /W  Affiche la liste au format large
  /A:attributs
      Affiche seulement les fichiers et dossiers avec les attributs
      spécifiés. La valeur par défaut est /a:hsdar. Vous pouvez utiliser
      les éléments suivants pour les attributs :
        h | -h  Fichiers caches         h fichiers qui ne sont pas cachés
        s | -s  Fichiers système       -s fichiers qui ne sont pas des
                                          fichiers système
        d | -d  Répertoires            -d fichiers
        a | -a  Fichiers avec le bit   -a fichiers sans le bit archive
                archive
        r | -r  Fichiers en lecture    -r fichiers qui ne sont pas en
                seule                     lecture seule
  /O:ordredetri
      Affiche la liste dans l'ordre spécifié. La valeur par défaut est
     /O:g.
      Vous pouvez utiliser les éléments suivant pour spécifier l'ordre
      de tri :
        a | -a  Alphabétiquement (A-Z) -a Alphabétiquement (Z-A)
        e | -e  Par l'extension A-Z    -e Extension (Z-A)
        d | -d  Par l'horodatage       -d Horodatage dans l'ordre inverse
        s | -s  Par la taille          -s Taille (le + > - le + <)
                (le + < - le + >)
        g | -g  Groupe les dossiers    -g Groupe les dossiers à la fin.
                d'abord
  /S  Affiche les fichiers dans le dossier spécifié et tous les
      sous-dossiers.
  /B  Utilise le format nu (aucune information de titre ou résumé).
      Cela liste un nom par ligne
  /L  Affiche les noms en minuscules
  /Y ou /4
      Affiche l'année en 4 chiffres.
  /?  Affiche l'aide.

Commentaires :

  Afin de personnaliser l'apparence par défaut de dir, utilisez la
  variable d'environnement DIRCMD. Par exemple :
    SET DIRCMD=/w/p
  Pour passer outre un élément défini dans DIRCMD, utilisez un trait
  d'union ("-") sur ce commutateur sur la ligne de commande. (par
  exemple /-w)
  DIR est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  dir c:\exemple /p
  dir c:\exemple /w
  dir c:\exemple /O: a
  dir c:\fdos | more

Commande : ctty

  CTTY change le terminal utilisé pour contrôler votre système.

Syntaxe :

  CTTY dispositif
  CTTY [/?]

Options :

  dispositif  Le terminal que vous souhaitez utiliser.
              Les dispositifs valides sont prn, lpt1, lpt2, lpt3, con,
              aux, com1, com2, com3, com4, NUL.
  /?          Affiche l'aide.

Commentaires :

  CTTY est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  CTTY lpt1
  CTTY /?

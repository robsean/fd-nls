Commande : prompt

  PROMPT modifie la manière dont l'invite de commande apparaît.
  PROMPT est une commande de FICHIER BATCH / AUTOEXEC.BAT.
  Elle peut également être utilisée sur la ligne de commande.

Syntaxe :

  PROMPT [Options] [texte]
         texte     Spécifie une nouvelle invite de commande.

Options :

  L'invite peut être composée de caractères normaux et des codes
  spéciaux suivants:

  $$   $ (signe du dollar)
  $_   Retour chariot et saut de ligne
  $B   | (pipe)
  $D   Date actuelle
  $E   Code d'échappement (Code ASCII 27)
  $G   > (signe supérieur à)
  $H   Retour arrière (efface le caractère précédent)
  $L   < (signe inférieur à)
  $N   Lecteur actuel
  $P   Lecteur et chemin actuels
  $Q   = (signe d'égalité)
  $T   Heure actuelle
  $V   Numéro de version du shell de commande de FreeDOS

Commentaires :

  Tapez PROMPT sans paramètre pour remettre l'invite à zéro vers sa
  valeur par défaut. PROMPT peut également être utilisée en dehors
  d'un fichier batch.
  PROMPT est une commande interne à command.com et en nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  DANS UN FICHIER .BAT / AUTOEXEC.BAT :
    prompt $p$g            Cela est identique au réglage par défaut
    prompt $p$g Veuillez entrer une commande !

Commande : loadfix

  LOADFIX charge un programme au-delà des premiers 64 Ko de mémoire et
  l'exécute. Il charge également un programme "exepacked".

Syntaxe :

  LOADFIX [lecteur][chemin]nomfichier [options]
          lecteur     La lettre du lecteur, par exemple C:
          chemin      Le répertoire, par exemple \exemple\
          nomfichier  Le fichier que vous souhaitez charger, par exemple
                      edit.exe

Options :

  options  Ce sont les options du programme "exepacked" que vous
           allez charger (=nomfichier). Tapez "nomfichier /?" en
           premier pour trouver ces options.

Commentaires :

  Utilisez LOADFIX pour charger un programme si vous avez reçu le
  message "Fichier empaqueté corrompu" quand vous avez essayé de le
  charger en mémoire basse.
  LOADFIX est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  EDIT /?                              affiche les options pour edit
  LOADFIX C:\exemple\edit.exe options

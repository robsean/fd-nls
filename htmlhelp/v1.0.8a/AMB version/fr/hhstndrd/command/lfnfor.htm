Commande : lfnfor

  LFNFOR autorise ou interdit la complétion des noms de fichier longs
  sur la ligne de commande lorsque la complétion "TAB" ou "TAB" "TAB"
  est utilisée.
  Elle autorise également les noms de fichier longs à être utilisés
  avec la commande FOR. Voyez aussi : Options dans DOSKEY.

Syntaxe :

  LFNFOR [ON | OFF] [/?]
  LFNFOR COMPLETE [ON | OFF]

Options :

  ON
  OFF
  /?   Affiche l'aide.

Commentaires :

  Tapez LFNFOR ou LFNFOR COMPLETE sans paramètre pour afficher le réglage
  actuel de LFNFOR.
  LFNFOR est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  lfnfor on            active les noms de fichier longs dans la
                       commande for.
  lfnfor off           désactive les noms de fichier longs dans
                       la commande for.
  lfnfor               affiche le réglage actuel de lfnfor.
  lfnfor complete on   active les noms de fichier longs dans
                       "TAB" "TAB".
  lfnfor complete off  désactive les noms de fichier longs dans
                       "TAB" "TAB".
  lfnfor complete      affiche le réglage actuel de lfnfor.

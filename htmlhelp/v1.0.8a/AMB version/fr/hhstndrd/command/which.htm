Commande : which

  WHICH recherche et affiche le fichier exécutable pour chaque commande
  spécifiée.

Syntaxe :

  which commande [/?]

Options:

  commande  La commande à trouver.
  /?        Affiche l'aide.

Commentaires :

  WHICH est un utilitaire né dans le monde UNIX et qui permet aux
  utilisateurs de retrouver des fichiers exécutables dans les
  répertoires listés dans la variable d'environnement PATH.
  WHICH est une commande interne à command.com et ne nécessite
  aucun autre fichier pour fonctionner.

Exemples :

  which xcopy  recherche l'exécutable pour la commande xcopy.
               La sortie est : "xcopy C:\FDOS\BIN\xcopy.exe"
  which copy   La sortie est : "copy" (copy est une commande
               command.com)

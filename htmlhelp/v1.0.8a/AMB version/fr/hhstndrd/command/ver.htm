Commande : ver

  VER affiche des informations de version FreeDOS. Par défaut,
  seule la version du shell de commande (FreeCOM) est indiquée.

Syntaxe :

  VER [/R] [/W] [/D] [/C]

Options :

  /R  Affiche les numéros de version du kernel et de DOS.
  /W  Affiche la garantie du shell de commande de FreeDOS.
  /D  Affiche les informations de distribution du shell de commandes
      de FreeDOS.
  /C  La section des remerciements de FreeCOM. Répertorie les
      contributeurs.

Commentaires :

  VER est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  ver /R
  ver /W
  ver /D
  ver /C

Voir également :

  callver
 (setver)

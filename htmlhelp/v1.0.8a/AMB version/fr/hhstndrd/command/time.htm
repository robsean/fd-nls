Commande : time

  TIME affiche ou définit l'heure du système.

Syntaxe :

  TIME [/T]
  TIME [nouvelleheure]

Options:

  nouvelleheure  hh:mm
                 hh:mm:ss
                 hh:mm:ss.ss
                 L'heure à définir pour votre système.
                 'hh' est l'heure sur une horloge de 12 ou 24 heures.
                 'mm' représente les minutes.
                 'ss.ss' représente les secondes et les centièmes de
                 secondes.
  /T       affiche l'heure sans une invite pour un nouveau réglage.
  /?       Affiche l'aide.

Commentaires :

  Tapez TIME sans paramètre pour afficher le réglage de l'heure actuelle
  et une invite pour la modifier. Appuyez sur ENTRÉE pour conserver
  l'heure.
  TIME est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  time /T
  time 15:10
  time 15:10:20
  time 15:10:20.75

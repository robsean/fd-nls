Commande : fdhelp (fasthelp)

  FDHELP est un clone de fasthelp pour FreeDOS. Affiche l'aide sur
  un sujet, de la même manière que les pages man de GNU/Linux.

Syntaxe :

  fasthelp [sujet]

Options :

  sujet  Le thème sur lequel vous souhaitez des informations.
  Environnement:
    PAGER:    Programme pager pour afficher un fichier textuel.
    HELPPATH: Répertoire qui contient vos fichiers d'aide.
    LANG:     Langue préférée pour vos fichiers d'aide.

Commentaires :

  Utilisez les variables d'environnement HELPPATH et LANG pour spécifier
  respectivement l'emplacement et la langue des fichiers d'aide. Le
  chemin par défaut est ../help et la langue par défaut est EN (anglais).
  Il existe également la variable d'environnement PAGER, qui peut être
  utilisée pour définir un programme alternatif pour l'affichage des
  fichiers d'aide textuels. Les appels à fasthelp détourneront les
  fichiers textuels vers le programme PAGER.
  Voyez SET pour davantage d'informations sur le réglage des variables
  d'environnement.
  Fasthelp est aussi appelé "FSUITE" ou "FSUITE04" ou "FS04de" (allemand)
  ou encore "FSUITE04de" (allemand).

Exemples :

  fasthelp       Vous montre la liste des thèmes
  fasthelp edit  Vous montre des informations au sujet d'edit

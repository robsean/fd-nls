Commande : help

  HELP est cette collection de fichiers d'aide de FreeDOS au format
  html. Cliquez ici si vous voulez en savoir plus sur l'utilisation
  de l'aide.

Syntaxe :

  HELP [options] [sujet]

Options :

  Options de base :
  sujet     Affiche l'aide sur ce sujet.
  /M        Force l'affichage monochrome.
  /F1       Utilise le thème de couleurs 1.
  /F2       Utilise le thème de couleurs 2.
  /A        Utilise l'ASCII au lieu des caractères étendus.
  /Cnnn     Indique à HELP que la page de code est nnn, plutôt que
            de le laisser détecter.
  /?        Affiche l'aide.
  Options avancées :
  /O[chemin[fichier]] Remplace le chemin de l'aide.
            Si aucun fichier n'est spécifié, index.htm est utilisé.
  /Lfichier Charge un fichier autre que index.htm par rapport au chemin
            d'accès à l'aide.
  /Hsujet   Lorsque l'utilisateur appuie sur F1 ou clique sur "Aide
            sur l'aide", help chargera ce fichier. La valeur par défaut
            est help.htm.
  Variables d'environnement :
  HELPPATH  Répertoire contenant vos fichiers d'aide.
  HELPCMD   Mettez /M, /A, F1, F2 ici pour les établir en tant que
            valeur par défaut.

Commentaires :

  HELP est aussi appelé HTMLHELP ou HHSTNDRD.
  Version actuelle : HHSTNDRD 1.0.8a fr.
  Sous FreeDOS (pas sur Internet) HELP se compose d'un visionneur html
  DOS (help.exe) et des fichiers d'aide html.
  HELP prend en charge NLS (support des langues nationales).

Commande : echo

  ECHO affiche des messages, ou active ou désactive l'écho des commandes.
  Il existe aussi une commande ECHO CONFIG.SYS / FDCONFIG.SYS.
  ECHO est une commande de FICHIER BATCH / AUTOEXEC.BAT.
  Elle peut être également utilisée sur la ligne de commande.

Syntaxe :

  ECHO [ON | OFF]
  ECHO [message]

Options :

  ON       Le mode ECHO est activé. Si vous utilisez ECHO dans un
           fichier batch, chaque ligne de commande sera affichée
           lorsqu'elle sera exécutée.
  OFF      Le mode ECHO est désactivé. Si vous utilisez ECHO dans
           un fichier batch, la ligne de commande ne sera pas affichée
           lorsqu'elle sera exécutée. Cela n'affecte pas la sortie de
           la commande elle-même.
  message  Le message que vous voulez afficher sur l'écran.

Commentaires:

  Les commandes peuvent aussi être réprimées en les commençant avec le
  symbole @ Tapez ECHO sans paramètres pour afficher le réglage actuel
  d'ECHO.
  ECHO peut aussi être utilisée en dehors d'un fichier batch.
  ECHO est une commande interne à command.com et n'a besoin d'aucun
  autre fichier pour fonctionner. Il existe aussi une commande ECHO
  pour CONFIG.SYS / FDCONFIG.SYS avec des options différentes.

Exemples :

  DANS UN FICHIER .BAT / AUTOEXEC.BAT:
    ECHO Salut,
    ECHO. comment allez-vous ?
    @ECHO Bien ?
    @ECHO. Il est bon d'entendre cela

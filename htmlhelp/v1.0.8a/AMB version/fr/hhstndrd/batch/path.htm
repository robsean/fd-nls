Commande : path

  PATH affiche ou définit un chemin de recherche pour les fichiers
  exécutables.
  PATH est une commande de FICHIER BATCH / AUTOEXEC.BAT.
  Elle peut également être utilisée sur la ligne de commande.

Syntaxe :

  PATH [[lecteur]chemin[;...]]
  PATH [listedechemins]
  PATH ;
  PATH [/?]

  lecteur   La lettre du lecteur, par exemple C:
  chemin    Le répertoire, par exemple \exemple\

Options :

  listedechemins  Définit une liste de chemins. Séparez les chemins
            avec des points-virgules (;).
  ;         Efface tous les réglages de chemins de recherche et dit au
            shell de commande de rechercher uniquement dans le répertoire
            courant.
  /?        Affiche l'aide.

Commentaires :

  Tapez "PATH ;" pour effacer tous les réglages de chemins de recherche
  et dire au shell de commande de rechercher uniquement dans le dossier
  courant. Tapez "PATH" sans paramètres pour afficher le chemin actuel.
  Certaines commandes sont internes à FreeDOS. Cependant, beaucoup sont
  dans les répertoires \bin ou \freedos\bin donc vous voudrez
  probablement avoir ce répertoire dans la liste des chemins (comme
  dans le premier exemple ci-dessous).
  Les chemins des programmes fréquemment utilisés peuvent être ajoutés
  sur la liste des chemins pour plus de commodité.
  Lorsqu'un programme ou un programme externe est entré sur l'invite
  de commande, FreeDOS le recherche dans le répertoire courant puis
  dans chacun des chemins spécifiés par la commande path.
  Si le même nom de programme est dans différents chemins, le programme
  sera démarré depuis le chemin qui est mentionné en premier sur la
  liste des chemins.
  PATH peut également être utilisée en dehors d'un fichier batch.
  PATH est une commande interne à command.com et ne nécessite aucun
  autre fichier pour fonctionner.

Exemples :

  DANS UN FICHIER .BAT / AUTOEXEC.BAT:
    path c:\fdos\bin;c:\util
    path %path%;c:\jeux
    Le second exemple ajoute c:\jeux à la liste existante, en conservant
    les chemins définis auparavant.

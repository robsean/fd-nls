Commande : prompt

  PROMPT modifie la manière dont l'invite de commandes apparaît.
  PROMPT est une commande de FICHIER BATCH / AUTOEXEC.BAT.
  Elle peut également être utilisée sur la ligne de commande.

Syntaxe :

  PROMPT [Options] [texte]
  texte   Spécifie une nouvelle invite de commande.

Options :

  L'invite peut être composée de caractères normaux et des codes
  spéciaux suivants :

  $$   $ (signe du dollar)
  $_   Retour chariot et saut de ligne
  $B   | (tube ou "pipe")
  $D   Date actuelle
  $E   Code d'échappement (code ASCII 27)
  $G   > (signe supérieur à)
  $H   Retour arrière (efface le caractère précédent)
  $L   < (signe inférieur)
  $N   Lecteur actuel
  $P   Lecteur et chemin actuels
  $Q   = (signe d'égalité)
  $T   Heure actuelle
  $V   Numéro de version du shell de commande de FreeDOS

Commentaires :

  Tapez PROMPT sans paramètres pour remettre l'invite à zéro avec la
  valeur par défaut. PROMPT peut aussi être utilisée en dehors d'un
  fichier batch.
  Lorsque NANSI.SYS est chargé, PROMPT peut également utiliser des
  couleurs d'avant et d'arrière-plan différents avec le réglage :
  prompt $e[#;#;#m. PROMPT est une commande interne à command.com
  et ne nécessite aucun autre fichier pour fonctionner.

Exemples :

  DANS UN FICHIER .BAT / AUTOEXEC.BAT:
    prompt $p$g            Cela est identique à la valeur par défaut
    prompt $p$g Veuillez entrer une commande !
  Lorsque NANSI.SYS est chargé, vous pouvez aussi modifier les
  couleurs :
    prompt $e[1;33;44m pour du gras / jaune / bleu.

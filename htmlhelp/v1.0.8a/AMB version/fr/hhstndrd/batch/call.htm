Commande : call

  CALL appelle un programme batch depuis un autre.
  CALL est une commande de FICHIER BATCH / AUTOEXEC.BAT.

Syntaxe :

  CALL [/S | /N] [/Y] [lecteur][chemin]nomfichier [paramètres-batch]
       lecteur    La lettre du lecteur, par exemple C:
       chemin     Le répertoire, par exemple \exemple\
       nomfichier Le fichier batch qui doit être appelé

Options :

  paramètres-batch   Spécifie les informations de ligne de commande
                     nécessaires au programme batch, par exemple
                     test.bat
  /S                 Applique l'échange de FreeCOM
  /N                 Empêche l'échange de FreeCOM
  /Y                 Active le mode de traçage pendant l'exécution de
                     la commande.

Commentaires :

  Lorsque le fichier batch appelé est terminé, le traitement du fichier
  batch d'origine reprend.
  Les fichiers batch peuvent être exécutés directement depuis un autre
  fichier batch sans la commande call, mais le traitement du fichier
  batch d'origine ne reprendra pas dans ce cas.
  CALL est une commande interne à command.com et n'a besoin d'aucun
  autre fichier pour fonctionner.

Exemples :

  DANS UN FICHIER .BAT / AUTOEXEC.BAT:
    call c:\fdos\bin\defrag.exe

Commande : autoexec.bat

  AUTOEXEC.BAT est un fichier de commandes qui est exécuté
  automatiquement lorsque DOS démarre.
  FreeDOS accepte également d'autres noms .bat au lieu de AUTOEXEC.BAT.
  Les fichiers batch sont des fichiers textes, mais avec l'extension
  de fichier .bat. Ils peuvent être créés et édités avec un éditeur
  de texte tel qu'EDIT ou EDLIN. Pour exécuter un fichier batch,
  tapez simplement son nom à l'invite DOS, avec ou sans l'extension
  .bat. AUTOEXEC.BAT doit TOUJOURS se trouver dans le répertoire racine
  du média d'amorçage.

Syntaxe :

  - aucune - copiez simplement autoexec.bat dans le répertoire racine
  (A:\ ou C:\)

Options :

  - aucune -

Commentaires :

  FreeDOS accepte également d'autres noms .bat au lieu d'AUTOEXEC.BAT
  Vous devez seulement charger :
  "shell=LECTEUR:\CHEMIN\command.com /p=LECTEUR:\CHEMIN\NOMFICHIER.bat"
  dans CONFIG.SYS / FDCONFIG.SYS.
  De nombreux fans de FreeDOS utilisent "FDAUTO.BAT" au lieu de
  "AUTOEXEC.BAT".
  Vous pouvez utiliser AUTOEXEC.BAT pour charger divers pilotes .com et
  .exe, par exemple un pilote clavier, souris, de lecteur de CD-ROM et
  ainsi de suite.
  Les commandes AUTOEXEC.BAT sont SEULEMENT des commandes qui sont
  internes à command.com et n'ont besoin d'aucun autre fichier pour
  fonctionner.
  Vous pouvez utiliser les commandes beep, cls, echo, lh, loadhigh,
  path, prompt et set en dehors d'un fichier batch également.

Exemples :

  DANS UN FICHIER .BAT / AUTOEXEC.BAT:
    lh C:\fdos\bin\mkeyb.exe
    lh C:\fdos\bin\ctmouse.exe

Voir également :

  fichiers batch beep          call          choice        cls
  echo           edit          edlin         for           goto
  if             lh            loadhigh      path          pause
  prompt         rem           set           shift

Entender el tema de Ayuda "Syntax":

  En la mayoría de los temas de ayuda se verá algo como esto:

    comando requerido [opcional] [/switch] [/A|/B]

  He aquí una explicación de las diferentes partes de esta línea:

    comando
  Este es el nombre del comando.

    necesario
  Todo lo que sigue al nombre del comando, que no está entre corchetes
  [y] es un parámetro necesario. El comando notificará un error si no se
  da esta información.

    [opcional]
  opciones se escriben entre corchetes [y]. Éstos cambian la forma en que
  el comando se ejecuta. Los corchetes no están destinados a ser escrito
  cuando se utiliza el comando.

    [/switch]
  Las opciones que comienzan con un / son llamados switches. El símbolo /
  es en realidad escrito cuando se utiliza el modificador .

    [/A|/B]
  Cuando hay un símbolo |, esto significa que hay que elegir entre dos
  opciones. O /A o /B está permitido, pero no ambos. El | no pretende ser
  escrito cuando se utiliza el comando.

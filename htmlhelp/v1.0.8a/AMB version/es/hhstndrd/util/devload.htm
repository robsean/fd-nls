Comando: devload

  DEVLOAD carga los controladores de dispositivo del shell.

Sintaxis:

  DEVLOAD [/H] [/Q] [/V] [/A] nombre_archivo [parámetros]
    Emula: DEVICE=nombre_archivo [parámetros] en CONFIG.SYS / FDCONFIG.SYS
  DEVLOAD [/?]
          nombre_archivo  El nombre del archivo que desea cargar
          parámetros      Parámetros del archivo para cargar

Opciones:

  /H  Trata de cargar el controlador de UMB.
  /Q  Modo silencioso.
  /V  Modo detallado.
  /A  Modo automático (lo fuerza para permanecer cargado).
  /?  Muestra la ayuda.

Comentarios:

  DEVLOAD Permite al usuario cargar los controladores mientras FreeDOS
  está corriendo en lugar de artualizar config.sys / fdconfig.sys
  y reiniciar.

Ejemplos:

  devload c:\fdos\bin\xcdrom.sys

Comando: trch

  TRCH convierte todos los casos de un cararcter en otro. Esto es realmente
  útil si usted necesita comvertir un archivo de texto entre UNIX (final
  de linea es "^J") y DOS (final de linea es "^M^J" o "\n").

Sintaxis:

  trch c1 c2 < archivo > archivo
       < archivo > = archivo de entrada
       archivo = archivo de salida

Opciones:

  c1  caracter1, ejem. 'A'
  c2  caracter2, ejem. 'B'

Comentarios:

   TRCH soporta NLS (soporte nacional de lenguaje).

Ejemplos:

  trch ^I * < test.txt > test2.txt   Remplaza tabs por '*'
  trch e X < test.txt > test3.txt    Remplaza 'e' por 'X'
  trch \n $ < test.txt > test4.txt   Remplaza fin de linea ('\n') por '$*'
  trch ^J ^M^J < unix.txt > dos.txt  Remplaza formato Unix por DOS
  trch \n ^J < dos.txt > unix.txt    Remplaza formato DOS por Unix

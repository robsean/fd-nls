Comando: dosfsck

  DOSFSCK chequea y repara sistemas de archivos de DOS.

Sintaxis:

  dosfsck [-aAflrtvVwy][-d ruta -d ...][-u ruta -u ...] dispositivo

Opciones:

  
  -a       Reparar de forma automática el sistema de archivos.
  -A       Formato del sistema de archivos Toggle Atari.
  -d ruta  Colocar ese archivo.
  -f       Salvar cadenas inutilizadas de los archivos.
  -l       Lista los nombres de ruta.
  -n       No-op, compruebe de forma no interactiva sin cambiar.
  -r       Reparar interactivamente el sistema de archivos.
  -t       Prueba los clústeres no válidos.
  -u ruta  Trata de recuperar ese archivo (no directorios).
  -v       Modo detallado.
  -V       Realizar un paso de verificación.
  -w       Escribe los cambios en el disco inmediatamente.
  -y       Similar a -a, para compativilizar con otro *fsck.
  --help   Muestra la ayuda.

Comentarios:

  Usa DOSFSCK si tu tienes problemas con archivos / particiones.
  DOSFSCK soporta nombres de archivos largos y FAT32.

Ejemplos:

  dosfsck -av c:

Comando: graphics

  GRAPHICS permite que la tecla ImpPnt (imprimir pantalla) imprima
  pantallas gráficas.
  GRAPHICS se ejecuta en modo de compatibilidad GRAPHICS de MS.

Sintaxis:

  GRAPHICS tipo [opciones]

Opciones:

  /B             Habilita el procesamiento de la paleta de color de
                 de fondo CGA 0
  /I             Imprima negro como blanco y blanco como negro (útil si
                 hay muchas áreas oscuras en la pantalla, para ahorrar
                 tinta / tóner: imprimirá negro donde la pantalla es
                 blanca, etc.).
  /LCD           Ignorado (aún no admitido).
  /LOW           No cargue alto (sin opción de MS)
  /PRINTBOX:LCD  Ignorado (aún no admitido).
  /PRINTBOX:STD  Ignorado (aún no admitido).
                 (intente usar /LCD para ver una explicación).
                 Esto le permitirá establecer una relación ancho-alto de
                 pantalla que no sea 4:3, que actualmente siempre se
                 supone.
  /R             Imprima en negro como blanco y negro como blanco.
  Tipos de impresoras compatibles:
    HPPCL (PCL), EPSON (ESC/P), POSTSCRIPT (PS)
  Puede usar lo siguiente para el tipo:
    Compatible con HP PCL:
      HPDEFAULT          (Compatible con HP PCL)
      DESKJET            (Compatible con HP PCL)
      RUGGEDWRITER       (Compatible con HP PCL)
      RUGGEDWRITERWIDE   (Compatible con HP PCL)
      THINKJET           (Compatible con HP PCL)
    Compatible con EPSON:
      EPSON              Muchas impresoras de impacto / cinta funcionan
                         con "EPSON"
    Compatible con POSTSCRIPT:
      LASERJET           (Compatible con POSTSCRIPT)
      LASERJET II        (Compatible con POSTSCRIPT)
      QUIETJET           (Compatible con POSTSCRIPT)
      QUIETJETPLU        (Compatible con POSTSCRIPT)
    Algunos de estos son equivalentes, como POSTSCRIPT y PS.
  Tipos no compatibles:
    Impresora de cinta de color IBM PC (color1, color4, color8)
    Impresora térmica convertible IBM (térmica)
    Impresora de gráficos IBM (gráficos [ancho])
    IBM proprinter
    IBM quietwriter

Comentarios:

  HP (GRAPH-HP - PCL) permite las siguientes opciones adicionales:
    /1  Use LPT1 (predeterminado)
    /2  Use LPT2
    /3  Use LPT3
    /C  para 300 ppp en lugar de 600 ppp
    /E  Modo económico
    /R  Para aleatorio en lugar de tramado ordenado
    /?  Muestra la ayuda
  PS (GRAPH-PS - Postscript) permite las siguientes opciones adicionales:
    /1  Use LPT1 (predeterminado)
    /2  Use LPT2
    /3  Use LPT3
    /C  Modo de compatibilidad (HP Laserjet)
    /E  Modo económico (solo 50 % de densidad)
    /?  Muestra la ayuda
  PIN (GRAPHPIN - EPSON) permite las siguientes opciones adicionales:
    /1  Use LPT1 (predeterminado)
    /2  Use LPT2
    /3  Use LPT3
    /9  Modo de 9 pines (120x72, con /C 60x72)
    /C  Modo de compatibilidad (120x60 ppp)
    /E  Modo económico (solo 50% de densidad)
    /R  Usa tramado aleatorio, no ordenado
    /X  Enviar datos de impresión para líneas enteras en lugar de en
        bloques
    /?  Muestra la ayuda
  Después de cargar el controlador, puede imprimir capturas de pantalla
  usando 'shift - ImpPnt' incluso en modos gráficos.

Ejemplos:

  graphics HPDEFAULT /low /I /E /1

Ver también:

  graphics-ps...

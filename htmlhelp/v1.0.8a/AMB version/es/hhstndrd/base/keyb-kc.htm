Comando: kc

 KC es la CLAVE del descriptor del lenguaje del teclado en un compilador
 de archivos. 
 KC toma un archivo descriptor del idioma del teclado fuente (en lenguaje
 KEY) y lo convierte en un KeybCB simple o en un archivo KL para usar con
 FD-KEYB.

Sintaxis:

  KC  fuente_nombre_archivo [salida_nombre_archivo] [/B] [/?]
      fuente_nombre_archivos  El nombre del archivo fuente que se abrirá.
                              La extensión KEY se agrega si no se 
                              encuentra ninguna extensión.
      salida_nombre_archivo   El nombre del nombre del archivo compilado
                              de destino. Si se omite, se crea el mismo
                              nombre de archivo con la extensión KL.

Opciones:

  /B  Crea archivos KeybCB desnudos, sin encabezado KL.
  /?  Muestra la ayuda.

Comentarios:

  Lea la documentación adjunta para saber más sobre el lenguaje KEY y los
  formatos de salida. KC viene con el paquete KC200. Los EXITCODES
  (ERRORLEVEL) son 0 para corecto, el número de error de sistaxis para el
  error de sitaxis, y agregan 100 al código de error crítico, para los
  códigos de error crítico de KC.

Ejemplos:

  KC FR                Compila FR.KEY en FR.KL
  KC SP.KEY SP.KCB /B  Compila SP.KEY en un bloque de control de teclado
                       desnudo SP.KCB

Comando: sort

  SORT ordena un archivo o entrada para mostrar.
  El archivo de entrada no se modifica.

Sintaxis:

  sort [/R] [/N] [/+num] [/A] [/?] [unidad][ruta]Nom_Archivo
  comando | sort [/R] [/N] [/A] [/+num] [/?]
       unidad       La letra de unidad, p. ej. C:
       ruta         El directorio, p. ej. \ejemplo\
       Nom_Archivo  El controlador (archivo) a cargar, p. ej. ejem.txt

Opciones:

  /R     Ordenar en orden inverso (Z - A).
  /A     Ordenar por ASCII, ignorar PAIS.
  /N     Habilita ORDEN de clasificación basada en NLS / CODEPAGE /
         COUNTRY.
  /+num  Comience a ordenar en la columna num en cada línea de texto.
         Predeterminado = 1.
  /?     Muestra la ayuda.

Comentarios:

  SORT no cambia el archivo de entrada, solo la salida.
  SORT admite NLS (Soporte de idioma nacional).

Ejemplos:

  Muestra el directorio actual en orden alfabético:
    dir | sort  
  Ordena un archivo y lo excribe en otro archivo:
    sort desordenado.txt > ordenado.txt  
    sort           -ENTER-   (alternativo: sort /+2)
    este           -ENTER-
    es             -ENTER-
    un             -ENTER-
    orden          -ENTER-
    de             -ENTER-
    sentencia      -ENTER-
  (CTRL-Z)  (sale del comando y lo ejecuta)
    da en salida:             ordenado por segunda letra por línea:
    de                        de
    es                        sentencia
    este                      un
    orden                     orden
    sentencia                 es
    un                        este

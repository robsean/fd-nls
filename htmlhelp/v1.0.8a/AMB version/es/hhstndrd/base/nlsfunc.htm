Comando: nlsfunc

  NLSFUNC agrega la funcionalidad NLS (Soporte Nacional de Lenguaje).

Sintaxis:

  NLSFUNC [/Y] [unidad][ruta]NomArchivo
  NLSFUNC [/?]
    unidad                     La letra de unidad, p. ej. C:
    ruta                       El directorio, p. ej. \ejemplo\
    NomArchivo                 El archivo que contiene la 
                               información NLS
    [unidad][ruta] NomArchivo  Unidad y ruta a un archivo válido
                               COUNTRY.SYS, que contiene la información
                               específica del país.

Opciones:

  /Y                 Carga la tabla opcional SI / NO.
  /?                 Muestra la ayuda.

Comentarios:

  Esta implementación de NLSFUNC es específica de FreeDOS.
  No funcionará con MS-DOS o cualquier otra variante de DOS.
  ¡Tampoco funciona con el núcleo 2.036!

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    COUNTRY=34,858,C:\FDOS\COUNTRY.SYS
  EN AUTOEXEC.BAT:
    LH DISPLAY CON=(EGA,858,2)
    MODE CON CP PREP=((850) C:\FDOS\CPI\EGA.CPX)
    MODE CON CP PREP=((,437) C:\FDOS\CPI\EGA.CPX)
    LH NLSFUNC /Y
  Luego cambie las páginas de códigos usando CHCP.
  Si no necesita cambiar las páginas de códigos, simplemente omita las
  líneas MODE.

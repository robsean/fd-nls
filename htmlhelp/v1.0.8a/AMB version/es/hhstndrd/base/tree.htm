Comando: tree / pdtree

  TREE / PDTREE muestra gráficamente la estructura del directorio 
  [carpeta] de una unidad o ruta determinada. Opcionalmente incluye una
  lista de archivos en la pantalla.

Sintaxis:

  TREE [d:][ruta] [/F] [/A] [/?]
  PDTREE [d:][ruta] [/F] [/A] [/V] [/S][/P] [/?]
       d:ruta  El camino y la ruta donde comenzar. Si no se proporciona
               ninguna ruta, tree/pdtree se iniciará desde el directorio
               actual.

Opciones:

   /F  Muestra los nombres de los archivos en cada directorio.
   /A  Utilice caracteres ASCII para representar visualmente la 
       estructura del directorio.
   /?  Muestra ayuda y salir.

 Opciones adicionales en FreeDOS pdTree: (No disponible en el árbol
 estándar).
   /V  Muestra información de versión y salir.
   /S  Solo nombres cortos (usa nombres cortos, es decir alternativos,
       cuando existe nombre de archivo). Solo la versión de DOS,
       deshabilita el uso de API de LFN para encontrar primero/siguiente.
   /P  Pausa después de cada página. (Nota: actualmente está
       predeterminado en la pantalla 80x25) Solo versión Win32, ignora la
       opción de pausa sin la salida se redirige a un archivo o se
       canaliza a otro programa, como tree | more; también intenta usar el
       tamaño actual de la consola (Todo para la versión DOS).

Comentarios:

  TREE es compatible con NLS (soporte de idioma nacional).

Ejemplos:

  Para mostrar la jerarquía completa de directorios de la unidad C y
  pausar después de cada pantalla completa, ejecute el árbol que
  especifica C: como la ruta de inicio y canalice su salida al comando
  MORE:
    tree C:\ | more
  Cuando se usa pdTree, el ejemplo anterior también se puede lograr
  usando:
    pdtree C:\ /p
  Para mostrar la jerarquía de directorios de un subdirectorio y hacer
  que la salida solo use caracteres ASCII estándar (como para imprimir o
  enviar a alguien que pueda verlo sin compatibilidad con caracteres de
  dibujo de línea/box de DOS), especifique el subdirectorio y el
  modificador /A:
    tree ASUBDIR /a
  Para mostrar la estructura de un árbol de directorios y los archivos
  dentro de cada directorio, agregue el modificador /F:
    tree ASUBDIR /F | more

Comando: comp

  COMP compara dos archivos.

Sintaxis:

  COMP [/opciones] [unidad][ruta]archivo1 [unidad][ruta]archivo2
       donde [archivo1] y [archivo2] son nombres de archivo (posiblemente
       con comodines) o nombres de directorio. Si se omite [archivo2], se
       supone el directorio actual.
       unidad    La letra de la unidad, por ejemplo C:
       ruta      El directorio, \ejemplo\
       archivo1  El primer archivo para comparar, test1.txt
       archivo2  El segundo archivo para comparar, test2.txt

Opciones:

  #       No imprima más de # errores por archivo. Si # = 0, no limite el
          número de errores. El valor predeterminado es 0 si [archivo1]
          se refiere a un solo archivo, 10 si hay más de un archivo.
  /d      Muestra las diferencias en formato decimal. El valor
          predeterminado es hexadecimal.
  /a      Muestra las diferencias usando caracteres.
  /L      Muestra el número de línea para las diferencias que ocurren.
  /n=nnn  Compara las primeras nnn líneas de ambos archivos, incluso si
          los archivos son de tomaño diferente.
  /c      Ignora mayúsculas y minúsculas (el valor predeterminado es
          mayúsculas y minúsculas).
  /?      Muestra la ayuda.

Comentarios:

  - ninguna -

Ejemplos:

  comp c:\program\help.txt c:\program\help1.txt

Comando: diskcopy

  DISKCOPY copia el contenido de un disquete a un segundo disquete.
  También crea, copia y restaura archivos de imagen de disquete. Las
  imágenes pueden ser utilizadas por SHSUFDRV.

Sintaxis:

  Diskcopy [fuente][destino][/a][/v][/m][/i][/o][/1][/x][/d][/r][/f][/t]
  Diskcopy [/?]
           fuente:      unidad o archivo de imagen para copiar.
           destino:     unidad o archivo de imagen para copiar.

Opciones:

  /a  da una advertencia audible para la acción del usuario.
  /v  verificar lecturas y escrituras.
  /m  solo usa memoria para copia de disco.
  /i  muestra el uso de memoria (informativo).
  /o  sobrescribir el destino, si ya existe (en el caso de un archivo de
      imagen).
  /x  siempre sale automáticamente.
  /d  supone que el disco ya está en la unidad.
  /r  entra en modo de recuperación de error de disco.
  /f  realiza una copia rápida de disco (solo copia sectores llenos).
  /t  no pregunte al disco de destino si copia el archivo de imagen al
      mismo disco.
  /1  no-op incluido para compatibilidad con MS-DOS. En otras versiones
      de DOS, esto copió solo el primer lado.
  /?  Muestra la ayuda.

Comentarios:

  Un signo menos después de una opción deshabilita la opción. Si el
  origen y/o el destino es una unidad, por ejemplo "a:", entonces se
  trata como un disquete, de lo contrario, se trata como un archivo de
  imagen.
  El origen y el destino pueden ser la misma unidad; DISKCOPY primero
  copiará de la fuente y luego le pedirá que cambie los disquetes. Luego
  escribirá al destino.
  DISKCOPY utiliza el formato rawrite de Linux para sus archivos de
  imagen. Por lo tanto, Rawrite puede usar los archivos de imagen de
  DISKCOPY y viceversa.
  DISKCOPY no puede leer ni sobrescribir los disquetes de Linux. Sin
  embargo, puede escribir una imagen de disquete de Linux en un disco de
  DOS. DISKCOPY admite NLS (soporte de idioma nacional).

Ejemplos:

  diskcopy a: a:            Copia de un disquete a otro
  diskcopy a: c:\image.img  Copia el disquete a una imagen en C:\image.img
  diskcopy c:\image.img a:  Copia la imagen en un disquete

Comando: print

  PRINT imprime un archivo en segundo plano, mientras ejecuta otros
  comandos de DOS.

Sintaxis:

   print [opciones] [[unidad:][ruta]archivo] 
         [[unidad:][ruta]archivo] [...]
     unidad:ruta\archivo  El archivo que desea imprimir.

Opciones:

  /D:dispositivo  Especifique el nombre del dispositivo para imprimir. Si
                  no se proporciona, Print utilizará el LPT
                  predeterminado: dispositivo.
  /B:tamaño       Establezca el tamaño del búfer, en bytes, en el rango
                  512-16384. El valor predeterminado es 512.
  /U:tics         El número de tics de reloj, en el rango 1-255, que
                  imprimir debe esperar a que la impresora esté
                  disponible antes de imprimir el tabajo.
  /M:tics         El número máximo de tics para enviar un solo carácter
                  a la impresora, En el rango 1-255.
  /S:tics         El número de tics de reloj, en el rango de 1-255, para
                  el proceso de cola de fondo. El valor predeterminado es
                  2. Los números más grandes acelerarán la impresión pero
                  telentizarán otros programas de DOS.
  /Q:nn           El número de archivos que se permitirán en la cola de
                  impresión, en el rango 1-32. El valor predeterminado
                  es 10.
  /T              Eliminar todos los archivos en la cola de impresión.
                  Termina el trabajo que se está imprimiendo actualmente.
  /C              Cancelar tabajos en la cola de impresión. Si se usa
                  con una lista de nombres de archivo, solo esos archivos
                  se eliminan de la cola. Si se usa solo, todos los
                  trabajos se cancelan (aunque el trabajo actual no
                  finaliza).
  /P              Agregue un archivo a la cola de impresión. Si esto
                  precede a una lista de archivos, solo se imprimen esos
                  archivos. Puede usar /P y /C juntos en la misma línea
                  de comando.

Comentarios:

  Si no se dan opciones para IMPRIMIR, solo archivos, se suponen /P. Si
  no se proporcionan opciones ni archivos, imprimir muestra el contenido
  de la cola de impresión.
  Esta IMPRESION aquí solo toma /1 /2 /3 como "/D:lpt1", etc., opciones
  equivalentes o un nombre de archivo para imprimir un archivo. Utilice
  la herramienta PRINTQ gratuita para "print archivo", "print /t" 
  (borrar cola incluyendo trabajo actual), "print /c" (borrar cola) y
  "print /p archivo" (agregar archivo a la cola).

Ejemplos:

  print c:\foo.txt c:\test.txt

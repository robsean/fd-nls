Comando: edit

  EDIT es el editor de texto de FreeDOS.

Sintaxis:

  edit [/B][/I][/H][/R][/?] [[unidad][ruta]archivo]
       unidad   La letra de unidad, p. C:
       ruta     El directorio, p. \ejemplo\
       archivo  El archivo, p. testo.txt. Los comodines se pueden usar
                aquí. También puede abrir archivos dentro del programa.

Opciones:

  /B     Utilice una pantalla en blanco y negro (mono).
  /I     Utilice el esquema de color inverso.
  /H     Utilice 43/50 líneas en EGA/VGA (la resolución de video/texto
         más alta disponible).
  /R     Abre todos los archivos de solo lectura.
  /?     Muestra la ayuda.

Comentarios:

  Se puede usar un editor como EDIT para crear archivos de texto simples.
  También se puede usar para modificar archivos como CONFIG.SYS / 
  FDCONFIG.SYS y AUTOEXEC.BAT. EDIT tiene incorporado un sistema de ayuda
  integral. EDIT no puede abrir archivos de tamaño superior a 64K.

Ejemplos:

  edit c:\ejemplo.txt   abre c:\ejemplo.txt para escribir en él (el
                        archivo ejemplo.txt ya debe existir)
  edit c:\autoexec.bat
  edit c:\fdconfig.sys

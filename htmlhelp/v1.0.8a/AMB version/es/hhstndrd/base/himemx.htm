Comando: himemx

  HIMEMX es el administrador de memoria de alto nivel.
  HIMEMX debe cargarse en CONFIG.SYS / FDCONFIG.SYS.

Sintaxis:

  HIMEMX [/MAX=####] [/METHOD:xxx] [/HMAMIN=n] [/NUMHANDLES=m]
         [/TESTMEM:ON|OFF] [/VERBOSE] [/NOABOVE16] [/X] [/LOG]

Opciones:

  /MAX=#####       Limite la memoria controlada por XMM a #####K.
  /METHOD:xxx      Especifica el método que se utilizará para el manejo
                   de A20.
                   Posibles valores para xxx:
                   ALWAYSON    Asuma que la línea A20 está ENCENDIDA
                               permamentemente
                   BIOS        Use BIOS para alternar la línea A20
                   FAST        Use el puerto 92h, omita la prueba INT 15h
                   PS2         Use el puerto 92h, omita la prueba PS/2
                   KBC         Usa el controlador del teclado
                   PORT92      Use el puerto 92h siempre
  /HMAMIN=n        Especifica la cantidad mínima de Kbs de HMA que un
                   programa debe solicitar para obtener acceso a la HMA
                   (valor predeterminado: 0Kb).
  /NOX2MAX32       No hay límite para los informes de memoria 
                   libre/disponible de XMS 2.0 (predeterminado).
  /NUMHANDLES=m    Especifica el número de manejadores XMS disponibles
                   (def: 48).
  /TESTMEM:ON|OFF  Realiza u omite una prueba de memoria extendida
                   (def: OFF).
  /VERBOSE         Proporciona información adicional durante la
                   instalación.
  /NOABOVE16       No utilice la función INT 15h E801h para detectar > 64M.
  /X               No utilice la función INT 15h E820h para detectar > 64M.
  /X2MAX32         Límite de XMS 2.0 libre/disponible. informe de memoria
                   a 32M-1K
  /LOG             Registra la actividad del controlador en la pantalla.
  /?               Muestra la ayuda.

Comentarios:

  HIMEMX requiere una CPU 80386+. JEMM386 se puede cargar después de
  este controlador. Cargue HIMEMX como DEVICE = HIMEMX.EXE en CONFIG.SYS
  (¡aunque es un archivo ".exe"!)

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    device=c:\fdos\bin\himemx.exe
    device=c:\fdos\bin\jemm386.exe

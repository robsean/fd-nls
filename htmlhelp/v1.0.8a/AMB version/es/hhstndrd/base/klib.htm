Comando: klib

  KLIB es una biblioteca para archivos KL: agrupa los archivos KL en un
  solo archivo de biblioteca. KLIB le permite crear bibliotecas, 
  enumerar, agregar y eliminar archivos KL dentro de una biblioteca, y
  finalmente dejar de editarla (cerrarla).

Sintaxis:

  KLIB  lib_nombre_archivo [{+|-}KL_nombre_archivo] [/B] [/Z] 
        [/M=fabricante] [/D=Desc]
  KLIB  /?

Opciones:


  lib_nombre_archivo  El nombre del archivo de la biblioteca.
  KL_nombre_archivo   Nombre del archivo KL que se agregará o eliminará.
                      Use + para AGREGAR el archivo KL (el archivo
                      debería existir)
                      Use - para ELIMINAR el archivo KL (el archivo
                      en la biblioteca)
  /B                  Evita crear archivos de respaldo
  /Z                  Después de la operación, la biblioteca se cierra
                      y ya no se puede editar.
  Fabricante          Nombre del fabricante de la biblioteca
                      (solo creación)
  Desc                Descripción de la biblioteca (solo creación)

Comentarios:

  Lea la documentación adjunta para saber más sobre el lenguaje KEY y los
  formatos de salida.
  KLIB tiene los siguientes EXITCODES (ERRORLEVEL):
    0                              = bien
    número de error de sintaxis    = número de error de sintaxis
    número de código crítico + 100 = número de error de código crítico
  KLIB viene con el paquete KC200.

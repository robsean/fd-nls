Comando: exe2bin

  EXE2BIN convierte programas EXE a formato binario. Esta es una
  herramienta de desarrollador de software.

Sintaxis:

  EXE2BIN [opciones] archivo_exe[.exe] [archivo_bin]
          archivo_exe  El archivo a convertir
          archivo_bin  El nuevo archivo bin

Opciones:

  /Q        Suprime mensajes informativos.
  /H        Muestra el encabezado exe.
  /R        Mostrar reubicaciones.
  /L=(seg)  Reubique exe_file en segmento (seg).
  /X        Comportamiento extendido, p. ej. archivos > 64KB.
  /?        Muestra la ayuda.

Comentarios:

  - ninguno -

Ejemplos:

  exe2bin c:\fdos\bin\ctmouse.exe c:\ctmouse

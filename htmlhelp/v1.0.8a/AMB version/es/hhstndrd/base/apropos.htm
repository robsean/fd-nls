Comando: apropos

  APROPOS muestra descripciones de programas basados en los archivos
  Appinfo (.LSM). Use APROPOS para encontrar programas que sean 
  apropiados para el tema dado. Use WHATIS con un nombre de programa
  para obtener una descripción de una línea de ese programa. Use WHATIS
  sin un nombre de programa para obtener una descripción de una línea de
  todos los programas y utilidades externos en su instalación de FreeDOS.
  Los comandos interno (comando de command.com) no se muestran.

Sintaxis:

  APROPOS [Opciones] tema
          Encuentra comandos relacionados con el tema dado.
  WHATIS [Opciones] [comando]
          Muestra una descripción de una línea del comando dado. Si no
          se especifica ningún comando, se enumeran todos los comandos
          y sus descripciones de una línea.
  WHATIS  /B [opciones]
          Reconstruye la base de datos whatis para usar con WHATIS y
          APROPOS.

Opciones:

  /S [ruta]  Ruta no predeterminada a los archivos lsm appinfo.
  /D [ruta]  Ruta no predeterminada para la ubicación de la base 
             de dados.
  /?         Muestra la ayuda.

Comentarios:

  Estos son clones de FreeDOS de los comandos de Linux "apropos y 
  whatis". APROPOS y WHATIS basan su ayuda en los archivos lsm 
  almacenados en el directorio APPINFO. La primera vez que se ejecuta,
  APROPOS/WHATIS crea una base de datos y la almacena en el directorio
  de ayuda en la instalación de FreeDOS. Esta base se datos se puede
  recrear usando:
    WHATIS /B

Ejemplos:

  apropos edit  mostrará una lista de editores.
  apropos disk  mostrará una lista de utilidades de disco.
  whatis edit   mostrará una breve información sobre la edición.
  whatis        mostrará una breve información sobre los comandos de DOS.

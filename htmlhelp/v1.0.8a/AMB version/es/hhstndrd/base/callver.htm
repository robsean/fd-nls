Comando: callver

  CALLVER establece el número de versión del intérprete de comandos de
  FreeCOM. 
  CALLVER ejecuta un programa (en un nuevo shell, utilizando COMSPEC)
  con un número de versión falso de DOS, o modifica globalmente (solo
  en FreeDOS) la versión de DOS informada (CALLVER 0.00 revierte)

Sintaxis:

  CALLVER x.xx [PROGRAMA [ARGUMENTOS]][/?]

Opciones:

  PROGRAMA    Nombre del programa que desea iniciar.
  ARGUMENTOS  Opciones del programa que desea iniciar.
  /?          Muestra la ayuda.

Comentarios:

  x.xx es el número de versión; cada x puede ser cualquier dígito de 
  0-9. Este comando puede usarse para programas antiguos que buscan
  una determinada versión de DOS.

Ejemplos:

  callver 6.20
  callver 6.22 format c: /q
  callver 0.00               -> muestra 7.10

Comandos: fasthelp (fdhelp)

  FASTHELP es un clon de fasthelp para FreeDOS. Muestra la ayuda sobre un
  tema, con el estilo de las páginas man de linux.

Sintaxis:

  fasthelp [asunto]

Opciones:

  asunto  El tema del que se desea información.
  Entorno:
    PAGER:    Programa Pager muestra un archivo de texto.
    HELPPATH: Directorio que contiene tus archivos de ayuda.
    LANG:     Lenguaje preferido de tus archivos de ayuda.

Comentarios:

  Utilice la variable de entrono HELPPATH y LANG  para especificar la
  ubicación y el idioma de los archivos de ayuda, respectivamente. La ruta
  de acceso predeterminada es ../help y el lenguaje por defecto es EN 
  (inglés).
  También existe la variable de entorno pager, que se puede utilizar para
  establecer un programa alternativo para mostrar texto de los archivos de
  ayuda. Las llamadas a fasthelp desviarán archivos de texto al programa
  de PAGER.
  Consulte SET para obtener más información sobre la configuración de
  variables de entorno.
  Fasthelp también se nombra como "FSUITE" o "FSUITE04" o "FS04de" 
  (alemán) o "FSUITE04de" (alemán)

Ejemplos:

  fasthelp       Muestra tu lista de temas
  fasthelp edit  Muestra tu información acerca de edit

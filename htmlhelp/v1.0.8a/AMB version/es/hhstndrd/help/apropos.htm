Comando: apropos

  APROPOS muestra las descripciones de los programas basados en los 
  archivos (.LSM) de Appinfo.
  Utilice APROPOS para encontrar programas que son apropiados para el tema
  dado.
  Utilice WHATIS con nombres de programas para optener la descripción en 
  linea de un programa.
  Utilice WHATIS solamente para obtener una descripción en linea de
  todos los programas y utilidades externos en la instalación de FreeDOS.
  Los comandos internos (comandos de command.com) no se muestran.

Sintaxis:

  APROPOS [opciones] asunto
          Busca comandos relacionados con el tema dado.
  WHATIS [opciones] [comando]
          Muestra en linea la descripción de un comando dado.
          Si no introduce un comando específico, todos los comandos y sus
          descripciones serán mostrados en linea.
  WHATIS  /B [opciones]
          Para recompilar la base de datos que se utiliza con WHATIS y 
          APROPOS

Opciones:

  /S [ruta]  Nombre por defecto de la ruta de los archivos lsm de appinfo.
  /D [ruta]  Nombre por defecto de la ruta donde se encuentra la base de
             datos de WHATIS.
  /?         Muestra la ayuda.

Comentarios:

  Estos son FreeDOS clones de los comandos de Linux "apropos y whatis"
  APROPOS y WHATIS basan su ayuda en los archivos lsm almacenados en el 
  directorio APPINFO.
  La primera vez que se ejecuta, APROPOS / WHATIS se construye una base de
  datos y la almacena en el directorio de ayuda en la instalación de
  FreeDOS. Esta base de datos puede ser recreada usando:
    WHATIS /B

Ejemplos:

  apropos edit  Muestra una lista de editores
  apropos disk  Muestra una lista  de utilidades de disco
  whatis edit   Muestra una corta información acerca de edit
  whatis        Muestra una corta información acerca de los comandos de 
                DOS

Comando: type

  TYPE muestra el contenido de los archivos de texto.

Sintaxis:

  TYPE [unidad][ruta]nombre_archivo
  TYPE [/?]
       unidad          Especifica la letra de la unidad donde se encuetra
                       el archivo, por ejemplo, C:
       ruta            Especifica la ruta de acceso donde se encuentra el
                       archivo, por ejemplo, \ejemplo\
       nombre_archivo  Especifica el archivo que se va a mostrar, por 
                       ejemplo, test.txt

Opciones:

  /?  Muestra la ayuda.

Comentarios:

  Type mestra el texto completo. Si el texto es demasiado largo se detiene
  al final. Se puede detener antes usando el conjunto de comandos:
  TYPE | MORE xy.txt
  TYPE es un comando interno de command.com y no necesita archivo para
  poder trabajar.

Ejemplos:

  type C:\autoexec.bat
  type | more C:\autoexec.bat
  type | more C:\config.sys
  type | more C:\example\test.txt

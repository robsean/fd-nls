Comando: popd

  POPD saca un directorio de la pila de directorios (creado por pushd)
  y cambios en el mismo.

Sintaxis:

  POPD [*] [/?]

Opciones:

  *   Borra la pila de directorios.
  /?  Muestra la ayuda.

Comentarios:

  PUSHD es muy parecido a un CD,excepto que almacena el directorio actual
  antes de cambiarlo más tarde, POPD se puede utilizar para restaurarlo.
  PUSHD su puede usar varias veces, almacenando una pila de directorios.
  El uso de PUSHD sin especificar un directorio almacenará el actual
  directorio, pero no cambiarlo.
  POPD es un comando interno de command.com y no necesita otro archivo
  para que funcione.

Ejemplos:

Este comando            cambia el directorio actual a /muestra:
============            =========================================
cd c:\fdos\bin          c:\fdos\bin
pushd c:\games          c:\games      (y almacena c:\fdos\bin)
pushd c:\utils          c:\utils      (y almacena c:\games)
pushd c:\packages       c:\packages   (y almacena c:\utils)
dirs                    muestra los directorios almacenados por pushd:
                          0001=c:\fdos\bin
                          0002=c:\games
                          0003=c:\utils
popd                    c:\utils      (y elimina c:\utils (0003)
                                      desde la pila de directorios)
popd                    c:\games      (y elimina c:\games (0002)
                                      desde la pila de directorios)
popd                    c:\fdos\bin   (y elimina c:\fdos\bin (0001)
                                      desde la pila de directorios)
popd                    muestra el mensaje: 
                          "Pila de directorios vacia"

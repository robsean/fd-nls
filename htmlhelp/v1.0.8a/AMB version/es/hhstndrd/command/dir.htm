Comando: dir

  DIR muestra una lista de archivos y subdirectorios en un directorio.

Sintaxis:

  DIR [unidad][ruta][nombre_archivo] [/P] [/W] [/A[[:]atributos]]
      [/O[[:]orden]] [/S] [/B] [/L] [/Y] [/?]
      unidad          La letra de la unidad, p.ej. C:
      ruta            El directorio, p.ej. \ejemplo\
      nombre_archivo  El archivo a mostrar, p.ej. test.txt
                      [unidad][ruta][nombre_archivo]
                      Especifique la unidad, directorio y/o archivos para
                      listar. (Podría ser espcificación de archivo 
                      mejorada o múltiples especificaciones de archivo)

Opciones:

  /P  Muestra una página de la lista a la vez.
  /W  Muestra el listado en formato ancho.
  /A:atributos  
      Muestra solo los archivos y directorios con el atributo especificado
      siendo por defecto /a:hsdar. Puesde usar lo siguiente para los 
      atributos:
        h | -h  Archivos ocultos        -h archivos que no están ocultos
        s | -s  Archivos del sistema    -s archivos que no son del sistema
        d | -d  Directorios             -d archivos
        a | -a  Archivos con archivo    -a archivos que no son archivos
                binario                    binarios
        r | -r  Archivos solo-lectura   -r archivos que no son solo-lectura
  /O:orden   
      Muestra el listado en el orden especificado. El valor predeterminado
      es /O:g. Puede usar lo siguiente para especificar el orden de 
      clasificación:
        a | -a  Alfabéticamente (A-Z)       -a Alfabéticamente (Z-A)
        e | -e  Por extensión (A-Z)         -e Por extensión (Z-A)
        d | -d  Por fecha y hora            -d Fecha/hora en orden inverso
        s | -s  Por tamaño(pequeño-grande)  -s tamaño (grande-pequeño)
        g | -g  Agrupar primero directorios -g directorios al final
  /S  Muestra los archivos en el directorio especificado y todos los 
      subdirectorios.
  /B  Usa formato simple (Sin información de encabezado o resumen).
      Esto muestra una lista de un nombre por línea
  /L  Muestra los nombres en minúsculas
  /Y o /4  
      Muestra año de 4 dígitos.
  /?  Muestra la ayuda.

Comentarios:

  Para personalizar la apariencia predeterminada de dir, use la variable
  de entrono DIRCMD. Por ejemplo:
    SET DIRCMD=/w/p
  Para anular algo establecido en DIRCMD, use un signo menos ("-") en ese
  interruptor en la línea de comando. (por ejemplo /-w)
  DIR es un comando interno de command.com y no necesita otro archivo para
  que funcione.

Ejemplos:

  dir c:\ejemplo /p
  dir c:\ejemplo /w
  dir c:\ejemplo /O: a
  dir c:\fdos | more

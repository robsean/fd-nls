Comando: rem

  REM registra comentarios (observaciones) en un archivo por lotes o en 
  CONFIG.SYS.
  REM es un comando de archivo por lotes / AUTOEXEC.BAT.

Sintaxis:

  REM [comentario]

Opciones:

  comentario   cualquier comentario que desee añadir para recordar por qué
               hizo esto o aquello; también puede ser un comando, que 
               no se ejecutará (tal vez con fines de prueba).

Comentarios:

  FreeDOS ignora todo en una línea REM.
  Utilice comentarios para agregar notas explicativas al archivo 
  config.sys, y para deshabilitar las líneas en un archivo config.sys para
  que no se ejecuten.
  En CONFIG.SYS también puede utilizar ";" al pricipio de la línea.
  REM es un comando interno a command.com y no necesita ningún otro 
  archivo para poder trabajar. REM también existe como comando de
  CONFIG.SYS / FDCONFIG.SYS.

Ejemplos:

  EN UN ARCHIVO .BAT / AUTOEXEC.BAT:
    REM establezca la última letra de unidad disponible:
    lastdrive=z
    REM ya no quiero user esta unidad:
    REM DEVICE=FOO.SYS

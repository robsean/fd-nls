Comando: cls

  CLS Borra la pantalla.
  CLS es un comando de ARCHIVO POR LOTES / AUTOEXEC.BAT.
  También se puede usar en línea de comando.

Sintaxis:

  cls

Opciones:

  - ninguna -

Comentarios:

  CLS elimina todo el texto en la pantalla.
  CLS también se puede usar fuera de un archivo por lotes.
  CLS es un comando interno command.com y no neceesita otro archivo para
      que funcione.

Ejemplos:

  EN UN ARCHIVO .BAT/AUTOEXEC.BAT O EN LA LíNEA DE COMANDOS:
    cls

Comando: pushd

  PUSHD empuja el directorio actual en la pila de directorios, con la
  opción de cambiar el directorio de trabajo actual.

Sintaxis:

  PUSHD [[unidad]ruta] [/?]
        [[unidad]ruta) es una ruta a la que desea cambiar.

Opciones:

  /?  Muestra la ayuda.

Comentarios:

  PUSHD es muy parecido a CD, excepto que almacena el directorio actual
  antes de cambiarlo. Más adelante, POPD se puede utilizar para 
  restaurarlo. PUSHD se puede utilizar varias veces, almacenando una pila
  de directorios. El uso de PUSHD sin especificar almacenará el directorio
  actual, pero no lo cambiará.
  PUSHD es un comando interno a command.com y no necesita ningún otro 
  archivo para poder trabajar.

Ejemplos:

Este comando            cambia el directorio actual a / muestra:
============            ========================================
cd c:\fdos\bin          c:\fdos\bin
pushd c:\games          c:\games      (y almacena c:\fdos\bin)
pushd c:\utils          c:\utils      (y almacena c:\games)
pushd c:\packages       c:\packages   (y almacena c:\utils)
dirs                    muestra los directorios guardados por PUSHD:
                          0001=c:\fdos\bin
                          0002=c:\games
                          0003=c:\utils
popd                    c:\utils      (y eliminará c:\utils (0003)
                                      de la pila de directorios)
popd                    c:\games      (y eliminará c:\games (0002)
                                      de la pila de directorios)
popd                    c:\fdos\bin   (y eliminará c:\fdos\bin (0001)
                                      de la pila de directorios)
popd                    muestra el mensaje: 
                          "Directory stack empty"
                          "Pila de directorios esta bacía"

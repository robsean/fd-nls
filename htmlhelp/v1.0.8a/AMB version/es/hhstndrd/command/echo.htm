Comando: echo (comando de command.com)

  ECHO muestra mensajes, o activa o desactiva el comando echo.
  También existe un comand ECHO CONFIG.SYS / FDCONFIG.SYS.
  ECHO es un comando ARCHIVO POR LOTES / AUTOEXEC.BAT.
  También se pude utilizar en la línea de comandos.

Sintaxis:

  ECHO [ON | OFF]
  ECHO [mensaje]

Opciones:

  ON       El modo ECHO es activado. Si tu usas ECHO dentro de un archivo
           por lotes, cada línea de comando se muestra cuando se ejecuta.
  OFF      El modo ECHO es desactivado. Si usa ECHO dentro de un archivo
           por lotes, la línea de comandos no muestra cuando se ejecuta.
           Esto no afecta la salida del comando en sí.
  mensaje  El mensaje que desea imprimir en la pantalla.

Comentarios:

  Los comandos también se pueden suprimir comenzándolos con el simbolo @.
  Escriba ECHO sin parámetros para mostrar la configuración actual de 
  ECHO. ECHO también se puede utilizar fuera de un archivo por lotes.
  ECHO es un comando interno de command.com y no necesita otro archivo
  para que funcione. También existe un comando ECHO con diferentes
  opciones para CONFIG.SYS / FDCONFIG.SYS

Ejemplos:

  EN UN ARCHIVO .BAT / AUTOEXEC.BAT:
    ECHO HOLA,
    ECHO. ¿cómo estás?
    @ECHO ¿bién?
    @ECHO. Me alegra escucha esto

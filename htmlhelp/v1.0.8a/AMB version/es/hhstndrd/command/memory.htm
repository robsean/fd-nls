Comando: memory

  MEMORY muestra la memoria interna utilizada por FreeCOM (command.com)
  internamente.

Sintaxis:

  MEMORY [/?]

Opciones:

  /?  Muestra información sobre el comando.

Comentarios:

  MEMORY muestra la cantidad de memoria que utiliza los alias, el 
  historial y el directorio Stack, el último directorio de cache, Swapinfo
  y así sucesivamente.
  MEMORY es un comando interno de command.com y no necesita otro archivo
  para que funcione.

Ejemplos:

  memory /?
  memory

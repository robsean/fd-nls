Comando: chdir

  CHDIR muestra el nombre o cambia el directorio actual.

Sintaxis:

  CHDIR [unidad] [ruta]
  CHDIR [..] [-] [\] [/?]
        unidad  La letra de la unidad, p. ej. C:
        ruta    El directorio, p. ej. \ejemplo\

Opciones:

  ..      Especifica que desea cambiar al directorio principal.
  -       Si la característica "último directorio" está habilitada, cambie
          al último directorio.
  \       Cambia al directorio raiz.
  /?      Muestra la ayuda.

Comentarios:

  Escriba CHDIR unidad: para mostrar el directorio actual en el directorio
  especificado de manejo.
  Escriba CHDIR sin parámetros para visualizar la unidad actual y 
  el directorio.
  Si no se especifica ninguna unidad, entonces se asume la unidad actual.
  Si no se especifica ninguna ruta, se muestra el directorio actual.
  También puede usar CD en lugar de CHDIR.
  CHDIR es un comando interno de command.com y no necesita ootro archivo
  para que funcione.

Ejemplos:

    chdir c:\fdos     Cambia al directorio c:\fdos
    chdir ..          Cambia de c:\fdos a c:\
    chdir \           Cambia de c:\anydirectory\anysubdirectory a c:\
  Si estás en c:\ :
    chdir fdos        Cambia al directorio c:\fdos
    chdir bin         Cambia al directorio c:\fdos\bin
    chdir d:\example  Cambia el directorio en d: a d:\example, pero no
                      cambia a la unidad d:

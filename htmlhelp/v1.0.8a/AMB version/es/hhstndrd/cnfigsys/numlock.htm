Comando: numlock

  NUMLOCK activa o desactiva el bloqueo de números del teclado.
  NUMLOCK es un comando de CONFIG.SYS / FDCONFIG.SYS.

Sintaxis:

  numlock=on|off

Opciones:

  on   bloqueo de números del teclado está activado.
  off  bloqueo de números del teclado está desactivado.

Comentarios:

  NUMLOCK activa o desactiva el bloqueo de números del teclado.
  Si desea utilizar las teclas numéricas como teclas de flechas, elija
  NUMLOCK=OFF.
  NUMLOCK es un comando interno de kernel.sys y no necesita ningún otro
  archivo para funcionar.

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    numlock=on   
  O:
    numlock=off  puede utilizar las teclas numéricas como teclas de flecha

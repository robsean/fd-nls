Comando: version

  VERSION le dice a FreeDOS qué versión de DOS debe informar.
  VERSION es un comando de CONFIG.SYS / FDCONFIG.SYS.

Sintaxis:

  version=x.y
    Donde x e y son números (cada uno puede tener uno o más dígitos).
    El valor predeterminado es 5.0.

Opciones:

  - ninguna -

Comentarios:

  VERSION es un comando específico de FreeDOS para especificar qué versión
  de DOS se debe informar. Para un kernel con soporte FAT32, 7.10 es una
  buena opción. De lo contrario, 5.0 y 6.22 son valores comunes. FreeDOS
  command.com 'ver /r' muestra los números de versión interna informada y
  FreeDOS. La VERSION no cambia el número de versión de FreeCOM como se
  informa ejecutando 'ver'. Sin embargo, el número de versión de DOS
  cambia, como se informa en 'ver /R'. VERSION es necesaria para hacer
  que algunos viejos programas de DOS piensen que funciona con la versión
  correcta de DOS. VERSION es un comando interno de kernel.sys y no 
  necesita ningún otro archivo para funcionar.

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    version=6.22

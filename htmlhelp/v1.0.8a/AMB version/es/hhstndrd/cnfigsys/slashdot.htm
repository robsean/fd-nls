Comando: ;

  ';' marca la línea de config.sys como un comentario. ';' tiene el mismo
  efecto que 'REM'.
  ';' es un comando de CONFIG.SYS / FDCONFIG.SYS.

Sintaxis:

  ; [comentario]
  REM [comentario]

Opciones:

  comentario  cualquier comentario que desee agregar para recordar por qué
              hizo esto o aquello; también puede ser un comando, que no se
              ejecutará (tal vez con fines de prueba).

Comentarios:

  ';' y 'REM' brindan la capacidad de colocar comentarios dentro del
  archivo de configuración. El texto que sigue al ';' o 'REM' hasta que
  se llegue al final de la línea se ignora. Esto también se puede usar
  para desactivar temporalmente la carga de un dispositivo en particular
  u otra opción.
  ';' es un comando interno de kernel.sys y no necesita otro archivo para
  que funcione.

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    REM DOS=HIGH,UMB  ¡tiene el mismo efecto!
    ;DOS=HIGH,UMB     ¡tiene el mismo efecto!

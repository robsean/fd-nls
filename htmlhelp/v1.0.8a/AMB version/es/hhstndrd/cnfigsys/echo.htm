Comando: echo (comando de config.sys)

  ECHO muestra mensajes. También existe una versión ARCHIVOS POR LOTES/
  AUTOEXEC.BAT/COMMAND.COM
  ECHO es un comando CONFIG.SYS / FDCONFIG.SYS

Sintaxis:

  ECHO [mensaje]

Opciones:

  mensaje  El mansaje que desea mostrar al usuario.

Comentarios:

  ECHO muestra (echos) sus argumentos en la consola durante CONFIG.SYS /
  FDCONFIG.SYS procesandolo cuando se cargan los controladores del
  dispositivo (cuando se ejecutan las líneas DEVICE).
  ECHO es un comando interno a KERNEL.SYS y no necesita ningún otro archivo
  para funcionar. También existe un comando ECHO en COMMAND.COM con
  diferentes opciones.

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    ECHO cargando controlador1
    device=Driver1.sys
    ECHO controlador1 cargado con éxito

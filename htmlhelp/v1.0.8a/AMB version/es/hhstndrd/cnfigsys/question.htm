Comando: ?

  Con '!' y '?' config.sys se puede hacer interactivo.
  Se consultarán las líneas Config.sys que comiencen por '?':
  Se le pedirá al usuario la confirmación antes de que se ejecute una
  línea config.sys que comience con '?'.
  '?' es un comando de CONFIG.SYS / FDCONFIG.SYS.

Sintaxis:

  ?  otro comando config.sys, como DEVICE?=...

Opciones:

  - ninguno -

Comentarios:

  '?' - SIEMPRE pregunta si una sola línea se ejecutará.
  '?' es un comando interno de kernel.sys y no necesita ningún otro 
  archivo para funcionar.

Ejemplos:

  EN CONFIG.SYS / FDCONFIG.SYS:
    FILES?=20
    DOS?=HIGH,UMB
    BUFFERS?=30
    DEVICE?=MyNetWorkDriver.sys

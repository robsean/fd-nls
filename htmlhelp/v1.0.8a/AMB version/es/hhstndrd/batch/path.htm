Comando: path

  PATH muestra o establece una ruta de búsqueda para archivos ejecutables.
  PATH es un comando de ARCHIVO POR LOTES / AUTOEXEC.BAT.
  También se puede usar en la línea de comandos.

Sintaxis:

  PATH [[unidad]ruta[;...]]
  PATH [lista_ruta]
  PATH ;
  PATH [/?]

  unidad    La letra de unidad, por ejemplo C:
  ruta      El directorio, \ejemplo\

Opciones:

  lista_ruta  Establece la lista de rutas. Separe las rutas con punto y
              coma (;).
  ;           Borra todas las configuraciones de ruta de búsqueda y 
              dirige el shell de comandos para que busque solo en el
              directorio actual.
  /?          Muestra la ayuda.

Comentarios:

  Escriba "RUTA ;" para borrar todas las configuraciones de la ruta de
  búsqueda y dirigir el shell de comandos para que busque solo en el
  directorio actual. Escriba "RUTA" sin parámetros para mostrar la ruta
  actual. Algunos comandos son internos de FreeDOS. Sin embargo, muchos
  están en el directorio \bin o \freedos\bin, por lo que es posible que
  desee tener este directorio en la lista de rutas (como en el primer
  ejemplo a continuación). Las rutas de cualquier programa utilizado con
  frecuencia se pueden incluir en la lista de rutas para mayor comodidad.
  Cuando se ingresa un programa o comando externo en el símbolo del
  sistema, FreeDOS lo busca en el directorio actual y luego en cada una
  de las rutas especificadas por el comando de ruta. Si el mismo nombre
  de programa está en directorios de ruta diferentes, el programa se
  iniciará desde la ruta que se menciona por primera vez en la lista de
  rutas. PATH también se puede usar fuera de un archivo por lotes. PATH
  es un comando interno de commando.com y no necesita ningún otro
  archivo para funcionar.

Ejemplos:

  EN UN ARCHIVO .BAT / AUTOEXEC.BAT:
    path c:\fdos\bin;c:\util
    path %path%;c:\games
    El segundo ejemplo agrega c:\games a la lista existente, preservando
    las rutas establecidas previamente.

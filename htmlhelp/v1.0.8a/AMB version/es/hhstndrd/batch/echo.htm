Comando: echo

  ECHO muestra mensajes, activa o desactiva el eco de comandos.
  También existe un comando ECHO de CONFIG.SYS / FDCONFIG.SYS.
  ECHO es un comando de ACHIVO POR LOTES / AUTOEXEC.BAT.
  También se puede usar en la línea de comandos.

Sintaxis:

  ECHO [ON | OFF]
  ECHO [mensaje]

Opciones:

  ON       El modo ECHO está activado. Si usa ECHO dentro de un archivo
           por lotes, cada línea de comando se muestra cuando se ejecuta.
  OFF      El modo ECHO está desactivado. Si usa ECHO dentro de un archivo
           por lotes, la línea de comando no se muestra cuando se ejecuta.
           Esto no afecta la salida del comando en sí.
  mensaje  El mensaje que desea imprimir en la pantalla.

Comentarios:

  Los comandos también se pueden suprimir comenzándolos con el símbolo @.
  Escriba ECHO sin parámetros para mostrar la configuración actual de
  ECHO.
  ECHO también se puede usar fuera de un archivo por lotes.
  ECHO es un comando interno de command.com y no necesita ningún otro
  archivo para funcionar.
  También existe un comando ECHO de CONFIG.SYS / FDCONFIG.SYS con
  diferentes opciones.

Ejemplos:

  EN UN ARCHIVO .BAT / AUTOEXEC.BAT:
    ECHO Hola,
    ECHO. ¿cómo estás?
    @ECHO ¿Bien?
    @ECHO. Me alegra escuchar esto

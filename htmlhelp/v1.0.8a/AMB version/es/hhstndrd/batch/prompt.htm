Comando: prompt

  PROMPT cambia la forma en que aparece el símbolo del sistema.
  PROMPT es un comando de ARCHIVO POR LOTES / AUTOEXEC.BAT.
  También se puede usar en la línea de comandos.

Sintaxis:

  PROMPT [Opciones] [texto]
  texto   Especifica un nuevo símbolo del sistema.

Opciones:

  Prompt puede estar compuesto por caracteres normales y los siguientes
  códigos especiales:

  $$   $ (signo de dólar)
  $_   Devolución de carro y salto de línea
  $B   | (tubo)
  $D   Fecha actual
  $E   Código de escape (código ASCII 27)
  $G   > (signo mayor que)
  $H   Retroceso (borra el carácter anterior)
  $L   < (signo menor que)
  $N   Unidad actual
  $P   Unidad actual y ruta
  $Q   = (signo igual)
  $T   Hora actual
  $V   El número de versión del shell del comando FreeDOS

Comentarios:

  Escriba PROMPT sin parámetros para restablecer la solicitud a la
  configuración predeterminada. PROMPT también se puede usar fuera de
  un archivo por lotes. Cuando se carga NANSI.SYS, PROMPT también puede
  usar diferentes colores de primer plano / fondo con la configuración:
  prompt $e[#;#;#m.
  PROMPT es un comando interno de command.com y no necesita ningún otro
  archivo para funcionar.

Ejemplos:

  EN UN ARCHIVO .BAT / AUTOEXEC.BAT:
    prompt $p$g       Esto es lo mismo que la configuración predeterminada
    prompt $p$g ¡Ingrese un comando!
  Cuando se carga NANSI.SYS, también puede cambiar los colores:
    prompt $e[1;33;44m    Para negrita / amarillo / azul.

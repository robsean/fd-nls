Comando: rem

  REM registra comentarios (observaciones) en un archivo por lotes o 
  config.sys.
  REM es un comando de ARCHIVO POR LOTES / AUTOEXEC.BAT.

Sintaxis:

  REM [comentario]

opciones:

  comentarios   cualquier comentario que desee agregar para recordar por
                qué hizo esto o aquello; también puede ser un comando, que
                no se ejecutará (tal vez con fines de prueba).

Comentarios:

  FreeDOS ignora todo en una línea REM. Utilice las observaciones para
  agregar notas explicativas al config.sys y para deshabilitar líneas en
  el archivo config.sys para que no se ejecuten. En CONFIG.SYS también
  puede usar ";" al principio de la línea. REM es un comando interno de
  command.com y no necesita ningún otro archivo para funcionar.
  REM también existe como comando CONFIG.SYS / FDCONFIG.SYS.

Ejemplos:

  EN UN ARCHIVO .BAT / AUTOEXEC.BAT:
    REM Configure la última letra de unidad disponible:
    lastdrive=z
   
    REM Ya no quiero usar este controlador:
    REM DEVICE=FOO.SYS

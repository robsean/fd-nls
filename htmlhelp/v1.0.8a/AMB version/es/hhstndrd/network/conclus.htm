Trabajo en red con FreeDOS - Conclusión

  Hay un montón de de aplicaciones de redes para usuarios disponebles de
  FreeDOS, algunos programas gratuitos, algunos programas de software
  libre. Esta es una buena noticia: Para servir archivos para navegar
  por la web - todo es posible con FreeDOS.
  Hay desventajas, sin embargo: Una integración sin costuras en otros
  sistemas operativos networks es difícil de lograr. Ni la moderna SMB
  (Windows y GNU/Linux networks) ni NFS (GNU/Linux, UNIX) networks están
  trabajando sin defectos con una máquina DOS.
  Y tarjetas modernas de interfaz de red casi dejaron de dar soporte con
  paquetes de controladores que una vez fueron los controladores
  predeterminado para la DOS networking. Otros
  controladores necesitan software adicional que no es libre y no pueden
  distribuirse libremente. Así que todavía hay mucho que hacer - para los
  desarrolladores, sino también para las personas que quieran ayudar con
  esta documentación:
    * PPP no se explica - ¿no hay nadie que tenga un Modem?
    * Una guia para el uso de FreeDOS con WiFi (LAN inalámbrica) sería
      genial.
    * NFS no está cubierto todavía.
    * Redes de FreeDOS dentro de una máquina virtual no se explica aquí.
    * Una mirada más profunda en el cliente MS podría ayudar a muchos
      usuarios.
    * Todavía no se documentó como Novell Netware se puede utilizar
      con FreeDOS.
  Como este es un wiki ahora, eres bienvenido para añadir informaciones y
  continuar este trabajo. Que se diviertan.

Redes de FreeDOS - Crynwr

  La colección de paquetes de controladores Crynwr proporcionan conexión de
  Redes. Estos fueron escritos por
  Russell Nelson del software Crynwr. Estos se basan en las especificación
  original ftp.com, el esqueleto es de libre disposión y la mayor parte de
  proveedores de tarjetas de Red proporcionan un paquete de controladores.
  Como se trata de un proceso estandar abiesto este es el más utilizado por
  GNU y los programadores de shareware. (Ver también WatTCP).
  Hay "cuñas", que permiten tanto ODI y NDIS controladores NIC que se
  utilizarán, pero que las aplicaciones creen que son paquetes de
  controladores. Los controladores Crynwr son software de código abierto
  certificado por OSI. A diferencia de NDIS o ODI, el paquete de
  controladores específicos es libremente copiable.

Redes de FreeDOS - Aplicaciones TCP/IP

Resumen

  Nosotros escuchamos en la "Historia de las Redes de DOS" como fueron
  muy populares las aplicaciones TCP/IP para DOS en los ochenta y los 
  noventa del pasado siglo. Esta era se acabó cuando Microsoft introdujo 
  Windows 95: en el suministró TCP/IP y otras funciones internas de red
  mas sencillas, con una interfaz de usuario amigable. Con estos clientes
  muchas aplicaciones de red comerciales para DOS se desvanecieron. Pero
  el software de dominio público, programas freeware y shareware fueron
  preservados en internet. Algunos todavía se desarollan hoy. Incluso se
  han escrito nuevos software.
  Las aplicaciones DOS TCP/IP incluyen herramientas de red clásicas como 
  ping, LPR o Finger, así como clientes ssh y NFS, servidores FTP, 
  servidores web, software de correo y noticias o navegadores Web. 
  Incluso existen una solución completa de red peer-to-peer llamada 
  "Network Enhanced Operating System" (NeOS - ver: http://ftp.carnet.hr/
  pub/pc/networking/NeOS/README (*01)) que fue desarrollada entre 1995
  -2000 como parte del proyecto "Eureka" de la Unión Europea (ver:
  http://www.eureka.be/inaction/AcShowProject.do;jsessionid=
  7f00000122b839cfc1d1a5c44d359913f31b02769f2f?id=1317 (*02)).
  Ahora es Postcardware.

Aplicaciones

  No es posible cubrir todas las aplicaciones TCP/IP de este documento.
  Pero aquí hay algunos ejemplos notables:

Arachne

    (Imagen de una captura de pantalla de Arachne)
  "Arachne" es un navegador web gráfico para DOS. fue creado en 1996
  por el programador checo Michael Polak y su empresa xChaos/Arachne
  Labs (ver: http://www.arachne.cz/ (*14)). En 2002 Michael Polak decidió
  hacer Arachne software libre. La nueva licencia es la GPL de GNU (ver:
  http://home.arachne.cz/archive/v01.n1917 (*15)).
  El navegador soporta varios formatos de imagen. Las tablas y los marcos
  se muestran correctamente, renderiza HTML 4 y CSS 1,0 y las versiones 
  1.90J3 y nuevas, llegando a entender UTF-8. Otros servicios de Arachne
  incluyen FTP, NNTP, IRC-chat, RSS, POP3 y SMTP. Mire aquí (ver: http://
  web.archive.org/web/20070311000724/http://www.cisnet.com/glennmcc/
  ara-gpl/history.txt (*16)) para una historia completa del software.
  Arachne está todavía activamente en desarrollo y los programadores est
 án invitados a unirse a una lista de correo (ver: http://tech.groups.
  yahoo.com/group/ArachneDevelopment/ (*17)). La última versión se puede
  encontrar en el sitio de Glenn McCorkle (ver: http://www.glennmcc.org/
  (*18)). Lea más sobre el mismo en Wikipedia.

EZ-NOS2

    (Imagen de una captura de pantalla de EZ-NOS2)
  "EZ-NOS2" es uno de los descendientes de KA9Q de Phil Karn. Autualmente
  está desarollado por DOS Solutions (ver: http://www.dossolutions.pwp.
  blueyonder.co.uk/eznos.htm (*19)). El paquete de software incluye un
  servidor Web, un servidor FTP y un servidor BOOTP, así como un cliente
  de correo electrónico.
  Como todos los descendientes de KA9Q EZ-NOS2 está sujeto a licencia GPL
  de GNU. La fuente se puede descargar como ez_src.zip (ver: http://www.
  dossolutions.pwp.blueyonder.co.uk/progs/eznos/ez_src.zip (*20)).

SSH2DOS

    (La captura de pantalla muestra una sesión SSH cerrada a un servidor
     Debian.)
  "SSH2DOS" es un cliente de SSH, SFTP y SCP para DOS. Estos servicios
  reemplazan más y más los clásicos servicios Telnet y FTP, que se 
  consideran menos seguros.
  SSH2DOS sue creado por el desarrollados Húngaro Nagy Daniel en 2000 y
  puede ser descargado en sourceforge.net (ver: http://sourceforge.net/
  projects/sshdos (*21)). Se libera bajo la GPL de GNU. Utiliza el código
  del cliente SSH de PuTTY así como la biblioteca de nucleo de WatTCP.

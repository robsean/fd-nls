Redes de FreeDOS - Red

  Las redes permiten dos o más equipos para conectarse y compartir
  información, como por ejemplo a través de Internet y un navegador web.
  Para conectarse, necesitará un módem o una tarjeta Ethernet. Si decide
  Ethernet, necesitará un controlador compatible, como Crynwr, ODI o NDIS.
  El protocolo más común es TCP/IP, pero muchas redes también admite
  IPX/SPX. Para compartir los datos con otros sistemas, necesitará NFS o
  CIFS.

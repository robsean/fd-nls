Redes de FreeDOS - NDIS

  El NDIS (Network Driver Interface Specification) es de Microsoft y 3Com
  y proporciona conectividad de red. Como ODI, NDIS da una interfaz
  uniforme para la aplicación, independientemente del método de red, pero
  uno diferente de ODI, o los controladores de paquetes.
  Los controladores NDIS utilizan la mayor cantidad de memoria de todos
  los controladores en el uso, por lo que sólo deben utilizarse cuando no
  hay otro conductor está disponible o el software que desea utilizar sólo
  funciona con NDIS.

Visor de ayuda HTML 5.3.2 - Etiquetas compatibles:

  Este documento describe el subconjunto de HTML compatible con la Ayuda.
  <!-- -->
    Comentario. La Ayuda HTML ignora todas las etiquetas y el texto del
    comentario.
  <A>
    La ayuda es totalmente compatible con los atributos HREF y NAME,
    excepto que:
      * Los enlaces a otros archivos solo pueden ser relativos, no
        absolutos. Entonces HREF="..\foo.htm" funciona pero 
        HREF="C:\foo.htm" no funciona 
      * Las comillas dobles "" siempre deben colocarse alrededor de los
        nombres de archivos y marcadores, por ejemplo HREF="foo.htm" pero
        no HREF=foo.htm 
      * Se ignoran las referencias a direcciones web y de correo
        electrónico.
      * Deben usarse nombres de archivos de estilo DOS (8.3)
  <B>
    El texto en Negrita se muestra en blanco brillante.
  <BODY>
    Indica el comienzo del texto a mostrar.
  <BR>
    Salto de lñinea.
  <CODE>
    Code solo se muestra como texto en negrita (en blanco brillante).
  <EM>
    El texto enfatizado, como el texto en cursiva, se muestra usando el
    color magenta (un tono de púrpura).
  <H1>
    La ayuda pone este texto de encabezado 1 en negrita y pone una línea
    gruesa debajo de este.
  <H2>, <H3>
    El texto de los encabezados 2 y 3 se pone en negrita y se inicia un
    nuevo párrafo.
  <HEAD>
    Encabezado HTML.
  <HR>
    Crea una regla horizontal a lo largo de la pantalla. Solo una regla
    horizontal sin ningún atributo es compatible en la actualidad.
  <HTML>
    Denota un archivo HTML.
  <I>
    El texto en cursiva se muestra usando el color magenta (una sombra de
    morado).
  <LI>
    Un objeto de lista.
  <OL>
    En la actualidad, las listas ordenadas se representan como listas sin
    ordenar. Las listas integradas no son compatibles.
  <P>
    Nuevo párrafo.
  <PRE>
    Texto preformateado
  <STRONG>
    El texto fuerte, como el texto en negrita, se muestra en blanco
    brillante.
  <TITLE>
    No hace nada, todavía.
  <TT>
    El texto TT, como el texto en cursiva, se muestra usando el color
    magenta (un tono de púrputa).
  <UL>
    Las listas integradas no son compatibles actualmente.

Entidades de caracteres compatibles con la Ayuda HTML:

  Se admiten más de 250 entidades de caracteres, incluidas todas las
  entidades Latin-1. NCR (referencias numéricas de caracteres) y UTF-8
  también son compatibles. Sin embargo, la ayuda solo puede mostrar los
  caracteres que están disponibles en la página de códigos DOS
  actualmente activa. Para obtener una lista de las páginas de códigos
  de las que la ayuda está "al corriente", escriba "AYUDA /C" en el
  simbolo del sistema.

Notas generales sobre la ayuda HTML:

   1. El texto se ajusta si no cabe en la línea.
   2. Las etiquetas no distinguen entre mayúsculas y minúsculas, por lo
      que <A HREF="FOO.HTM"> equivale a <a href="foo.htm">.
   3. El tamaño máximo para un documento es 64k.
   4. Solo se admiten nombres de archivo cortos (8.3).
   5. La ayuda distingue entre tipos de archivo por extensión de nombre
      de archivo. Por lo tanto, la extensión .htm se recomienda para
      archivos HTML.

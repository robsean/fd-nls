El saludo del inventor

  ¡Hola y bienvenidos a FreeDOS! Ayudé a crear FreeDOS en 1994 para que
  pudiéramos tener una versión de DOS que todos pudieran usar y compartir.
  La mayor parte de FreeDOS se lanza bajo la GNU GPL; el resto está bajo
  una licencia gratuita similar. Si es desarrollador, no dude en consultar
  nuestro código fuente y realizar mejoras. O mejor aún, ¡conviértase en
  un colaborador habitual de FreeDOS y ayude a mejorar FreeDOS para todos!
  También hay muchas otras formas de contribuir a FreeDOS además de
  escribir código. Necesitamos ayuda para escribir documentación, facilitar
  las cosas para el usuario y probar errores. Todo lo que necesita es
  interés en FreeDOS y un deseo de contribuir. ¡Por favor únete a 
  nosotros!

  Jim Hall

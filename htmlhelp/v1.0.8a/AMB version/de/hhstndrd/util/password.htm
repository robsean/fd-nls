Befehl: password

  PASSWORD ist ein Passwort-Tool für DOS, das mehrere Benutzer
  verwalten kann.

Syntax:

  password [/adduser] [/login] [/remuser] [/?]

Optionen:

  /adduser  Fuegt dem Programm einen neuen Benutzer hinzu.
            Dieser Befehl muß zuallererst ausgeführt werden.
  /login    Startet die Login Prozedur. Wenn Sie einen existierenden
            Benutzer, der mit /adduser erzeugt wurde und sein Passwort
            eingeben, können Sie FreeDOS starten.
  /remuser  Entfernt einen Benutzer aus dem Programm.
  /?        Zeigt die Hilfe.

Kommentar:

  Sie sollten "PASSWORD" ganz am Anfang der AUTOEXEC.BAT eintragen.
  Sie sollten ganz am Anfang der CONFIG.SYS "SWITCHES=/F" eintragen,
  um zu vermeiden, daß der Benutzer durch Umgehung von CONFIG.SYS 
  /AUTOEXEC.BAT das Passwort umgeht, indem er beim Booten "F5" drückt.
  Sie sollten außerdem im BIOS einstellen, daß zuerst von Festplatte
  gebootet wird (vergessen Sie nicht, ein BIOS-Passwort zu setzen).
  PASSWORD unterstützt NLS (national language support - nationale Sprach-
  unterstützung).

Beispiel:

  IN DER KOMMANDOZEILE:
    password /adduser  zuerst geben Sie die möglichen Benutzer und deren
                       Passwort ein, dann:
  IN DER CONFIG.SYS / FDCONFIG.SYS:
    switches=/f        diesen Befehl in der ersten Zeile der config.sys
                       einfügen, um zu vermeiden, daß ein Benutzer die
                       config.sys umgeht, indem er beim Booten "F5" 
                       drückt.
  IN DER AUTOEXEC.BAT:
    password /login    Der Benutzer muß in Zukunft seinen Benutzernamen
                       und sein Passwort eingeben.
  Beim nächsten Neustart werden Sie beim Hochfahren nach dem Benutzer-
  namen und dem Passwort gefragt.

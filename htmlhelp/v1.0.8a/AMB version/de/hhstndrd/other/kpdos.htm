KPDOS v3.0 - Keyboard Layout Pack - DOS Codepages

  KPDOS entält Bibliothekdateien, die zusammen mit KEYB verwendet
  werden.
  Das offizielle FreeDOS Tool um eine der unten angeführten Dateien
  mit einer der möglichen Codepages zusammenarbeiten zu lassen, ist
  MODE. Um MODE verwenden zu können, muß DISPLAY bereits laufen.
  Alle CPI Dateien in dieser Liste sind im CPIDOS Paket enthalten.
  Die Quelldateien können mit KC kompiliert werden, welches im
  Gegenzug mit KLIB in die angepassten Bibliothekdateien integriert
  werden können. Weitere Details entnehmen Sie bitte dem KC Paket,
  das auch KLIB entält.

  FreeDOS offizielle Tastaturlayouts, v3.0
  ========================================
  Dies ist die Liste aller Namen der Tastaturlayouts, die für FreeDOS
  erhältlich sind. Wenn der Name das Format zz999 hat, bedeutet das,
  daß ein Name und eine Identifizierungsnummer vorhanden sind, die
  angegeben werden MUESSEN, andernfalls wird KEYB das Layout für eine
  andere Tastatur dieses Landes laden.
  In diesem Fall muß der Anwender folgenden Befehl eingeben:
    "KEYB ZZ /ID:999".
  Falls der Name das Format "aa,bb" hat, bedeutet das, daß eigentlich
  (mindestens) zwei Namen vorhanden sind, die verwendet werden können,
  um die Tastaturbelegung dieses Landes/dieser Sprache zu laden. (Sie
  können andere Namen und/oder Identifikationsnummern im Unterbereich
  [GENERAL] der KEY Dateien finden.)
  Bitte lesen Sie in den Anmerkungen am Ende dieser Datei nach, wann
  immer Sie eine (Nummer) auf der rechten Seite des Namens Ihrer Tastatur
  haben.

  Arbeiten mit Zweischrift-Tastaturen
  ===================================
  Wenn Sie mit Tastaturen arbeiten, die dafür gedacht (oder ausgelegt)
  sind, mit mehr als einem Alphabet / einer Schrift zu arbeiten, ist
  es notwendig, eine Tastenkombination zu drücken, um das nicht-
  lateinische Layout (Schriftzeichen) zu erhalten. Siehe auch die
  Tabellen weiter unten.
  Zweischrift-Tastaturen - arabisch, armenisch, azerisch, belarusisch,
  bulgarisch, cherokee, georgisch, griechisch, hebräisch, kasachisch,
  kirgisisch, mazedonisch, mongolisch*, russisch, serbisch,
  tadschikisch, ukrainisch und usbekisch:

      Layer Beschreibung     Tastenkombination
      ------------------     -----------------
      Lateinisch             [ALT]+[LINKE_HOCHTASTE]
      Nationales Alphabet    [ALT]+[RECHTE_HOCHTASTE]

  *Kyrillisches Alphabet.

  Italianische Tastaturen, wenn sie mit cp869 (griechisch)
  arbeiten:

      Layer Beschreibung     Tastenkombination
      ------------------     -----------------
      Lateinisch             [ALT]+[LINKE_HOCHTASTE]
      Griechisch phonetisch  [ALT]+[RECHTE_HOCHTASTE]

  Lettische Tastatur lv455:
    (Kyrillischer Layer verfügbar, wenn mit cp3012
    gearbeitet wird)

      Layer Beschreibung     Tastenkombination
      ------------------     -----------------
      Lateinisch QWERTY      [ALT]+[LINKE_HOCHTASTE]
      Lateinisch UGJRMV      [ALT]+[RECHTE_HOCHTASTE]
      Kyrillisch phonetisch  [ALT]+[LINKE_STRG-TASTE]

  Litauische Tastatur lt456:
    (Kyrillischer Layer verfügbar, wenn mit cp771 oder
    cp772 gearbeitet wird)

      Layer Beschreibung     Tastenkombination
      ------------------     -----------------
      Lateinisch QWERTY      [ALT]+[LINKE_HOCHTASTE]
      Lateinisch AZERTY      [ALT]+[RECHTE_HOCHTASTE]
      Kyrillisch phonetisch  [ALT]+[LINKE_STRG-TASTE]

  Norwegische und schwedische/finnische Tastaturen, wenn
  mit cp 30000 gearbeitet wird (cp30000: Saami)

      Layer Beschreibung     Tastenkombination
      ------------------     -----------------
      Lateinisch QWERTY      [ALT]+[LINKE_HOCHTASTE]
      Lateinisch ASERTT      [ALT]+[RECHTE_HOCHTASTE]

  US Tastatur, wenn mit cp30034 gearbeitet wird:

      Layer Beschreibung     Tastenkombination
      ------------------     -----------------
      Lateinisch             [ALT]+[LINKE_HOCHTASTE]
      Cherokee               [ALT]+[RECHTE_HOCHTASTE]

  Turkmenische Tastatur, wenn mit cp59234(*) gearbeitet wird:

      Layer Beschreibung     Tastenkombination
      ------------------     -----------------
      Lateinisch             [ALT]+[LINKE_HOCHTASTE]
      Kyrillisch phonetisch  [ALT]+[RECHTE_HOCHTASTE]

    (*) Turkmenisch kann mit dem lateinischen Alphabet
        geschrieben werden, wenn es mit cp852 verwendet wird.
        In diesem Fall ist der kyrillische Layer nicht
        verfügbar.

  Das nachfolgende kyrillische phonetische Schema wendet
  folgendes an:

  1) Lettische Tastaturen (alle außer lv455), wenn sie mit
     cp 3012 ("RusLat") arbeiten.
  2) Litauische Tastaturen (alle außer lt456), wenn sie mit
     cp771 oder cp772 arbeiten.
  3) Polnische Tastaturen, wenn sie mit cp848 (ukrainisch)
    oder cp849 (belarusisch) arbeiten.
  4) Rumänische Tastaturen, wenn sie mit cp848 (ukrainisch)
     oder cp30010 (kyrillisches gagauz/rumänisch) arbeiten.

      Layer Beschreibung     Tastenkombination
      ------------------     -----------------
      Lateinisch             [ALT]+[LINKE_HOCHTASTE]
      Kyrillisch phonetisch  [ALT]+[RECHTE_HOCHTASTE]

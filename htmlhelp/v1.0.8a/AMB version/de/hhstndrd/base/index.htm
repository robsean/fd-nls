Externe Befehle:

  Diese Dateien sind Teil der Standard FreeDOS Distribution.
  Damit sie funktionieren, benötigt man eine existente ausführbare
  Datei (*.com oder *.exe-Datei) und FreeDOS muß bereits gestartet
  sein. Man kann sie ausführen, indem man den Namen eintippt, z.B.
  "edit" oder "format" und - wenn notwendig - zusätzlich die Option/
  den Schalter usw. vgl. SYNTAX. Das Programm sollte sich im aktuellen
  Verzeichnis befinden, andernfalls sollten Sie vorher den CD oder
  den PATH Befehl benutzen.

Befehl: htmlhelp

  HTMLHELP ist diese Sammlung von Hilfedateien im HTML Format.
  Klicken Sie hier wenn Sie wissen möchten, wie Sie die 
  Hilfe benutzen können.

Syntax:

  HELP [Optionen] [Thema]

Optionen:

  Grundlegende Optionen:
  Thema            Zeige zu diesem Thema Hilfe.
  /M               Erzwinge monochrome Anzeige.
  /F1              Benutze schönes Farbschema 1.
  /F2              Benutze schönes Farbschema 2.
  /A               Benützt ASCII anstelle von erweiterten Zeichensätzen.
  /Cnnn            Sage HELP, daß der Codepage nnn ist; besser als
                   anzeigen lassen.
  /?               Zeigt die Hilfe.
  Weitere Optionen:
  /O[Pfad[Datei]]  Hebt den Hilfepfad auf. Wenn keine Datei angegeben
                   wird, wird index.htm unterstellt.
  /LDatei          Lädt eine andere Datei als index.htm, die im 
                   Bereich des Hilfepfads liegt.
  /HThema          Wenn der Benutzer F1 drückt oder auf "Hilfe 
                   über Hilfe" klickt wird die Hilfe diese 
                   Datei laden. Standard ist help.htm.
  Umgebungsvariablen:
  HELPPATH         Verzeichnis, das Ihre Hilfedateien entält.
  HELPCMD          Setzen Sie hier /M, /A, F1, F2 um sie zum Standard
                   zu machen.

Kommentar:

  HTMLHELP wird auch HELP oder HHSTNDRD genannt. 
  Aktuelle Version: HHSTNDRD 1.0.8a de.
  Unter FreeDOS (nicht im Internet) besteht es aus einem DOS HTML-
  Betrachter (help.exe) und den HTML Hilfedateien.
  HELP unterstützt NLS (national language support).

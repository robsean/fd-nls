Befehl: echo (config.sys-Befehl)

  ECHO zeigt Meldungen an. ECHO ist ein CONFIG.SYS / FDCONFIG.SYS
  Befehl. Von ECHO gibt es auch eine BATCH-DATEI/AUTOEXEC.BAT/COMMAND.COM
  Version.

Syntax:

  ECHO [Meldung]

Optionen:

  Meldung  Die Meldung, die Sie ausgeben wollen.

Kommentar:

  ECHO zeigt (echot) seine Argumente auf der Konsole aus, während die
  CONFIG.SYS / FDCONFIG.SYS ausgeführt wird und die Treiber
  geladen werden (wenn die DEVICE= Zeilen ausgeführt werden).
  ECHO ist ein interner Befehl, d.h. der Befehl ist in der kernel.sys
  enthalten. Es wird zur Ausführung des Befehls kein zusätzlicher
  Befehl (Datei) benötigt.
  Den ECHO-Befehl gibt es auch COMMAND.COM-Befehl mit anderen Optionen.

Beispiel:

  IN DER CONFIG.SYS / FDCONFIG.SYS:
    ECHO lade Treiber1
    device=treiber1.sys
    ECHO Treiber1 wurde erfolgreich geladen

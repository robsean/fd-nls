Befehl: eecho

  EECHO zeigt Meldungen an.
  EECHO ist ein CONFIG.SYS / FDCONFIG.SYS-Befehl.

Syntax:

  EECHO [Meldung] mit ANSI Escape Sequenz

Optionen:

  Meldung  Die Meldung, die Sie an den Benutzer ausgeben wollen.

Kommentar:

  EECHO erlaubt für das Echo-ing ANSI Escape Sequenzen (definiert z.B.
  die Tastatureingabe um). Verwenden Sie ein Dollar-Zeichen ($) um das
  ANSI Escape Zeichen zu repräsentieren. Anmerkung: Es muß zuvor ein
  ANSI-Treiber wie z.B. NANSI geladen werden, damit man EECHO verwenden
  kann.
  EECHO ist ein interner Befehl, d.h. der Befehl ist in der kernel.sys
  enthalten. Es wird zur Ausführung des Befehls kein zusätzlicher
  Befehl (Datei) benötigt.

Beispiel:

  IN DER CONFIG.SYS / FDCONFIG.SYS:
    EECHO feeling $[33;44;1m blue :-)

Befehl: date

  DATE zeigt oder setzt das Datum.

Syntax:

  DATE [neues Datum]
  DATE [/D] [/?]
       Geben Sie DATE ohne Parameter ein, um die aktuelle 
       Datumseinstellung und die Abfrage für eine neue anzuzeigen.
       Drücken Sie ENTER, um das Datum beizubehalten.

Optionen:

  neues Datum  in folgendem Format:
               mm-dd-yy oder mm-dd-yyyy
               mm/dd/yy oder mm/dd/yyyy
               mm.dd.yy oder mm.dd.yyyy
               Das neue Datum Ihres Systems. Die Werte für den Tag 
               (dd), Monat (mm) und das Jahr (yy bzw. yyyy) können
               durch einen Punkt, Bindestrich oder einen Schrägstrich
               getrennt werden. Das Jahr (yy oder yyyy) kann entweder
               zweistellig oder vierstellig angegeben werden.
  /D           zeigt das Datum, ohne nach einer neuen Einstellung 
               zu fragen.
  /?           Zeigt die Hilfe

Kommentar:

  Wenn ein zweistelliges Jahr angegeben wird, wird bei Werten von
  00-79 davon ausgegangen, daß die Jahre 2000-2079 gemeint sind. 
  Bei Werten zwischen 80-99 wird davon ausgegangen, daß die Jahre 
  1980-1999 gemeint sind. 
  Wenn Sie nach einem neuen Datum gefragt werden, können Sie ENTER 
  drücken, um die derzeitigen Einstellungen ohne Änderung zu 
  beizubehalten.
  DATE ist ein interner Befehl, d.h. der Befehl ist in der command.com
  enthalten. Es wird zur Ausführung des Befehls kein zusätzlicher 
  Befehl (Datei) benötigt.

Beispiel:

  date 04-22-2007 oder date 04-22-07
  date 04/22/2007 oder date 04/22/07
  date 04.22.2007 oder date 04.22.07

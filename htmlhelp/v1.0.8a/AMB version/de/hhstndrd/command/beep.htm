Befehl: beep

  BEEP gibt einen Piepston aus (in der Regel über den Systemlautsprecher).
  BEEP ist ein BATCH-DATEI / AUTOEXEC.BAT Befehl.
  Er kann aber auch in der Kommandozeile verwendet werden.

Syntax:

  beep [Optionen]

Optionen:

  /?  Zeigt die Hilfe

Kommentar:

  BEEP kann auch außerhalb einer Batch-Datei verwendet werden.
  BEEP ist ein interner Befehl, d.h. der Befehl ist in der command.com
  enthalten. Es wird zur Ausführung des Befehls kein zusätzlicher 
  Befehl (Datei) benötigt.

Beispiel:

  IN EINER .BAT DATEI ODER DER AUTOEXEC.BAT:
    beep

Command: exit

  EXIT exits the current FreeDOS Command Shell.

Syntax:

  EXIT [/?]

Options:

  /?  Shows the help.

Comments:

  EXIT is a command internal to command.com and needs no other file
  in order to work.

Examples:

  EXIT
  EXIT /?

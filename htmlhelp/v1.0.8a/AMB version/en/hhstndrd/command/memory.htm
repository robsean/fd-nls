Command: memory

  MEMORY displays the memory used by FreeCOM (command.com)
  internally.

Syntax:

  MEMORY [/?]

Options:

  /?  Shows information about the command.

Comments:

  MEMORY shows how much memory is used by Aliases, History, 
  Directory Stack, Last dir cache, Swapinfo and so on.
  MEMORY is a command internal to command.com and needs no other file
  in order to work.

Examples:

  memory /?
  memory
